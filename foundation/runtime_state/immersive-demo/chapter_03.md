# 第3章 · 黑雾初醒

痛。

不是尖锐的、爆裂的痛，而是从骨髓深处渗出的、缓慢而顽固的钝痛，像无数细小的虫豸在血管里爬行、啃噬。每一次心跳，都带来一阵沉闷的鼓胀，仿佛有什么东西在胸腔里生根发芽，要撑破皮肉钻出来。

他睁开眼。

视野里是浓得化不开的黑暗，不是纯粹的黑，而是掺杂着某种粘稠、流动的灰。空气里弥漫着腐败的甜腥，混合着金属锈蚀和某种难以名状的、类似陈旧皮革被水浸泡后的气味。他躺在地上，身下是冰冷、粗糙、布满细小颗粒的硬物——不是泥土，更像是某种风化的岩石，表面附着滑腻的苔藓。

记忆是一片空白。

不，不是完全的空。有碎片，尖锐而混乱，像被打碎的镜子，每一片都映出扭曲的光影：刀锋划过血肉的触感，温热的液体溅在脸上的黏腻，黑暗中急促的呼吸，还有……坠落。无尽的、失重的坠落，穿过层层叠叠的、粘稠如墨的雾。

他是谁？

这个问题像一根冰冷的针，刺入混沌的意识。没有答案。只有一种近乎本能的、刻入骨髓的警觉，如同黑暗中潜伏的野兽，在嗅到危险气息时瞬间绷紧了全身的肌肉。

他试着动了动手指。关节僵硬，但还能听从指令。他缓缓撑起身体，动作牵扯到胸腹间的钝痛，让他闷哼一声。视野稍微适应了昏暗，他发现自己身处一个狭窄的通道。通道两侧是粗糙开凿的岩壁，高不见顶，隐没在上方的黑暗里。岩壁上布满了纵横交错的、非自然的刻痕，有些像是爪印，有些则像是被某种强酸腐蚀出的坑洞。通道向前延伸，没入更深的黑暗，向后……同样是无尽的幽邃。

他低头看向自己。身上穿着一件破损的、看不出原本颜色的紧身衣物，材质坚韧，但多处撕裂，露出下面苍白的皮肤。皮肤上，尤其是手臂和胸口，布满了细密的、暗红色的纹路。那不是伤口，更像是血管异常凸起、颜色加深后形成的网络，纹路的源头，似乎都指向心脏的位置。他伸手触碰胸口，指尖传来异样的搏动感，比心跳更慢，更沉重，每一次搏动，都伴随着那深入骨髓的钝痛。

**微粒残留**。

一个陌生的词毫无征兆地跳入脑海，带着冰冷的、不容置疑的确定性。这不是记忆，更像是某种被强行植入的知识，或者说，是身体在向他发出警告。有什么东西留在了他的体内，不属于他，却在侵蚀他，改变他。

他深吸一口气，试图压下那股源自存在本身的恐惧——对空白的过去，对未知的现在，对体内异物的恐惧。恐惧是毒药，会让人软弱。他不需要软弱，他需要……活下去。

这个念头清晰而坚定，仿佛是他此刻唯一能抓住的浮木。

他检查身上。除了那身破烂衣物，腰间还挂着一个皮质刀鞘。他抽出里面的武器。是一把短刃，长度不足小臂，通体黝黑，只在刃口处有一线极淡的、仿佛吸收了所有光线的暗芒。刀柄缠绕着某种干燥的、带有鳞片质感的皮革，握在手中，冰凉，却异常贴合掌心。他翻转刀身，借着岩壁上不知何处渗出的、极其微弱的、仿佛磷火般的幽光，看到了刀脊上刻着的纹路。

那不是装饰。纹路极其古老、繁复，由无数细小的、仿佛活物般扭曲的符号连接而成，构成一个他无法理解的整体。它们不像文字，更像是一种……封印？或者，是某种力量的轨迹？凝视久了，那些符号似乎会微微蠕动，散发出一种难以言喻的、既神圣又亵渎的气息。他移开目光，心脏的搏动似乎与短刃上残留的某种冰冷韵律产生了短暂的共鸣，随即又沉寂下去。

**刻有古老符文的短刃**。这是他的武器。也是此刻他唯一能确认的、与“过去”可能有关的物件。伏笔，如同黑暗中埋下的种子，等待着破土的时机。

他将短刃重新插回刀鞘，动作流畅，仿佛已经重复了千万遍。然后，他开始感知周围的环境。空气是凝滞的，带着湿冷和压抑。通道里没有风，但能感觉到某种极其缓慢的、来自更深处的“流动”，那不是气流，更像是……黑暗本身在呼吸。

他尝试调动体内的某种力量。意念微动，身体周围的阴影似乎变得浓郁了一些，他的轮廓在昏暗的光线下变得更加模糊，仿佛随时可以融入岩壁的暗影之中。**影迹潜行**。这是他的能力，一种近乎本能的技艺。等级……五？又一个突兀的概念。他无法理解“等级”的确切含义，但能感觉到这种能力的强度与局限，以及它此刻处于可以随时激发的状态。

他需要信息，需要离开这里，需要弄清楚体内那该死的“微粒残留”到底是什么，以及……他究竟是谁。

他选择了向前。没有理由，只是直觉。向后可能是来路，也可能是绝路，而前方，至少代表着未知的可能性。

通道蜿蜒向下，坡度平缓但持续。脚下的地面从粗糙的岩石逐渐变成某种更加致密、光滑的物质，像是被高温熔融后又冷却形成的琉璃，踩上去发出轻微的、空洞的回响。岩壁上的刻痕也发生了变化，开始出现更多规则的几何图形，以及一些难以辨认的、仿佛仪式阵列般的图案。空气里的腐败甜腥味越来越浓，其中还夹杂了一丝……硫磺的气息。

他放轻脚步，呼吸调整到最细微的频率，整个人如同融入环境的幽灵。影迹潜行并未全力发动，只是让他的存在感降到最低。在这种地方，任何一点多余的声音或气息，都可能招致无法预料的危险。

走了大约一刻钟，前方出现了岔路。三条通道，分别向左、向右和继续向下延伸。三条通道的入口处，岩壁的颜色和质地都有细微差别：左边的通道岩壁泛着暗红色，温度似乎略高；右边的通道岩壁覆盖着厚厚的、灰白色的菌毯，微微蠕动；中间的通道则最为幽深，黑暗仿佛实质，连那微弱的磷光都无法渗透进去。

他停在岔路口，目光扫过三个入口。没有标记，没有足迹，没有任何可供判断的线索。就在他凝神感知时，胸口那异样的搏动突然加剧了。

咚……咚……

沉重，缓慢，带着一种诡异的牵引力。他捂住胸口，感觉到那暗红色的血管纹路似乎在微微发烫。不是疼痛，而是一种……指向。那搏动的力量，隐隐指向中间那条最黑暗的通道。

是体内的微粒残留对某种东西产生了反应？还是说，这条通道深处，有与这残留相关的事物？

他无法判断这是陷阱还是指引。但体内的异动是此刻唯一明确的“线索”。他咬了咬牙，压下翻涌的不适和更深的警惕，迈步走向中间的通道。

一踏入其中，温度骤降。不是物理上的寒冷，而是一种沁入灵魂的阴冷。黑暗浓稠得如同液体，包裹着身体，挤压着感官。连那微弱的磷光也彻底消失了，他只能依靠远超常人的黑暗视觉和皮肤的感知来前进。脚下的琉璃地面变得湿滑，覆盖着一层薄薄的、冰冷的黏液。通道似乎变得更加狭窄，岩壁几乎要擦到他的肩膀，上面布满了湿漉漉的、类似苔藓但更粘稠的附着物。

他走得很慢，每一步都小心翼翼。影迹潜行的能力被提升到当前能维持的极限，他的身影几乎完全消失在黑暗里，只剩下最轻微的气息流动。

又前行了数十米，通道似乎到了尽头。前方是一个相对开阔的空间，隐约有极其暗淡的、非自然的光源在闪烁。他贴在通道出口的岩壁边缘，向内窥视。

那是一个不大的石室，呈不规则的圆形。石室中央的地面上，刻画着一个复杂的、直径约三米的圆形阵列。阵列的线条由某种暗银色的物质镶嵌而成，在绝对的黑暗中散发着微弱的、冰冷的荧光。阵列的图案与他短刃上的古老符文有某种相似的气质，但更加庞大、系统，也似乎更加……破损。许多线条已经断裂，镶嵌物质剥落，整个阵列显得残缺不全。

而在阵列的中心，匍匐着一具……东西。

那东西有着类似人类的轮廓，但体型异常高大、佝偻。它的皮肤是灰败的、仿佛被抽干了水分的皮革，紧紧包裹着嶙峋的骨骼。四肢细长，关节反向扭曲，指尖是尖锐的、乌黑的钩爪。它的头颅低垂，看不清面目，只能看到稀疏的、如同干枯水草般的灰白色毛发，以及从后颈延伸出的、几根扭曲的、仿佛骨质增生般的尖刺。

它一动不动，仿佛早已死去多时。

但石室内弥漫的那股浓烈到令人作呕的腐败甜腥味，源头正是它。而且，阵列上那些微弱的荧光，似乎正以一种极其缓慢的速度，流向那具躯体，如同细小的溪流汇入干涸的河床。

掠夺。

这个词毫无征兆地浮现。不是他在掠夺，而是这个环境，这个阵列，或者这个空间本身，在掠夺着某种东西，然后注入那具躯体？还是说，是这具躯体在通过阵列，掠夺着什么？

他屏住呼吸，目光锐利如刀，扫过石室的每一个角落。除了中央的阵列和那具诡异的躯体，石室边缘散落着一些东西：几块破损的、像是陶罐的碎片；几根颜色暗沉、疑似骨骼的条状物；还有……一小堆极其微小的、闪烁着黯淡金属光泽的颗粒，聚集在阵列边缘的一个凹陷处。

那些颗粒非常小，比沙砾还要细微，但它们散发出的气息……与他体内的微粒残留，有着某种同源但更加“纯粹”的感觉。它们静静地躺在那里，仿佛是被遗弃的、无用的残渣。

**微粒**。

这个世界的通货。总量有着硬上限的、840000000的终极资源。知识再次浮现，冰冷而客观。那些颗粒，就是微粒。如此稀少，如此黯淡，却蕴含着……力量？或者说，是某种构成这个黑暗世界基础的“物质”？

他体内的钝痛，在看到那些微粒的瞬间，似乎轻微地悸动了一下。不是渴望，更像是一种……共鸣，或者排斥？

他的目光回到阵列中央的躯体上。它依旧一动不动。是死物？还是沉睡？如果是沉睡，是什么在维持它？那些流入它体内的荧光是什么？与微粒有关吗？

他需要更近一些观察，或许能从那具躯体上，或者从那个破损的阵列上，找到关于这个地方，关于微粒，甚至关于他自身状态的线索。

他像一片真正的影子，贴着石室冰冷的岩壁，无声无息地向内滑入。影迹潜行发挥到极致，他甚至能感觉到自己的体温、心跳、乃至生命气息都被压缩到近乎虚无。他绕过散落的碎片和骨骼，避开阵列上那些尚且完好的、散发荧光的线条，如同最谨慎的捕食者，缓缓靠近阵列的边缘，靠近那堆黯淡的微粒。

距离那堆微粒还有三步之遥时，异变陡生。

阵列中心，那具一直匍匐不动的躯体，猛地抬起了头！

没有眼睛。本该是眼眶的位置，只有两个深不见底的、蠕动着粘稠黑暗的窟窿。它的嘴巴裂开，一直延伸到耳根，露出里面层层叠叠的、螺旋状的、尖锐的利齿。没有声音发出，但一股冰冷、暴虐、充满纯粹饥饿感的意念，如同实质的冲击波，瞬间扫过整个石室！

被发现了！

不是通过视觉、听觉或嗅觉。而是通过某种更加本质的感知——或许是对生命气息，或许是对空间扰动，或许……是对他体内那“微粒残留”的异样波动！

几乎在那股意念扫过的同时，他的身体已经做出了反应。没有思考，没有犹豫，纯粹是千锤百炼（即使记忆缺失）的战斗本能。他猛地向后疾退，同时右手闪电般探向腰间的短刃。

那具躯体——现在可以称之为怪物——动了。它的动作快得违反常理，四肢以扭曲的角度在地面一撑，整个身体如同被弹射出的枯骨，带着一股腥风，直扑他刚才所在的位置！钩爪划过空气，带起凄厉的尖啸，狠狠抓在岩壁上，留下数道深深的、冒着丝丝黑烟的沟壑。

退后中的他，瞳孔骤缩。这速度，这力量……远超寻常。而且，在怪物扑击的瞬间，他清晰地看到，怪物那干瘪的胸膛内部，隐约有一点极其微弱的、暗红色的光芒在跳动，频率……与他胸口的异样搏动，有瞬间的同步！

这怪物体内，也有类似的微粒残留？或者说，是更完整、更活跃的“微粒”存在？

没有时间细想。怪物一击不中，几乎没有停顿，那没有眼睛的“脸”准确地对准了他疾退的方向，四肢再次发力，以更快的速度扑来！它的动作带着一种野兽般的凶残，却又有着精准的、近乎预判般的轨迹，封死了他左右闪避的空间。

狭窄的石室，无处可逃。

凶性，在绝境中被点燃。那并非愤怒，而是一种冰冷的、剔除了所有杂念的杀戮本能。存在性恐惧被压下，对未知的迷茫被暂时遗忘，只剩下最原始、最直接的诉求：杀死对方，或者被对方杀死。

他不再后退。

在怪物扑至面前的刹那，他脚步骤停，身体以毫厘之差侧滑，险之又险地避开了直取咽喉的钩爪。怪物的利爪擦着他的肩头掠过，带起几片破碎的衣料和一丝血线。与此同时，他手中的黑色短刃，如同毒蛇出洞，悄无声息地刺向怪物左侧肋下——那里，是刚才惊鸿一瞥中，那点暗红光芒隐约所在的位置。

短刃上的古老符文，在刺出的瞬间，似乎微微亮了一下，不是光芒，而是一种更深沉的“暗”的流转。刃尖毫无阻碍地没入了怪物灰败的皮肤。

没有刺入血肉的感觉，更像是刺入了一团凝固的、充满韧性的油脂。阻力极大。

怪物发出一声无声的、却直接冲击灵魂的尖啸！它猛地扭身，另一只钩爪横扫而来，速度更快，力量更猛，爪尖甚至带起了肉眼可见的、扭曲空气的黑色涟漪。

他来不及拔出短刃，只能松手，身体极力后仰，同时左臂格挡。

“嗤啦！”

钩爪划过他的左臂，坚韧的衣物如同纸糊般撕裂，三道深可见骨、皮肉翻卷的伤口出现。鲜血涌出，但流出的血液颜色……竟然带着一丝不正常的暗金，并且在接触到空气中弥漫的腐败甜腥味时，发出轻微的“滋滋”声，仿佛在被腐蚀。

剧痛传来，但他眉头都没皱一下。借着怪物横扫的力量，他顺势向后翻滚，拉开距离，半跪在地，右手再次摸向腰侧——刀鞘已空。

怪物踉跄了一下，左肋下插着那柄黑色短刃。暗红色的、粘稠如沥青的液体从伤口渗出，滴落在阵列的线条上，发出“嗤嗤”的声响，竟然将那暗银色的荧光物质腐蚀出一个个小坑。怪物体内的那点暗红光芒剧烈地闪烁、明灭，仿佛极不稳定。它的动作明显迟缓、僵硬了许多，那股暴虐的意念也出现了紊乱。

短刃有效！不仅造成了物理伤害，似乎更干扰了它体内那种与微粒相关的能量运转！

机会！

他顾不上左臂火辣辣的疼痛和血液的异常，目光锁定怪物。影迹潜行的能力在激烈的运动中难以维持，但他此刻也不需要隐藏。他需要的是速度，是致命一击。

怪物似乎也意识到了短刃的威胁，它伸出钩爪，试图去拔掉肋下的刀刃。

就在它分神的瞬间，他动了。没有武器，他便将自己化为武器。脚下一蹬，地面琉璃炸裂，他整个人如同离弦之箭，直射怪物！目标明确——怪物那没有眼睛的头颅。

怪物察觉，放弃拔刀，抬起另一只完好的前肢，利爪直插他的胸膛，竟是以伤换命的打法！

电光石火间，他在空中强行拧身，险险避开心脏要害，任由那钩爪刺入自己的右胸偏上的位置，穿透！冰冷的、带着强烈腐蚀感的异物感瞬间充斥胸腔。而他蓄满力量的右手，并指如刀，凝聚了此刻所能调动的、源自身体深处某种微弱却锐利的气息（是影迹潜行的能量？还是别的什么？），狠狠地插入了怪物那蠕动着黑暗的眼眶！

“噗嗤！”

手感怪异，像是捅破了一个装满粘液的皮囊。一股冰冷、污秽、充满疯狂碎片的意念顺着他的手臂逆冲而上，试图污染他的精神。他闷哼一声，强忍着脑海中的刺痛和眩晕，手腕猛地一拧，一搅！

怪物的动作僵住了。

插在他胸口的钩爪，力量迅速消退。怪物体内那点暗红光芒，如同风中残烛，剧烈闪烁了几下，骤然熄灭。

随即，怪物的躯体开始崩溃。不是倒下，而是从内部瓦解，灰败的皮肤寸寸龟裂，露出下面空荡荡的、如同被彻底蛀空的朽木般的内部结构。骨骼化为齑粉，血肉蒸腾成一股浓烈的、令人作呕的黑烟。短短几个呼吸，原地只剩下一小堆灰烬，以及……三粒比之前那堆黯淡微粒稍大一些、光泽也稍微明亮一丝的暗红色晶粒。

那柄黑色短刃，“当啷”一声掉落在灰烬旁，刃身依旧黝黑，符文沉寂，只是刃口处沾染的暗红色粘液，正缓缓滑落，滴入灰烬，消失不见。

他踉跄后退，终于支撑不住，单膝跪倒在地。右胸的贯穿伤传来火烧火燎的剧痛，左臂的伤口也在不断渗血，暗金色的血液与灰烬、黏液混合，发出令人牙酸的声音。更糟糕的是，体内那原本缓慢的钝痛，在经历了刚才激烈的战斗和受伤后，仿佛被刺激到了，开始变得尖锐、活跃，如同有活物在血管里挣扎、撕咬。胸口的暗红色血管纹路，颜色似乎加深了一些，并向周围蔓延了细微的几丝。
