# 第2章 · 黑雾中的刻痕

痛。

是那种从骨髓深处渗出来的、绵密而冰冷的痛。它不像刀刃划开皮肉那般尖锐分明，而是像无数条细不可见的虫子，沿着血管与神经的路径，缓慢地、持续地啃噬着。每一次心跳，都将这种啃噬的痛楚泵向四肢百骸。

他睁开眼。

视野里没有光，只有一片浓得化不开的灰黑。空气黏稠、滞重，带着腐朽的金属和某种难以言喻的有机质腐烂混合的气味，沉甸甸地压在口鼻之上。他试图移动，身体却像被无数无形的锁链捆缚，每一块肌肉都僵硬、酸痛，仿佛已经在这里躺了……不知道多久。

记忆是一片空无的废墟。我是谁？我在哪？为什么会在这里？

只有本能，像野兽般尖锐的本能，在空白的意识里率先亮起獠牙——警惕。他强迫自己放缓呼吸，让几乎要冲出喉咙的惊悸与恐慌沉淀下去。耳畔只有自己压抑的心跳，以及更远处，某种极其细微的、仿佛金属摩擦又像砂砾滚动的声响，断断续续，在绝对的死寂中显得格外刺耳。

他动了动手指。触感传来，冰冷、粗糙，是某种岩石或金属板。他缓缓撑起上半身，骨骼发出不堪重负的呻吟。视线逐渐适应了这片绝对的黑暗，勉强能分辨出一些模糊的轮廓。他似乎在一个狭窄的通道里，两侧是布满锈蚀与未知污渍的金属墙壁，向上看，看不到顶，只有更深沉的黑暗。

他低头，看向自己的身体。

衣衫褴褛，沾满干涸的、颜色可疑的污迹。裸露的皮肤上，布满了细密的、新旧交叠的疤痕，像一张无声诉说着过往残酷的地图。而在他的右手腕内侧，皮肤之下，有什么东西在微微发光。

不是光。更准确地说，是一种“存在感”。一点极其微小的、幽蓝色的光粒，嵌在皮肉与血管之间，缓慢地、有规律地脉动着。每一次脉动，都伴随着那股源自骨髓的啃噬痛楚。它不属于他。他能清晰地“感觉”到这一点，一种异物入侵骨髓、扎根灵魂的冰冷排斥感。这东西……是什么？

他伸出左手，颤抖着想要触碰那幽蓝的光点。指尖还未触及，一股强烈的、几乎要撕裂灵魂的剧痛猛地从手腕炸开，瞬间席卷全身！他闷哼一声，蜷缩起来，牙齿死死咬住，额头上冷汗涔涔。那痛楚并非纯粹的生理伤害，更像是一种……警告？或者，是某种更深层次的、触及存在本质的侵蚀。

过了许久，剧痛才如潮水般退去，留下更深的疲惫与寒意。他喘息着，靠在冰冷的墙壁上。那幽蓝的光点依旧在脉动，像一颗寄生在他体内的、冰冷的心脏。

就在这时，一种近乎肌肉记忆般的反应发生了。他的右手，未经任何思考，便闪电般探向腰间——那里，有一个简陋的皮质刀鞘。入手冰凉、坚硬。他抽出它。

一柄短刃。

长度不过小臂，刃身呈现出一种历经岁月磨洗的暗沉色泽，并非纯粹的金属光泽，更像某种骨质或奇异矿石的质感。刃口并不显得特别锋利，甚至有些地方带着细微的崩缺。真正吸引他目光的，是刀身上那些深深镌刻的符文。

那不是任何他“记得”的文字或图案。线条扭曲、繁复，带着一种古老而蛮荒的气息，仿佛不是被人刻上去的，而是从刀刃本身的“存在”中生长出来的。指尖抚过那些凹痕，触感冰凉刺骨，却又隐隐感到一丝极其微弱的、如同脉搏般的悸动，与手腕内那幽蓝光点的冰冷脉动截然不同。这悸动……很淡，几乎难以察觉，却奇异地让他狂跳的心脏稍稍平复了一丝。

短刃在手，一种莫名的、仿佛与生俱来的熟悉感涌了上来。不是记忆，是手感，是重量在掌中分布的平衡，是五指收拢时刀柄上那些细微磨损处与指腹的贴合。仿佛这柄刀，曾是他肢体延伸的一部分。

他握着刀，背靠墙壁，缓慢地站了起来。双腿虚浮，但勉强能支撑身体。失忆带来的不仅是空白的恐惧，还有一种剥离了所有社会身份与道德束缚后，赤裸裸的、仅剩生存本能的“警觉”。他不信任这里，不信任这具身体里陌生的痛楚，不信任手中这柄来历不明的刀，更不信任这片吞噬一切的黑暗。

他需要离开这里。需要知道发生了什么。需要弄明白手腕里那该死的东西是什么。

他深吸一口那污浊的空气，开始沿着狭窄的通道，向那隐约传来细微声响的方向，极其缓慢地移动。每一步都小心翼翼，落脚无声——这是一种近乎本能的技巧，身体仿佛记得如何将重量均匀分散，如何利用阴影和墙壁的夹角来隐藏身形。影迹潜行。这个词毫无征兆地浮现在空白的意识里，像一颗投入死水潭的石子。

通道并非笔直，时有岔路，但主路似乎一直向下倾斜。墙壁上的锈蚀越来越严重，开始出现大片大片剥落的痕迹，露出下面更加晦暗的材质。空气中那股腐烂的气味越来越浓，还混合了一丝……甜腥？是血的味道，但又不完全是，更陈旧，更……诡异。

前方传来清晰的“咔嚓”声，像是某种脆硬的东西被踩碎了。

他立刻停下，身体紧贴墙壁凹处，呼吸屏住，整个人仿佛融入了那片阴影。手腕内的幽蓝光点似乎也感应到了什么，脉动加快了一丝，带来一阵细微的刺痛。

片刻后，一个身影摇摇晃晃地从拐角处“走”了出来。

那已经不能称之为“人”了。

它穿着破烂的、类似某种制服的衣服，但布料早已朽烂，与下面溃烂的皮肉黏连在一起。裸露在外的皮肤呈现出一种不自然的青灰色，布满脓疱和溃烂的伤口。它的动作僵硬而怪异，关节仿佛生锈的铰链，每动一下都发出令人牙酸的摩擦声。最令人心悸的是它的头部——半边脸似乎被什么强酸或火焰严重腐蚀，露出下面黑红色的肌肉和森白的颧骨，一只眼球浑浊地挂在眼眶外，另一只则闪烁着一种非人的、空洞的幽绿色微光。

它漫无目的地移动着，喉咙里发出“嗬嗬”的、仿佛破风箱般的声音。它经过他藏身的阴影时，似乎停顿了一下，那只完好的、泛着绿光的眼睛转向他的方向。

他握紧了短刃，指节发白。身体里的每一根弦都绷紧了，不是恐惧，而是一种更原始、更冰冷的狩猎与被狩猎的紧张。那东西……是活的？还是死的？

腐烂的“人”歪了歪头，似乎没发现什么，又继续摇摇晃晃地向前走去，消失在另一条岔路的黑暗中。空气中留下更浓的甜腥腐臭。

他缓缓吐出一口浊气，后背的衣衫已被冷汗浸透。那是什么？守卫？遗迹里的“居民”？还是……和他一样的“迷失者”，只不过变成了那副模样？手腕内的幽蓝光点，与那东西眼中的绿光，是否有关联？

他不敢细想，只是将警惕提到了最高。这里比他想象的更危险。

继续前行。通道开始变得开阔，但环境也更加破败。地上开始出现散落的碎片——金属零件、破碎的容器、还有一些难以辨认的、类似骨骼的残骸。墙壁上开始出现一些模糊的壁画或刻痕，但大多已被岁月和某种力量侵蚀得难以辨认。偶尔能看到一些扭曲的符号，与短刃上的古老符文有某种神似，却又似是而非。

他蹲下身，捡起一块相对完整的金属板碎片。碎片边缘锋利，表面覆盖着厚厚的污垢。他用袖子擦了擦，隐约看到下面蚀刻着一些文字。不是符文，更像是某种记录。他努力辨认，几个破碎的词组勉强拼凑出来：

“……天堑……不可逾越……”

“……微粒总量……恒定……”

“……内宇宙……崩解……”

每一个词都像一把冰冷的钥匙，试图打开他记忆里锈死的锁，却只换来一阵剧烈的头痛和更深的茫然。天堑？微粒？内宇宙？这些词带着沉甸甸的分量，仿佛触及了这个黑暗世界的某种根本规则，但他却无法理解它们的含义。唯一清晰的是，这些词与他手腕内的异物，与他所处的这个诡异地方，必然有着深刻的联系。

他扔掉碎片，正要起身，眼角余光瞥见前方不远处的地面上，有什么东西在极其微弱地反光。

他悄无声息地靠近。

那是几颗极其微小的、半透明的晶体，散落在灰尘和碎屑中。大小不一，最大的也不过米粒般，最小的则细如尘埃。它们本身并不发光，只是反射着不知从何处渗来的、极其稀薄的微光，呈现出一种黯淡的、近乎无色的质感。但当他凝视它们时，手腕内的幽蓝光点猛地剧烈跳动起来！一股强烈的、近乎贪婪的“渴望”从那光点中传来，顺着神经直冲他的大脑，几乎要控制他的身体扑过去。

他强行压制住这股冲动，额头上青筋暴起。这就是……“微粒”？那些刻痕里提到的“微粒”？这个世界的“通货”？他隐约感觉，吸收这些微粒，或许能缓解手腕内的痛苦，甚至……获得力量？

但本能再次发出尖锐的警告。未知。危险。那幽蓝光点的“渴望”本身，就充满了不祥。他不敢贸然触碰。

他移开目光，强迫自己不去看那些晶体。视线扫过周围，落在不远处一具倚靠在墙边的骸骨上。那骸骨早已风化，衣物也几乎烂尽，但骸骨的右手，却紧紧攥着什么。他走过去，用短刃小心翼翼地拨开那枯朽的手指。

“叮。”

一声极其轻微的脆响，一颗比地上那些都要大一些、颜色也更加深邃（呈现出一种极淡的灰色）的微粒，从骸骨指间滚落在地。与此同时，那骸骨的头颅似乎微微动了一下，空洞的眼眶仿佛“看”了他一眼，随即彻底散架，化为一堆灰白的尘埃。

他捡起那颗灰色微粒。入手冰凉，没有任何能量波动的感觉，轻若无物。但当他将其握在掌心时，手腕内的幽蓝光点骤然变得灼热，脉动疯狂加速，那股渴望几乎要化为实质的疼痛。他死死攥着微粒，指缝间却没有任何变化。他无法吸收它。或者说，他体内的那个异物，无法直接从这个状态获取它。

他需要……方法。知识。或者，某种“许可”。

他将灰色微粒小心地放入腰间一个尚未完全破损的小皮囊中——这皮囊似乎是原本就挂在他身上的。然后，他看向那堆骸骨化成的尘埃。尘埃中，似乎有一点金属的反光。他用刀尖拨开，发现那是一块小小的、边缘不规则的金属牌，上面刻着一个编号：**7749**。

或许，这曾是一个名字。

他将金属牌也收起。就在这时，通道深处，更远的地方，传来一阵截然不同的声响。

那是金属靴底踩踏地面的规律声响，整齐，有力，带着一种冰冷的秩序感，与之前那腐烂行尸的拖沓截然不同。同时，还有低沉的、经过某种装置处理的对话声断断续续传来，伴随着一种独特的、仿佛蜂鸣般的能量波动。

“……第三清扫区……残留反应……回收……”

“……注意‘游荡者’……尤其是‘携带者’……”

“……微粒痕迹……向B-7通道集中……”

声音在空旷的通道中产生轻微的回响，越来越近。

携带者？是在说他吗？因为他手腕里的东西？还是因为他捡到的微粒？

没有任何犹豫，他像受惊的阴影般向后疾退，重新没入来时的狭窄通道。那些规律的脚步声和冰冷的对话声，比腐烂的行尸更让他感到致命的威胁。那是一种组织性的、带有明确目的的威胁。

他退到一处相对隐蔽的岔道凹角，蜷缩起来，将呼吸和心跳压制到最低，手中的短刃横在身前，刃身上那些古老的符文在绝对的黑暗中，仿佛自行吸收着周围稀薄的微光，流转着更晦暗的色泽。他闭上眼，不再用视觉，而是用全部的感觉去捕捉外界的动静。

脚步声在主干道交汇处停下。

“痕迹到这里分散了。”一个冰冷的声音说。

“有‘基础微粒’被移动的痕迹，量很少。不是自然消散。”另一个声音回应，带着某种扫描设备的轻微嗡鸣，“检测到微弱的‘异常微粒’残留信号……型号无法匹配数据库。新型号？还是……‘污染变体’？”

“提高警戒。如果是‘污染变体携带者’，优先执行‘净化’协议。注意，目标可能具有潜行类能力。”

“明白。投放‘显迹尘’。”

一阵轻微的喷射声，随后，某种极其细微的、泛着惨绿色荧光的粉尘，开始顺着气流在通道中弥漫开来。

他屏住呼吸，连眼皮都不敢动一下。手腕内的幽蓝光点，似乎也感应到了外界的扫描和那弥漫的粉尘，脉动变得极其缓慢而微弱，仿佛进入了某种休眠伪装状态，但那深入骨髓的冰冷与刺痛，却更加清晰。

时间在死寂中缓慢流逝。每一秒都像在刀尖上煎熬。

不知过了多久，那规律的脚步声再次响起，似乎朝着另一个方向远去，逐渐消失在通道深处。惨绿色的荧光粉尘也慢慢沉降、消散。

他又等了很久，直到周围彻底恢复死寂，才缓缓吐出一口带着铁锈味的浊气。身体因为长时间的紧绷而微微颤抖。他低头，看向手腕。那幽蓝的光点依旧在，像一颗嵌入生命的毒种。

他看向手中的短刃，那些古老的符文沉默着。看向腰间皮囊里那颗冰冷的灰色微粒。看向这具布满伤痕、承载着异物、却丢失了所有过往的躯体。

我是谁？

这个问题不再仅仅是空白的恐慌，而是混合了异物侵蚀的痛楚、黑暗遗迹的威胁、神秘追猎者的阴影，以及手中这柄唯一熟悉的短刃所带来的、一丝微弱却坚韧的“存在证明”。

他必须活下去。必须弄清楚这一切。必须找到方法，要么剥离这该死的幽蓝光点，要么……掌控它。

他撑着墙壁，再次站起，目光投向通道更深处，那片更加浓稠、仿佛孕育着一切答案与终极恐怖的黑暗。

第一步，是离开这个即将被“清扫”的区域。

他迈开脚步，将身影再次融入阴影，向着与那些追猎者相反的方向，无声潜行而去。手腕内的幽蓝，与短刃上的暗痕，在绝对的黑暗里，明灭不定。
