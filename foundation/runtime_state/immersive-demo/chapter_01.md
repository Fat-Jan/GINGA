# 第一章 · 黑雾初醒

他醒来时，嘴里全是铁锈味。

那不是真正的铁锈，是血，干涸后凝结在舌苔与齿缝间的腥甜粘稠。他睁开眼，视野里没有光，只有一片浓稠的、仿佛能用手捧起来的黑暗。空气湿冷，带着腐朽的泥土和某种更深的、类似腐烂内脏的甜腻气息。他躺在地上，身下是粗糙不平的石面，棱角硌着脊椎，带来钝痛。

记忆是空的。

不，不是空，是破碎。碎片在脑海里沉浮，无法拼凑成形。他记得……影。长而扭曲的影子，在墙壁上爬行。记得刀刃切入皮肉时的滞涩感，记得温热液体溅在脸上的触感。记得一个名字，或者代号？没有。只有一片虚无的恐慌，像冰冷的潮水从脊椎底部漫上来。

他试图坐起，肌肉传来撕裂般的抗议。左臂尤其沉重，从肩胛到指尖都麻木着，仿佛不属于自己。他用右手撑地，缓慢地、一点一点地直起上半身。黑暗并非绝对，远处有极其微弱的光，是某种青灰色的、仿佛来自地底深处的冷光，勉强勾勒出轮廓。

这是一个通道。

狭窄，高耸。两侧石壁相距不过三步，向上延伸，隐没在头顶更深沉的黑暗里。石壁表面粗糙，布满凿痕，但那些凿痕……不对。他凑近，用还能活动的右手去触摸。指尖触到的不是简单的刻痕，而是某种纹路，细密、繁复，像是文字，又像是某种生物的脉络。它们蜿蜒交错，在冰冷的石面上构成一幅幅扭曲的图案，看得久了，眼睛会刺痛，仿佛那些纹路在蠕动。

他收回手，低头看自己。

身上是一件破烂的黑色紧身衣，多处撕裂，露出下面苍白的皮肤和深可见骨的伤口。伤口边缘泛着不正常的紫黑色，没有流血，但皮肉微微外翻，像被什么东西腐蚀过。最触目惊心的是左臂——从肩膀到手肘，皮肤下布满了细密的、发着幽蓝色微光的纹路。它们像活物一样，随着他心脏的跳动，极轻微地脉动。

微粒残留。

这个词毫无征兆地跳进脑海，带着冰冷的、不容置疑的确定性。他盯着那些幽蓝纹路，一种本能的厌恶和恐惧涌上来。这不是治疗，不是祝福。这是污染，是某种东西强行注入他体内的痕迹。他试图调动左臂，手指勉强蜷缩了一下，随之而来的是一阵尖锐的、直达骨髓的刺痛，仿佛有无数细针在血管里搅动。

他闷哼一声，额头上渗出冷汗。

不能停留。

这个念头清晰而强烈。黑暗的通道深处，有什么东西在动。不是声音，是感觉，一种被窥视的、毛骨悚然的感觉，像冰冷的蛛丝拂过后颈。他强迫自己站起来，双腿颤抖，但支撑住了。他检查身上——除了破烂的衣服，腰间还挂着一件东西。

一把短刃。

刀鞘是某种暗沉的金属，入手冰凉沉重。他拔出刀，刀刃在微弱的冷光下泛着哑光的黑，没有反光，仿佛能吸收光线。刀身靠近护手处，刻着密密麻麻的符文。那些符文与石壁上的纹路有某种相似的神韵，但更加古老、更加简洁，每一个笔画都透着一种沉甸甸的、近乎暴戾的意味。他看不懂，但指尖抚过时，能感觉到微微的温热，与周围环境的冰冷格格不入。

刻有古老符文的短刃。

这是他唯一的武器。也是他唯一能抓住的、与“过去”可能有关的实物。他将短刃握紧，冰凉的刀柄带来一丝奇异的安全感。影迹潜行——另一个词浮现。他心念微动，身体似乎本能地想要融入阴影，但左臂的剧痛和体内那种陌生的滞涩感打断了他。技能还在，但被身体的状态严重干扰了。

他深吸一口气，压下翻腾的恶心和眩晕，开始沿着通道向前走。

脚步声在狭窄的空间里回荡，被石壁反复折射，变得诡异而空洞。青灰色的冷光来自镶嵌在石壁高处的一些晶体，它们不规则地分布着，光线微弱且不稳定，时而明灭，让通道里的影子不断扭曲变形。空气越来越潮湿，甜腻的腐臭味也越来越浓。地面开始出现积水，暗红色的，粘稠，踩上去发出“咕唧”的声响。

他走得很慢，每一步都全神贯注。失忆带来的空白并没有剥夺他的本能——对危险的嗅觉，对环境的观察，对自身极限的评估。这些像烙印在骨头里的技艺，随着呼吸自然流淌。他注意到石壁上的纹路开始变化，从无序的蜿蜒，逐渐汇聚，指向通道深处。那些纹路在发光，极其微弱，是比周围冷光更深的暗红色，像干涸的血。

通道开始向下倾斜。

坡度不大，但能感觉到。脚下的积水变深，没过了脚踝。水是温的，带着令人不适的滑腻感。他低头，借着微光，看到水中有细小的、半透明的絮状物在漂浮，随着他的动作缠绕上小腿。

然后，他听到了声音。

不是来自前方，也不是后方。是来自……脚下？不，是来自石壁。极其细微的摩擦声，沙沙作响，像无数细小的脚在爬行。他停下，屏住呼吸，将短刃横在胸前。

声音停了。

寂静。只有水滴从高处落下，砸进水面的嘀嗒声。

他等了十个心跳的时间，声音没有再出现。他继续向前，更加谨慎。通道在这里变得略微宽敞了一些，出现了一个不规则的岔口。主通道继续向下延伸，而左侧有一条更窄的、几乎被坍塌的碎石堵住一半的缝隙，里面有风，微弱但持续的风，带着一丝……相对新鲜的、不那么腐臭的气息。

选择。

他犹豫了不到一息。主通道的深处，那种被窥视的感觉浓烈得几乎化为实质。而缝隙后的风，意味着可能的出口，或者至少是空气流通的地方。他侧身，挤进那条狭窄的缝隙。

碎石锋利，刮擦着他的伤口，带来新的疼痛。他咬紧牙关，一点一点向内挪动。缝隙比他预想的要长，而且曲折。黑暗中，他全靠触觉和那一点微弱的风向来判断方向。左臂的幽蓝纹路在绝对的黑暗里发出更清晰的光，照亮前方一小片区域，也照亮了他苍白汗湿的脸。

大约挤了二十几步，前方豁然开朗。

是一个不大的石室，天然形成的，洞顶垂下许多钟乳石。石室中央有一小潭水，水是清澈的，泛着淡淡的蓝色荧光，光源来自水底铺着的细碎晶体。水潭边，散落着一些东西。

几具骸骨。

不是完整的骨架，是散乱的，像是被暴力拆解后随意丢弃。骨骼的颜色很奇怪，不是白骨，而是泛着灰败的金属光泽，有些部位甚至呈现出结晶化的质感。骸骨之间，散落着一些破损的衣物碎片和零碎的物品——几个锈蚀的金属水壶，几段断裂的绳索，还有……几颗拇指大小、不规则的多面体晶体。

那些晶体是半透明的，内部仿佛有雾气在缓缓流转，颜色各异，但大多黯淡无光。只有一颗，落在离水潭最近的一具头骨眼眶里，散发着微弱的、稳定的淡金色光芒。

微粒。

这个词再次撞击他的意识。这就是这个世界的通货，硬上限八亿四千万的绝对资源，修炼、交易、生存的根基。它就在那里，唾手可得。

他没有立刻上前。

骸骨的姿态，散落的方式，以及那些结晶化的骨骼，都透着诡异。他蹲下身，仔细观察地面。石室地面覆盖着一层薄薄的灰尘，但在水潭周围，灰尘有被扰动过的痕迹，不是脚印，而是某种……拖曳的痕迹，宽而平滑，从水潭边一直延伸到石室另一侧的阴影里。

那里，石壁底部，有一个不起眼的孔洞，只有碗口大小，深不见底。

他握紧短刃，缓缓靠近那颗淡金色的微粒。每一步都踩得极轻。距离还有三步时，他停了下来。左臂的幽蓝纹路突然剧烈地脉动了一下，刺痛感加剧。与此同时，他听到了一种声音。

一种极其细微的、高频的震颤声，从那个孔洞里传来。

咻——

不是声音先到。

是一道黑影，细长，迅捷如电，从孔洞里激射而出，直扑他面门！他根本来不及思考，身体已经做出反应——侧身，拧腰，短刃由下向上撩起！动作因为左臂的拖累慢了半分，刀刃擦着那东西的边缘划过。

“嗤！”

一声轻响，伴随着几滴冰冷的液体溅在他脸上。那东西落在不远处的地上，扭曲翻滚。他这才看清，那是一条……虫子？大约手臂长短，身体呈环节状，没有眼睛，前端是一个圆形的、布满细密利齿的口器。身体是暗红色的，与他之前看到的石壁纹路颜色相同。被短刃划开的地方，流出粘稠的、散发甜腻腐臭的暗红色体液。

虫子受伤，发出更加尖锐的震颤声，猛地弹起，再次扑来！这次速度更快，口器大张，直咬他的咽喉！

凶性在瞬间被点燃。

恐惧被压下去，取而代之的是一种冰冷的、近乎暴戾的专注。失忆的空白里，杀戮的本能汹涌而出。他没有后退，反而迎了上去，在虫子即将咬中的瞬间，身体以不可思议的角度扭转，短刃反握，自左向右，狠狠横斩！

刀刃上的古老符文，似乎亮了一下。

“噗！”

刀身毫无阻滞地切入了虫子的身体中段，将其斩为两截！两截残躯落地，疯狂扭动，体液四溅。但危机没有解除，孔洞里，更多的震颤声响起，密密麻麻，汇成一片令人头皮发麻的嗡鸣！

不止一条！

他瞳孔收缩，毫不犹豫地向前扑去，目标就是那颗淡金色微粒！左手不顾剧痛，猛地探出，一把将那颗从头骨眼眶里抠出！晶体入手温润，内部的金色光芒似乎微微活跃了一丝。

几乎在他拿到微粒的同一瞬间，至少七八条同样的暗红虫子从孔洞里蜂拥而出，它们没有立刻攻击，而是迅速散开，呈半圆形将他围住，口器开合，发出威胁的震颤。它们的身体表面，那些暗红色的纹路开始发光，与石壁上的纹路隐隐呼应。

个体存续 vs 跨天堑生态。

这些虫子，就是这黑雾遗迹底层生态的一部分。它们守护着这里的微粒，或者说，以这里的某种东西为生，而微粒是副产品？他没时间细想。被包围了，退路是那条狭窄的缝隙，速度不够快，会被追上咬死。

只能杀出去。

他深吸一口气，将短刃换到相对灵活的右手，左手紧紧攥着那颗淡金色微粒。微粒接触皮肤，一股微弱但清晰的暖流顺着手臂蔓延，左臂那幽蓝纹路带来的刺痛，竟然被稍稍压制了一瞬。

虫子动了。

三条从正面扑来，两条从左右两侧包抄，还有两条高高跃起，封堵上方！配合默契，俨然捕猎阵型。

他动了。

没有使用花哨的技巧，只有最简洁、最致命的移动和劈砍。脚步在方寸间腾挪，身体贴着虫子的攻击轨迹闪避，每一次闪避都伴随着一次凌厉的反击。短刃在他手中仿佛活了过来，划出一道道黑色的轨迹。刀刃上的符文持续散发着微光，每一次斩击，都能轻易破开虫子坚韧的外皮。

嗤！噗！咔嚓！

粘液飞溅，残肢断体乱飞。甜腻的腐臭味浓烈到令人作呕。一条虫子咬住了他的小腿，利齿深深嵌入皮肉。他闷哼一声，反手一刀将虫子钉死在地上，用力拔出，带出一蓬血花。伤口火辣辣地疼，但更麻烦的是，虫子的体液似乎有麻痹效果，小腿开始发麻。

不能停。

他像一头被困的受伤野兽，在石室中左冲右突，短刃每一次挥动都带着必杀的决心。影迹潜行的本能偶尔闪现，让他的身影在微光中变得模糊，误导虫子的攻击。但数量太多了，孔洞里还在源源不断地涌出。

必须离开！

他看准一个空隙，猛地向缝隙入口冲去！两条虫子拦截，被他挥刀斩开。第三条从侧面撞来，他来不及完全躲闪，被撞在腰侧，踉跄了一步，喉头一甜。他强行咽下，借着冲势，扑进了狭窄的缝隙！

身后的震颤声瞬间变得狂暴。虫子追了上来，但它们庞大的身躯在缝隙里行动受限，速度慢了下来。他手脚并用，不顾一切地向外爬，碎石和虫子的利齿在身后刮擦、碰撞。

光明——主通道那青灰色的冷光——出现在前方。

他奋力一挣，从缝隙里滚了出来，重重摔在主通道的积水里。他立刻翻身，背靠石壁，短刃指向缝隙出口。

震颤声在缝隙里响了一会儿，渐渐停息。那些虫子没有追出来，似乎它们的活动范围仅限于石室和那条通道。

他剧烈地喘息着，浑身湿透，分不清是汗水、积水还是血水。左臂的幽蓝纹路因为刚才的剧烈运动和微粒的刺激，光芒变得有些不稳定，忽明忽暗。小腿的伤口麻木感在扩散，腰侧的撞击也隐隐作痛。

他低头，摊开左手。

那颗淡金色的微粒静静躺在掌心，光芒柔和。刚才的搏杀中，似乎有极其微量的金色光点从微粒中析出，融入了他的手掌。与此同时，他感觉到，自己体内，那因为“微粒残留”而滞涩、混乱的感觉，似乎……松动了一丝。

非常细微的一丝，像是堵塞的河道被冲开了一粒沙子。

但感觉是真实的。

这就是掠夺。从这黑暗的、吞噬生命的遗迹中，夺取维系自身存续的资粮。微粒，总量恒定，流转不息。有人囤积，就有人缺失。而像他这样的弃徒，只能在这最底层、最危险的边缘，用命去换。

他将微粒紧紧攥住，冰冷的晶体硌着掌心的伤口，带来清晰的痛感。

名字……依然没有。

身份……依然迷雾。

但目的，在这一刻清晰起来：活下去。弄清楚左臂的“微粒残留”是什么。弄明白短刃符文的含义。然后……找到将他变成这样的人，或者东西。

他撑着石壁，再次站起。主通道深处，黑暗依旧，窥视感依旧。但有了手中这一点微弱的光芒，有了刚才生死搏杀后残存的凶戾，那黑暗似乎不再那么绝对地令人窒息。

他调整了一下呼吸，将短刃插回腰间的刀鞘。握着微粒的左手收进怀里，贴着胸口。那里，心脏在沉稳地跳动。

他迈开脚步，继续向下。

向着黑雾遗迹的更深处，向着未知的真相，也向着更多、更残酷的掠夺与厮杀。

黑暗的通道，吞没了他的背影。

只有石壁上那些暗红色的纹路，在他经过时，似乎微微亮了一下，又迅速熄灭。
