# 第一章 · 血痂与符文

痛。

不是那种尖锐、清晰的痛，而是黏稠的、渗透性的，像无数根生锈的细针，缓慢地碾磨着每一寸骨骼与神经末梢。他睁开眼，视野里没有光，只有一片浑浊的、仿佛沉淀了万古尘埃的黑暗。鼻腔里充斥着铁锈、腐土和某种难以言喻的甜腥混合的气味，浓烈得几乎要凝结成实体，堵塞他的呼吸。

他躺着。身下是冰冷、潮湿、凹凸不平的岩石。皮肤能感觉到粗糙的颗粒和某种滑腻的、可能是苔藓或菌毯的东西。他尝试移动手指，关节发出干涩的、几乎要断裂的摩擦声。疼痛随之加剧，从四肢百骸汇聚到胸腔，最终在心脏的位置炸开一团冰冷的火焰。

记忆是空的。

不，不是完全的空。有一些碎片，像是被暴力撕扯后残留的纸屑，边缘焦黑卷曲，字迹模糊。黑暗。更深的黑暗。急速下坠的风声。金属切入肉体的闷响。还有……光？不，不是光，是某种更幽暗、更本质的东西的流动，像无数细微的沙，在绝对的虚无中勾勒出短暂而狰狞的轮廓。

他不知道自己是谁。

这个认知带来的不是恐慌，而是一种更深沉的、近乎虚无的寒意。存在本身被抽空了根基，只剩下这具疼痛的躯壳和一片荒芜的内心。他缓缓地、极其艰难地侧过身，用肘部支撑起上半身。碎石和泥土簌簌落下。黑暗依旧浓稠，但似乎有极其微弱的光源，来自极远处，或者……上方？那光不足以照亮任何东西，只是让黑暗显出层次，像一潭死水底部沉淀的、更深的淤泥。

他的左手触碰到腰间一个坚硬、冰凉的物体。本能驱使着他握住了它。触感熟悉得令人心悸——那是一把短刃的柄。皮革缠绕，已经被血和汗浸透，变得僵硬而滑腻。他将其抽出，动作流畅得仿佛演练过千万遍，与身体的剧痛和僵硬形成诡异的反差。

短刃入手沉甸甸的，刃身大约一尺余长，在绝对的晦暗中，它本身并不反光，却似乎能吸收周围那点可怜的微光，勾勒出自身流畅而险恶的曲线。他将刃身凑到眼前，几乎贴着脸颊。视力在适应，或者说，某种残留的本能在驱使感官突破极限。他看到了刃身上蚀刻的纹路。

不是装饰性的花纹，也不是任何他（或许曾经）认知的文字。那是更加古老、更加本质的东西。线条扭曲盘绕，时而尖锐如荆棘突刺，时而圆融如毒蛇盘踞，在某些转折处，纹路会突然中断，形成微小的、仿佛能吞噬一切的凹陷。他用指尖轻轻拂过那些符文，冰冷的金属触感下，似乎有极其微弱、几乎无法察觉的脉动传来，像一颗沉睡在金属深处的心臟，缓慢地搏动了一下。

**符文。** 这个词突兀地出现在空白的脑海，带着铁锈和血腥的气息。它属于这把刀，或许，也曾属于他。

他撑着短刃，试图完全站起。双腿颤抖得厉害，肌肉纤维仿佛在哀鸣。就在他身体重心变化的瞬间，胸腔深处，那股冰冷的火焰猛地窜升。

不是疼痛。

是某种……存在。

它就在那里，在他心脏偏左的位置，深深嵌入，或者说，生长在他的血肉与能量脉络之中。它不是实体，却能被他“感觉”到——一粒极其微小，却沉重得仿佛能压塌山岳的“点”。它寂静无声，却又散发着一种原始的、饥渴的波动，像一头蛰伏在巢穴最深处的凶兽，仅仅是无意识的呼吸，就搅动着他体内残存的一切。

**微粒。**

又一个词浮现，伴随着更强烈的寒意和……杀意。不是他对这微粒的杀意，而是这微粒本身散发出的、对一切外物的、纯粹的吞噬与毁灭倾向。它残留在这里，像一颗射入体内却未被取出的弹头，不同的是，这颗“弹头”是活的，并且正以他的生命为温床，缓慢地……生长？还是衰变？他无法判断。

他剧烈地咳嗽起来，咳出的不是血，而是一小团带着暗金色星点、迅速在空气中消散成黑色灰烬的雾气。喉咙里火辣辣地疼，而胸腔内的那个“点”，似乎因为这次扰动，稍微“亮”了一瞬。就是这一瞬，他“看”到了更多。

不是用眼睛。

某种内视，或者说，是那微粒强行投射在他意识中的景象。他“看到”自己体内一片狼藉，能量通道（他不知道为什么知道这个词）大多断裂、淤塞，像被暴力犁过的田地。唯有心脏附近，几条最粗壮、最深沉的通道，还勉强维持着轮廓，而那微粒，就卡在几条主脉交汇的一个关键节点上，如同一个不祥的黑色枢纽。微粒周围，丝丝缕缕暗红色的、他自己的生命能量（或者别的什么），正被它缓慢而坚定地抽取、吞噬，转化为一种更凝练、更晦暗的、带着金属质感的暗金色流质，那流质沿着未被完全破坏的细微支脉，极其缓慢地向全身扩散。

它在改造他。或者说，在把他当成燃料和原料。

这就是“残留”的真相。

他扶着岩壁，喘息着，汗水混合着污垢从额角滑落。警觉，像一头受伤的野兽，瞬间绷紧了他每一根神经。失忆带来的存在性恐惧，被更迫在眉睫的生存危机压倒。这不属于他的东西，正在侵蚀他。而他，必须找到办法，要么控制它，要么……剔除它。前提是，他能活下去。

他开始打量周围。这是一个天然形成的岩洞，或者说是地裂的某一部分。空间不大，怪石嶙峋，头顶极高处有隐约的裂缝，那极其微弱的、让黑暗分层的光，或许就是从那里渗下来的。空气凝滞污浊。除了他躺倒的地方，地面有明显的拖拽痕迹，一直延伸到洞穴深处更黑暗的角落。痕迹旁，散落着一些东西：几片看不出原本颜色的碎布，半截断裂的、非骨非木的细杆，还有……几滴已经干涸发黑，但在微粒带来的异常感知中，依旧散发出微弱能量波动的血迹。

不是他的血。这血迹的能量性质与他体内的截然不同，更加狂暴，更加……廉价？仿佛掺杂了太多杂质。

他蹲下，用短刃的尖端轻轻触碰那血迹。刃身上的古老符文，似乎极其轻微地闪烁了一下，比呼吸更短暂。血迹上残留的最后一丝能量波动，消失了，仿佛被短刃“吃”掉了。

他握紧了刀柄。这刀，也不简单。

沙沙——

极其轻微的声音，从洞穴入口的方向传来。不是风声，是某种东西摩擦岩石的声音，谨慎，缓慢，带着捕食者的耐心。

他瞬间静止，呼吸压到最低，身体本能地调整重心，贴近最近的一块凸起的岩柱阴影中。影迹潜行——这个能力如同烙印在肌肉和骨髓里，无需思考，自然发动。他感觉到周身的光线（尽管几乎没有）似乎发生了细微的偏折，气息、温度、甚至存在感都迅速淡化，与岩石的冰冷、洞穴的黑暗融为一体。等级五的潜行，或许曾经意味着更多，此刻，只是他赖以藏身的唯一依仗。

声音近了。

不是一个人。是三个，或许四个。脚步很轻，但落地时的节奏和力度显示他们并非精于隐匿，只是习惯在复杂地形移动。他们带来了光——一种惨白色的、不稳定跳动的冷光，来自他们手中提着的、像是某种生物腺体制成的灯笼。光线撕破洞穴入口的黑暗，将扭曲的岩壁投影拉得老长，如同舞动的鬼怪。

“残留信号最后消失就是这附近。”一个嘶哑的声音响起，像是沙砾摩擦，“‘载体’应该跑不远，受了‘幽蚀之种’的污染，又坠下‘初黯天堑’，没立刻化成脓水就算命硬。”

“少废话，赶紧找。”另一个声音更冷，带着不耐烦，“上面催得紧，8400万总量，每一粒流转都要记录在案。这粒‘幽蚀’虽然只是次级衍生体，但性质特殊，万一被‘下面’的东西先感应到，我们都得填进去。”

“嘿嘿，怕什么，这鸟不拉屎的‘碎岩层’，连个像样的聚集地都没有，除了石头就是那些半死不活的‘矿渣’。”第三个声音显得猥琐，“找到了，先把微粒剥离，那‘载体’……看样子以前是个好手，说不定身上还能榨出点油水。心核、筋骨，都是材料。”

他们的对话，每一个词都像冰冷的凿子，敲打进他空白的脑海。

微粒。幽蚀之种。天堑。载体。剥离。油水。

这些词汇组合在一起，勾勒出一个冰冷、残酷、以微粒为绝对核心的世界的轮廓。而他，就是那个不幸的“载体”，一个被植入致命异物、正被猎手追踪的猎物。

提灯的光斑扫过洞穴地面，掠过他刚才躺卧的地方，扫过那些拖痕和血迹。他屏住呼吸，潜行的状态被催发到极限，身体仿佛真的化作了岩石的一部分。心跳被强行压制，连胸腔内那粒“幽蚀之种”的波动，似乎也因他极致的收敛而变得晦暗不明。

“有痕迹！”嘶哑声音的主人蹲下，查看那几滴干涸的血迹，“是‘载体’的？不对，这能量反应太杂……像是‘矿渣’的？难道有‘矿渣’先来过？”

“矿渣敢碰‘幽蚀’？嫌死得不够快？”冷酷声音走近，“拖痕往里面去了。小心点，说不定那家伙还有口气，躲在暗处。”

三个身影呈松散的三角阵型，提着惨白灯笼，谨慎地向洞穴深处，也就是他藏身岩柱的相反方向探去。他们的注意力被拖痕和可能存在的“矿渣”吸引了。

机会。

杀意，冰冷而纯粹，如同他手中的短刃，悄然出鞘。不是愤怒，不是仇恨，仅仅是生存的本能，以及对“掠夺者”这个身份天然的敌对。他们的话已经判了他死刑——找到，剥离，榨取。没有第二条路。

他动了。

像一道从岩石本身剥离下来的影子，悄无声息地滑出藏身处，贴着凹凸不平的地面，逆着灯笼光线的方向，袭向最后面那个声音猥琐的家伙。潜行状态在移动中并未完全解除，而是与环境光影的变动完美契合，直到他的影子笼罩了目标的背影。

短刃抬起。

刃身上的古老符文，在如此近的距离，似乎感应到了即将到来的杀戮与生命能量的消散，那些扭曲的线条微微发烫，散发出肉眼不可见、但能被他清晰感知的渴求。

他左手如铁钳般捂住目标的嘴，右手的短刃从颈侧最柔软、连接头颅与躯干的位置，精准无比地刺入，横向一切。动作干脆利落，没有一丝多余。刀刃切开皮肉、软骨、血管，最后擦过颈椎骨的缝隙。

猥琐的声音戛然而止，变成喉咙被血沫堵塞的“咯咯”声。身体剧烈地抽搐一下，便软了下去。灯笼脱手，被他用脚背轻轻接住，没有发出声响。惨白的光映照出死者瞬间凝固的、充满惊愕与茫然的脸，还有喷溅在岩壁上、尚且温热的血液。

他没有立刻去捡灯笼，而是将尸体轻轻放倒，拖到一块岩石后。整个过程不过两三息时间，前面两人尚未走远，注意力仍被黑暗深处可能存在的危险吸引。

胸腔内的“幽蚀之种”，在杀戮完成的瞬间，似乎悸动了一下。一股微不可查的、带着铁锈味的暖流，从短刃接触的血液中，逆着刀柄传入他的手臂，最终汇向胸口的微粒。不是补充，更像是一种……共鸣，或者刺激。微粒本身没有壮大，但那暗金色的流质在体内扩散的速度，似乎加快了微不可察的一丝。

杀戮，能刺激它。

这个发现让他心底的寒意更甚。但他没有时间细想。

他捡起那盏惨白灯笼，将其光线调至最暗，仅仅能照亮脚前尺许之地。然后，他像幽灵一样，再次没入阴影，朝着剩下的两个猎物潜行而去。

洞穴深处，拖痕的尽头，是一处稍微开阔的凹陷。那里堆着一些杂物，几具蜷缩的、早已僵硬的尸体，穿着破烂，骨瘦如柴，正是所谓的“矿渣”。看来之前确实有倒霉的拾荒者发现了昏迷的他，试图拖走，却不知何故死在了这里。或许是他们接触了他，引发了“幽蚀之种”的无意识反噬？

此刻，那嘶哑声音和冷酷声音的两人，正站在那几具“矿渣”尸体旁，低头查看着。

“都死了，能量被吸干，死前极度痛苦。”嘶哑声音说，“是‘幽蚀’的特性，没错。‘载体’肯定在这里停留过，甚至可能醒来过。”

“分开找，他一定还在洞里！”冷酷声音果断下令，“你左我右，以这里为中心，搜遍每一个角落！注意阴影，那家伙可能……”

他的话没能说完。

因为一道影子，从他自己的影子里“站”了起来。

冷酷男子反应极快，腰间一把带着锯齿的弯刀瞬间出鞘，反手向后劈去！刀风凄厉，带着明显的能量波动，比之前猥琐男子的实力强出不止一筹。

然而，他劈中的只是空气。

真正的攻击来自侧面。那盏被调暗的惨白灯笼，如同鬼火般飘忽而至，直砸他的面门。他下意识挥刀格挡，灯笼破碎，里面某种生物腺体破裂，溅射出黏稠的、带着强烈麻痹和致幻效果的荧光液体。

“啊！”他惨叫一声，眼睛和口鼻瞬间被糊住，视线模糊，呼吸受阻，动作一滞。

就在这一滞的瞬间，短刃到了。

从极其刁钻的角度，自下而上，刺入他的下颌，穿透口腔，直抵颅腔。刃身上的符文再次微微发烫，这一次，他清晰地感觉到，不仅仅是杀戮的刺激，当短刃刺入对方头颅的刹那，对方体内某种稀薄而散乱的能量（或许是修炼所得，或许是长期在微粒环境中浸染的积累），被短刃强行抽取、吞噬了一小部分，然后反馈回他体内，大部分被胸口的“幽蚀之种”吸收，极小一部分，散入了他干涸受损的经脉。

冷酷男子连最后的挣扎都未能做出，便瞪大着被荧光液覆盖的双眼，直挺挺倒下。

“老三？！老四？！”嘶哑声音这时才惊觉不对，猛地转身，提灯照向这边。

他看到的是同伴倒下的尸体，和站在尸体旁，手中短刃滴着混合了血液与荧光液、脸上沾着几点溅射血污的“载体”。

那张脸在惨白与荧绿交织的光线下，没有任何表情。眼神空洞，却又深处燃烧着两点冰冷的、属于野兽的凶光。失忆抹去了他的过去，却似乎将某种更本质的、属于杀戮者的本能，淬炼得更加纯粹和危险。

“你……你醒了？！怎么可能！‘幽蚀之种’应该……”嘶哑声音又惊又怒，连连后退，手中的提灯光芒剧烈晃动。

他没有给对方说完话的机会。

脚下一蹬，身体前冲，不再是纯粹的潜行暗杀，而是带着一股蛮横的、被体内冰冷火焰和杀戮刺激催发出来的凶性。速度极快，身影在晃动灯光下拉出残影。

嘶哑男子狂吼一声，丢开提灯，双手在胸前结出一个简陋的手印，一层灰蒙蒙的、带着土石质感的光晕浮现在他体表。“岩肤术！”他试图防御。

短刃刺到。

没有技巧，只有力量与速度。刃尖点在那灰蒙蒙的光晕上，发出刺耳的摩擦声，竟然没能立刻刺穿。但短刃上的符文，骤然亮起——不是光芒，而是一种更深沉的“暗”，仿佛将周围的光线都吸了进去。符文的力量加持在刃尖。

噗嗤！

灰蒙蒙的光晕如同脆弱的蛋壳般破碎。短刃长驱直入，刺入嘶哑男子的心口。这一次，短刃的吞噬感更加明显，对方体内那点可怜的、与大地属性相关的能量，如同遇到海绵的水，被迅速抽走。嘶哑男子双眼暴凸，喉咙里发出“嗬嗬”的声音，充满了难以置信的恐惧。

他抽回短刃，男子软倒。

洞穴重归寂静，只有三盏破碎或未破的提灯，在地上散发着惨白或荧绿的光，映照着三具迅速冰冷的尸体，和岩壁上喷溅的、正在缓缓凝固的血液。

他站在原地，剧烈地喘息。连续两次爆发，尤其是最后一次强行破防，对他这具重伤未愈的身体负担极大。胸口内的“幽蚀之种”因为吸收了额外的能量（来自短刃吞噬反馈的部分），变得活跃了一些，暗金色流质的扩散速度明显加快，带来一阵阵酥麻与轻微撕裂般的痛楚，仿佛有新的根系在他体内生长。

他走到嘶哑男子的尸体旁，蹲下，开始搜索。动作有些生疏，但很快变得熟练。一些零碎的、像是货币的、散发着极其微弱能量波动的灰色晶体（但不是微粒），几块干硬的肉脯，一个水囊，还有一块半个巴掌大小、非金非木的黑色令牌。令牌入手冰凉，正面刻着一个扭曲的、像是多条毒蛇缠绕的图案，背面则是一串他看不懂的符号，以及一个清晰的数字烙印：“丙-七四”。

掠夺者集团的标识？等级编号？

他将令牌收起，又检查了另外两人，收获大同小异，只是令牌的编号不同：“丁-二九”、“丁-三一”。看来嘶哑男子（丙-七四）是这个小队的头目。

没有找到任何关于他身份的信息，也没有关于“幽蚀之种”和这场追捕的更多资料。这些只是最底层的爪牙。

他喝了几口水，吃了点肉脯。身体得到些许补充，但疲惫和疼痛依旧。更重要的是，体内的微粒残留，在吸收了这些“养料”后，与他身体的结合似乎更紧密了
