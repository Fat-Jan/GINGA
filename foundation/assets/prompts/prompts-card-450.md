# 450. 同人创作：大结局设计 (Grand Finale)

## 提示词内容

```json
{
  "task": "write_fanfic_ending",
  "type": "Return Home / Stay in World / Travel to Next World",
  "final_battle": "Vs Ultimate Villain (Buffed) / Vs System",
  "resolution": "Loose ends tied / CP confirmed / World peace",
  "farewell": "Saying goodbye to Canon Characters",
  "epilogue": "Years later / Legend left behind",
  "emotion": "Satisfying / Melancholic / Open-ended"
}
```

## 使用场景
同人创作第十步。给同人故事一个完美的句号。

## 最佳实践要点
1.  **去留问题**：主角是回到现实世界，还是留在二次元老婆孩子热炕头？这是最终的选择。
2.  **传说延续**：主角即使离开，也在这个世界留下了不可磨灭的传说。

## 示例输入
击败最终BOSS后，系统提示可以返回地球。主角看着身边的伙伴和爱人，选择捏碎系统核心，永远留在这个世界。
