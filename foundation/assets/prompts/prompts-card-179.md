# 179. 虫族/蜂群意识沟通逻辑

## 提示词内容

```json
{
  "task": "write_hive_mind_dialogue",
  "entity": "The Swarm",
  "style": "Plural pronouns ('We', 'Us'), fragmented thoughts from multiple drones",
  "message": "Resistance is futile. Your biology will be added to ours.",
  "tone": "Cold, efficient, overwhelming"
}
```

## 使用场景
科幻/星际文。描写集体意识的压迫感。

## 最佳实践要点
1.  **复数第一人称**：始终使用“我们”，强调个体在集体面前的渺小。
2.  **高效冷酷**：蜂群思维没有情感，只有效率和生存本能。

## 示例输入
- 蜂群主体：星舰腹舱中的虫族幼巢
- 沟通对象：被俘的人类舰长
- 表现重点：复数意识、情绪同步、个体概念缺失
