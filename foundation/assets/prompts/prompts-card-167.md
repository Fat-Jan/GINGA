# 167. 梦核 (Dreamcore) 场景构建

## 提示词内容

```json
{
  "task": "write_dreamcore_scene",
  "elements": [
    "Liminal spaces (Empty hallways)",
    "Oversaturated blue sky",
    "Faceless figures",
    "Nostalgic playground equipment"
  ],
  "feeling": "Comfortable yet unsettling / Déjà vu",
  "sensory": "The hum of fluorescent lights, smell of chlorine, dead silence"
}
```

## 使用场景
惊悚/超现实主义。营造一种“熟悉的陌生感”。

## 最佳实践要点
1.  **阈限空间**：描写那些过渡性的、本该有人却无人的场所（走廊、候机厅）。
2.  **怀旧恐怖**：利用童年元素（游乐场、老电视）引发潜意识的不安。

## 示例输入
```json
{
  "elements": ["空无一人的小学走廊", "永远停在下午四点的操场", "没有五官的值日生"],
  "feeling": "怀旧但不安",
  "sensory": "粉笔灰、日光灯嗡鸣、远处循环的下课铃"
}
```
