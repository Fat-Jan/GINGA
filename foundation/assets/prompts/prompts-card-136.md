# 136. 房地产/地皮开发策划案

## 提示词内容

```json
{
  "task": "real_estate_development_plan",
  "land_type": "Haunted House / Wasteland / Prime District",
  "mc_vision": "Turn it into a Theme Park / High-end Cultivation Abode",

    "Feng Shui array included",
    "Spirit vein access",
    "Absolute security system"
  ],
  "marketing_slogan": "Your sanctuary in a chaotic world."
}
```

## 使用场景
神豪/经营/种田文。主角进行资产增值或领地建设。

## 最佳实践要点
1.  **化腐朽为神奇**：主角通常接手一块废地或凶宅，通过特殊手段（风水、阵法）变废为宝。
2.  **精准定位**：明确目标客户（修士、富豪），定制卖点。

## 示例输入
- 地块：旧码头仓储区
- 目标：改造成蒸汽朋克商业街
- 限制：预算紧张，地下管线复杂
