# 129. 蒸汽波/复古未来主义场景描写

## 提示词内容

```json
{
  "task": "write_vaporwave_scene",
  "elements": [
    "Neon grids",
    "Greek statues with glitches",
    "Palm trees against a pink sunset",
    "Windows 95 aesthetic"
  ],
  "mood": "Nostalgic, surreal, lonely",
  "sensory": "Hum of old CRT monitors, smell of artificial cherry, lo-fi music"
}
```

## 使用场景
科幻/异质空间。营造独特的视觉美学。

## 最佳实践要点
1.  **拼贴感**：将古典元素（雕像）与电子元素（故障风）强行拼贴，制造超现实感。
2.  **色调**：强调粉色、紫色、青色的霓虹色调。

## 示例输入
```json
{
  "elements": ["粉紫色霓虹网格", "故障希腊雕像", "无人商场里的棕榈树"],
  "mood": "怀旧、空旷、轻微眩晕",
  "sensory": "老式 CRT 嗡鸣和廉价樱桃香精味"
}
```
