# 214. 初吻/定情场景生成

## 提示词内容

```json
{
  "task": "write_first_kiss",
  "mood": "Romantic / Accidental / Desperate",
  "setting": "Fireworks festival / Under the rain / Post-battle",
  "buildup": "Long silence, looking into eyes, leaning in slowly",
  "interruption_risk": "High (Phone ringing / Friends walking in)",
  "the_kiss": "Soft, tentative, tasting like strawberries/rain"
}
```

## 使用场景
言情/CP文。描写关键的情感突破点。

## 最佳实践要点
1.  **氛围铺垫**：环境（烟花、雨声）必须配合气氛。
2.  **被打断**：经典的“被打断”梗可以增加焦急感，或者反套路直接吻下去。

## 示例输入
- 场景：烟花大会后的小巷。
- 氛围：试探、紧张、差点被电话铃打断，吻带一点雨水味。
