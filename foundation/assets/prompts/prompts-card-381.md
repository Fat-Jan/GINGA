# 381. 场景：赛博空间/黑客潜入 (Cyberspace Netrun)

## 提示词内容

```json
{
  "task": "write_cyberspace_scene",
  "visual_style": "Tron-like / Abstract data streams / Glitch art",
  "avatar": "A glowing knight / A shapeless cloud",
  "defense_ice": "Firewalls appearing as literal walls of fire / Guardian programs",
  "action": "Dodging data spikes / Injecting viruses",
  "sensation": "Information overload / Synaptic overheating",
  "goal": "Steal the encrypted file"
}
```

## 使用场景
赛博朋克/科幻。将枯燥的代码攻防转化为具象化的战斗。

## 最佳实践要点
1.  **具象化**：把黑客技术（病毒、防火墙）比喻成武器和怪兽。
2.  **通感**：网络伤害会反馈到肉体（如流鼻血、脑部剧痛）。

## 示例输入
描写主角潜入“荒坂塔”的子网，对抗名为“地狱犬”的防卫程序。
