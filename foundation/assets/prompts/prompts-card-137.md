# 137. 宗教/教派教义生成

## 提示词内容

```json
{
  "task": "create_religion_dogma",
  "god_name": "The Lord of Mist",
  "core_belief": "Knowledge is dangerous; Ignorance is bliss.",
  "commandments": [
    "Thou shalt not record history.",
    "Thou shalt wear a blindfold in the temple.",
    "Offer secrets as tribute."
  ],
  "ritual": "Burning books on the solstice."
}
```

## 使用场景
奇幻/诡秘流。设计独特的信仰体系。

## 最佳实践要点
1.  **自洽性**：教义、戒律和仪式必须逻辑统一（如信奉无知，就要烧书）。
2.  **神秘感**：仪式应当具有强烈的视觉冲击力和神秘氛围。

## 示例输入
- 教派：信奉海潮与记忆的边境教团
- 核心禁忌：遗忘亡者姓名
- 冲突点：新教义要求献出个人记忆
