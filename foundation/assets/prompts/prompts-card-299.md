# 299. 结局：开放式结局生成 (Open Ending)

## 提示词内容

```json
{
  "task": "write_open_ending",
  "final_scene": "MC walking into sunset / A mysterious knock on door / A coin spinning",
  "ambiguity": "Did he survive? / Is it a dream? / Did the cycle start again?",
  "tone": "Hopeful yet uncertain / Melancholic / Thought-provoking",
  "last_line": "A powerful sentence that leaves interpretation to the reader",
  "symbolism": "A flower blooming in wasteland / A broken clock ticking again"
}
```

## 使用场景
大结局/短篇。留有余韵，引发读者讨论。

## 最佳实践要点
1.  **余韵**：不要把话说满，留出想象空间。
2.  **象征**：用象征性的意象来暗示可能的结局。

## 示例输入
《盗梦空间》式的结局，陀螺还在转，但主角笑了。
