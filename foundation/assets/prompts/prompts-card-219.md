# 219. 动作戏节奏表 (Action Beat Sheet)

## 提示词内容

```json
{
  "task": "action_beat_sheet",
  "scene": "Sword fight on a rooftop",
  "beats": [
    "Beat 1: Engagement (Clash of steel)",
    "Beat 2: Disadvantage (MC slips, enemy presses attack)",
    "Beat 3: Turnaround (MC uses environment, throws sand)",
    "Beat 4: Climax (Final strike)",
    "Beat 5: Aftermath (Heavy breathing, enemy falls)"
  ],
  "pacing": "Fast -> Slow (Tension) -> Fast -> Stop"
}
```

## 使用场景
玄幻/动作文。规划精彩的打斗流程。

## 最佳实践要点
1.  **起承转合**：打斗不能是一味的平铺直叙，要有优势劣势的转换（Turnaround）。
2.  **环境互动**：利用场地元素（如瓦片、沙子）增加战斗的真实感和策略性。

## 示例输入
- 动作场景：狭窄厨房追逐战
- 目标：主角夺回装有证据的硬盘
- 节奏：起手压迫、中段反转、结尾短暂停顿
