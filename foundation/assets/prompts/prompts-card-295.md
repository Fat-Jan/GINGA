# 295. 情节：误会解除 (Misunderstanding Cleared)

## 提示词内容

```json
{
  "task": "write_misunderstanding_resolution",
  "misunderstanding": "She thought he cheated / He thought she betrayed him",
  "trigger": "Found a diary / Third party confession / Life-and-death situation",
  "dialogue": "Tearful confession / 'Why didn't you tell me?'",
  "action": "Hug / Kiss / Shared silence",
  "emotional_release": "Relief / Guilt / Renewed trust"
}
```

## 使用场景
言情/都市/虐文。虐后的甜，释放读者积压的情绪。

## 最佳实践要点
1.  **爆发力**：积压的误会一朝解除，情感爆发要强烈。
2.  **悔恨感**：误会方往往会产生强烈的悔恨和补偿心理。

## 示例输入
女主终于发现男主当年的绝情是为了保护她。
