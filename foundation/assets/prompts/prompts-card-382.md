# 382. 润色：恐怖谷效应描写 (Uncanny Valley)

## 提示词内容

```json
{
  "task": "write_uncanny_valley_description",
  "subject": "A robot / A possessed doll / A doppelganger",
  "features": "Skin too smooth / Eyes unblinking / Smile too wide",
  "movement": "Jerky / Fluid but wrong / Head tilting at impossible angle",
  "mc_reaction": "Instinctual revulsion / Hair standing on end",
  "atmosphere": "Silence, focusing on the subtle wrongness"
}
```

## 使用场景
恐怖/悬疑/科幻。描写那种“像人但不是人”的毛骨悚然感。

## 最佳实践要点
1.  **细节**：专注于那些微小的、不对劲的细节（如不会呼吸）。
2.  **本能**：强调人类基因里对异类的本能排斥。

## 示例输入
描写一个“完美的仿生人”对着主角微笑，但眼神是死的。
