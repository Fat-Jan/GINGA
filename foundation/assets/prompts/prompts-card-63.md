# 63. 团宠文配角互动模式

## 提示词内容

```json
{
  "task": "design_group_pet_dynamics",
  "mc_archetype": "Cute Baby / Lucky Star / Genius",
  "guardians": [
    {"role": "Brother 1 (CEO)", "interaction": "Spoils with money"},
    {"role": "Brother 2 (Hacker)", "interaction": "Protects privacy/Revenge"},
    {"role": "Brother 3 (Idol)", "interaction": "Public flaunting"},
    {"role": "Father (Strict)", "interaction": "Secretly doting"}
  ],
  "conflict": "Outsider bullies MC -> Guardians unite to destroy outsider"
}
```

## 使用场景
团宠/萌宝文。设计围绕主角的守护者联盟。

## 最佳实践要点
1.  **功能区分**：每个守护者有不同的宠溺方式（钱、权、力、颜）。
2.  **反差萌**：严厉的角色私下宠溺，效果最佳。

## 示例输入
- 主角：三岁半小郡主，能听懂灵兽说话。
- 守护者：战神父亲、毒舌哥哥、太后祖母、护短灵鹤。
- 冲突：外来使臣嘲笑她，所有人用不同方式替她撑腰。
