# 173. 遗迹/古文明考古日志

## 提示词内容

```json
{
  "task": "write_archeology_log",
  "site": "Atlantis / Martian Ruins",
  "discovery": "A mural depicting humans serving aliens",
  "artifact": "A power core still humming",
  "theory": "They didn't die out... they left.",
  "atmosphere": "Dusty, ancient, awe-inspiring"
}
```

## 使用场景
探险/科幻。通过考古发现揭示世界观背景。

## 最佳实践要点
1.  **碎片化叙事**：通过破碎的壁画、日记片段拼凑出历史真相。
2.  **颠覆认知**：考古发现往往会推翻现有的历史认知（如人类起源）。

## 示例输入
```json
{
  "site": "沙漠下的倒置神庙",
  "discovery": "壁画描绘人类向星门献祭影子",
  "artifact": "仍在低声倒数的青铜星盘"
}
```
