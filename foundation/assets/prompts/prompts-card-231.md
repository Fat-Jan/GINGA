# 231. 心理惊悚：被监视的恐惧

## 提示词内容

```json
{
  "task": "write_stalking_horror",
  "signs": [
    "Objects moved slightly in the house",
    "Unknown number calling and breathing",
    "A silhouette in the streetlamp every night"
  ],
  "escalation": "Finding a photo of MC sleeping taken from inside the closet",
  "mc_reaction": "Paranoia, checking locks, buying a weapon"
}
```

## 使用场景
悬疑/惊悚文。通过细节堆叠制造心理压力。

## 最佳实践要点
1.  **入侵感**：家是最后的避风港，一旦家不再安全，恐惧感倍增。
2.  **渐进**：从微小的违和感（东西挪位）到直接的威胁（照片）。

## 示例输入
- 场景：独居女主连续三晚发现门口鞋印朝向屋内。
- 线索：冰箱便签多了一句“你今天回来晚了”。
- 输出：写一段递进式心理惊悚，不提前揭露跟踪者身份。
