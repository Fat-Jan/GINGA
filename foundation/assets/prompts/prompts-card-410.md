# 410. 场景：直播鉴宝捡漏 (Antique Appraisal)

## 提示词内容

```json
{
  "task": "write_appraisal_stream",
  "item": "Rusty Sword / Fake Porcelain / Old Book",
  "ability": "Golden Eyes / System Scan / History Knowledge",
  "owner": "Scammer / Clueless Person / Arrogant Expert",
  "twist": "Fake hiding Real / National Treasure",
  "dialogue": "This is very punishable / 5 years minimum",
  "outcome": "Police called / Value skyrockets"
}
```

## 使用场景
鉴宝/直播文。通过辨别真伪、发现国宝来制造悬念和爽点。

## 最佳实践要点
1.  **造梗**：使用“很刑”、“这就去自首”等直播梗增加趣味性。
2.  **反转**：看似不起眼的垃圾，实则是价值连城的国宝。

## 示例输入
宝友拿出一个“微波炉专用碗”，主角一看是“汝窑”，反手建议宝友立刻上交国家。
