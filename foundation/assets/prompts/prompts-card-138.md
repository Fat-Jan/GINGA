# 138. 星际战舰参数设计

## 提示词内容

```json
{
  "task": "design_spaceship",
  "class": "Dreadnought / Scout / Carrier",
  "name": "The Indomitable",
  "specs": {
    "Length": "5km",
    "Weapons": "Railguns, Laser Arrays, Planet Killer",
    "Defense": "Void Shields, Nanite Armor",
    "Engine": "Hyperdrive / Warp Core"
  },
  "crew_capacity": "5000 + AI Core"
}
```

## 使用场景
科幻/星际文。设计主角的座驾或敌方旗舰。

## 最佳实践要点
1.  **巨物恐惧**：强调战舰的尺寸和火力，营造压迫感。
2.  **科技树**：武器和引擎设定应符合该世界观的科技水平。

## 示例输入
- 舰级：边境巡洋舰。
- 参数：长度 900 米，主武器为轨道炮，弱点是护盾过热。
