# 447. 同人创作：弥补遗憾/改变悲剧 (Fix-it Climax)

## 提示词内容

```json
{
  "task": "write_fix_it_scene",
  "tragedy": "Ace's Death / Jiraiya's Death / Rengoku's Death",
  "preparation": "Long-term planning / Secret Training / Gathering Allies",
  "intervention": "Blocking the attack / Healing the wound / Killing the killer",
  "twist": "Fate tries to resist / Villain has a backup plan",
  "emotion": "Tears of joy / Relief / Shock of Canon Characters",
  "outcome": "Character lives / New timeline branch"
}
```

## 使用场景
同人创作第七步。同人最大的爽点之一：救下那些意难平的角色。

## 最佳实践要点
1.  **千钧一发**：在悲剧发生的最后一秒出手，情绪张力最大。
2.  **代价**：改变命运通常需要付出代价（受伤、暴露底牌），这样才显得珍贵。

## 示例输入
在自来也即将被佩恩六道杀死的一刻，主角利用飞雷神之术带着纲手从天而降，一拳轰飞佩恩修罗道。
