# 266. 西幻：吟游诗人的史诗传唱

## 提示词内容

```json
{
  "task": "write_bard_song",
  "hero": "{{Hero Name}}",
  "feat": "Slaying the Dragon / Saving the Kingdom",
  "style": "Epic / Folk / Melancholic",
  "lyrics_snippet": "Four lines of rhyming verse",
  "audience_reaction": "Cheering / Crying / Tossing coins",
  "truth_vs_fiction": "The song exaggerates the hero's height / Omits the embarrassing parts"
}
```

## 使用场景
西幻/奇幻。通过侧面描写展现主角的声望。

## 最佳实践要点
1.  **史诗感**：歌词要有韵律，带有传说色彩。
2.  **反差萌**：主角听到夸张的传唱时尴尬的反应。

## 示例输入
主角：莱恩。事迹：在下水道杀了一只变异老鼠（被传唱为击败了深渊魔龙）。
