# 276. 奇幻：召唤兽/使魔的叛变

## 提示词内容

```json
{
  "task": "write_familiar_betrayal",
  "creature": "Demon / Dragon / Spirit",
  "bond": "Blood Contract / Soul Bind",
  "trigger": "MC becomes weak / A higher power calls / The contract expires",
  "sign": "Disobeying orders / Cold stares / Hiding power",
  "climax": "The creature attacks MC at a critical moment",
  "twist": "It was actually trying to save MC from a trap (optional)"
}
```

## 使用场景
御兽/奇幻/召唤流。打破“忠诚”的设定，增加危机感。

## 最佳实践要点
1.  **铺垫**：前期的一些不听话的小细节。
2.  **压迫感**：曾经的助力变成最了解自己的敌人。

## 示例输入
主角重伤时，签订契约的恶魔咧嘴笑了。
