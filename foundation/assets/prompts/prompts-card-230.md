# 230. 现代修真：灵气与科技结合产品

## 提示词内容

```json
{
  "task": "design_magitech_product",
  "item": "Smartphone / Car / Weapon",
  "feature": "Battery charges via ambient Qi / AI spirit assistant / Flying mode",
  "marketing": "Runes inside, sleek glass outside",
  "flaw": "Explodes if user's Qi is unstable"
}
```

## 使用场景
都市修真/科幻修真。设计脑洞大开的混合科技产品。

## 最佳实践要点
1.  **痛点解决**：修真技术解决了现代科技的哪些痛点（如无限续航）？
2.  **违和感**：用现代商业语言包装修真概念（如“灵气5G基站”）。

## 示例输入
- 产品：灵气智能手环。
- 功能：监测经脉负荷、自动绘制小型聚灵阵、危险时弹出护身符投影。
- 缺陷：佩戴者心境不稳时会误判境界，导致灵气反噬。
