# 242. 异界基建：魔法工业化流程

## 提示词内容

```json
{
  "task": "design_magic_industry",
  "product": "Mass-produced Potions / Magic Rifles",
  "process": [
    "Step 1: Slimes refine raw herbs (Automation)",
    "Step 2: Fire elementals control kiln temperature",
    "Step 3: Golems handle heavy assembly"
  ],
  "bottleneck": "Shortage of mana crystals / Union strikes by Goblins"
}
```

## 使用场景
种田/西幻文。利用魔法生物实现工业革命。

## 最佳实践要点
1.  **生物替代**：用魔法生物（史莱姆、元素）替代机器，既有奇幻感又符合逻辑。
2.  **规模化**：强调从手工作坊到流水线的转变带来的产量暴增。

## 示例输入
- 产品：量产型治疗药剂。
- 流程：史莱姆过滤药渣，火精灵恒温蒸馏，符文传送带封瓶。
- 瓶颈：高纯魔晶短缺，导致夜班产线只能降级生产。
