# 182. 中二病 (Chuunibyou) 技能咏唱

## 提示词内容

```json
{
  "task": "write_chuuni_chant",
  "skill": "Opening a soda can / Turning on a light",
  "style": "Over-the-top, Gothic, Apocalyptic",
  "chant": "By the pact of the crimson moon, I summon thee... CARBONATION RELEASE!",
  "reaction": "Bystanders are embarrassed / MC thinks he's cool"
}
```

## 使用场景
二次元/搞笑文。将日常琐事包装成毁灭世界的魔法。

## 最佳实践要点
1.  **词汇堆砌**：大量使用“深渊”、“契约”、“封印”、“混沌”等词汇。
2.  **反差萌**：咏唱越宏大，实际做的事情（如开易拉罐）越琐碎，效果越好。

## 示例输入
- 技能：打开汽水罐。
- 风格：黑暗哥特、夸张末日感，旁观者尴尬到沉默。
