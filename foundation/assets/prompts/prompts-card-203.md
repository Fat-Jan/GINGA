# 203. "老爷爷"/随身系统性格设定

## 提示词内容

```json
{
  "task": "design_ring_spirit",
  "origin": "Ancient Emperor / Fallen God / Future AI",
  "personality": "Grumpy Mentor / Sarcastic Troll / Motherly Figure",
  "quirk": "Addicted to spicy food (despite having no body) / Sleeps 23 hours a day",
  "teaching_style": "Spartan training / cryptic riddles",
  "goal": "Reconstruct their body using MC's resources"
}
```

## 使用场景
玄幻/系统文。设计主角的随身金手指人格。

## 最佳实践要点
1.  **性格鲜明**：老爷爷不应只是工具人，要有自己的喜怒哀乐和怪癖。
2.  **利益绑定**：明确老爷爷帮助主角的动机（通常是为了复活自己），增加可信度。

## 示例输入
- 来源：戒指里的未来 AI。
- 性格：毒舌、护短、爱用谜语教学，目标是重建实体。
