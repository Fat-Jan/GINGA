# 85. 都市商战布局 (反收购/做空)

## 提示词内容

```json
{
  "task": "plot_business_war",
  "target_company": "Evil Corp",
  "mc_tactic": "Short Selling / Hostile Takeover / Poaching Talent",
  "steps": [
    "Step 1: Spread rumors to lower stock price",
    "Step 2: Use dummy accounts to buy shares",
    "Step 3: Storm the board meeting"
  ],
  "climax": "The moment the villain realizes he lost control"
}
```

## 使用场景
都市商战/神豪文。设计高智商的商业博弈。

## 最佳实践要点
1.  **信息战**：商战的核心往往是信息不对称。
2.  **爽点爆发**：高潮在于主角亮牌那一刻，反派从嚣张到绝望的转变。

## 示例输入
- 目标公司：濒临财报暴雷的新能源集团。
- 主角策略：先做空，再联合小股东逼宫董事会。
- 高潮：反派在发布会现场发现供应链造假证据已同步给监管。
