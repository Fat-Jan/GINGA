# 396. 流派：签到/打卡流 (Sign-in System)

## 提示词内容

```json
{
  "task": "write_sign_in_event",
  "location": "Forbidden Area / Empress's Bedroom / Ancient Battlefield",
  "risk": "High risk of being discovered",
  "reward": "Sign-in successful: Acquired [God-Slaying Art]",
  "mc_action": "Sneaking in, touching the marker, running away",
  "aftermath": "The area collapses / Guards are alerted"
}
```

## 使用场景
系统/爽文。利用地点的特殊性来制造紧张感。

## 最佳实践要点
1.  **地点**：签到地点越危险/越私密，奖励越丰厚。
2.  **苟**：主角通常要苟住发育，不被发现。

## 示例输入
主角潜入魔教圣女的浴室打卡签到。
