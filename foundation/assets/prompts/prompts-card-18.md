# 18. 中式民俗恐怖场景构建

## 提示词内容

```json
{
  "task": "build_folk_horror_scene",
  "elements": ["{{element1}}", "{{element2}}"],
  "atmosphere": "Suffocating, Eerie",
  "sensory_focus": {
    "visual": "Color contrast (Red on White)",
    "auditory": "Opera singing, Wood creaking",
    "olfactory": "Incense, Rotting wood"
  },
  "taboo": "The rule that must not be broken in this scene"
}
```

## 使用场景
灵异/悬疑文创作。构建具有中式恐怖美学的场景。

## 最佳实践要点
1.  **文化符号**：利用特定的民俗元素（如纸人、绣花鞋）触发文化共鸣的恐惧感。
2.  **禁忌设计**：设置“Taboo”（禁忌），增加场景的互动性和紧张感。

## 示例输入
将 `{{element1}}` 替换为“无头菩萨”，`{{element2}}` 替换为“人皮灯笼”。
