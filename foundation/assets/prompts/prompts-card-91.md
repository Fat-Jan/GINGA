# 91. 宠物/召唤兽互动日常

## 提示词内容

```json
{
  "task": "write_pet_interaction",
  "pet": "Baby Dragon / Slime / Spirit Fox",
  "activity": "Eating / Sleeping / Training",
  "cute_factor": "Making noises, rubbing against MC, jealous of other pets",
  "unexpected_power": "Accidentally sneezes fire and burns the table"
}
```

## 使用场景
御兽/宠物文。通过日常互动增加萌点和读者粘性。

## 最佳实践要点
1.  **拟人化**：赋予宠物类似人类的情感（嫉妒、贪吃），使其更生动。
2.  **破坏力**：展示宠物“萌”外表下的强大力量，制造反差。

## 示例输入
- 召唤兽：幼年火龙。
- 日常：偷吃烤肉、训练喷火、因主角摸别的宠物而吃醋。
