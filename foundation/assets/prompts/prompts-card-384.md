# 384. 润色：时间流逝描写 (Passage of Time)

## 提示词内容

```json
{
  "task": "describe_passage_of_time",
  "duration": "A season / A year / A century",
  "method": "Montage / Environmental changes / Character aging",
  "imagery": "Leaves falling / Snow melting / A tree growing",
  "focus": "Routine / Repetition / Gradual change",
  "transition": "One day, something changed..."
}
```

## 使用场景
写作/过渡。用优美的文字概括漫长的时间跨度。

## 最佳实践要点
1.  **蒙太奇**：选取几个代表性的瞬间来拼凑时间流逝。
2.  **环境**：通过环境的变化（如四季更替）来暗示时间。

## 示例输入
描写主角在深山“闭关十年”的岁月流逝。
