# 31. 高情商上架感言/请假条

## 提示词内容

```json
{
  "task": "write_announcement",
  "type": "VIP Launch / Sick Leave / Writer's Block",
  "tone": "Sincere, Slightly Humorous, Humble",
  "elements": [
    "Gratitude to readers",
    "Reason (Briefly)",
    "Promise (Update schedule)",
    "Call to Action (Subscribe / Vote)"
  ],
  "forbidden": "Complaining too much, blaming platform"
}
```

## 使用场景
运营维护。撰写与读者沟通的公告。

## 最佳实践要点
1.  **真诚互动**：保持谦逊和幽默，建立良好的作者形象。
2.  **行动号召 (CTA)**：明确引导读者订阅或投票，提高转化率。

## 示例输入
- 类型：上架感言。
- 语气：真诚轻松，感谢读者，说明爆更计划和长期更新承诺。
