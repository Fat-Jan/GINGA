# 175. 异能失控/暴走状态描写

## 提示词内容

```json
{
  "task": "describe_power_overload",
  "power": "Telekinesis / Fire",
  "trigger": "Extreme rage / Grief",
  "effect_on_mc": "Nosebleed, glowing veins, loss of consciousness",
  "effect_on_environment": "Gravity reversing, objects floating / Ground melting into glass",
  "aftermath": "MC wakes up in a crater, weak and amnesiac"
}
```

## 使用场景
异能/玄幻文。展现力量的代价和恐怖。

## 最佳实践要点
1.  **环境破坏**：通过周围环境的剧烈变化（如重力反转）侧面描写力量的失控。
2.  **非人化**：描写主角在暴走状态下失去人性，变成纯粹能量体的过程。

## 示例输入
- 异能：操控重力
- 失控诱因：同伴受伤后情绪崩溃
- 描写重点：环境扭曲、身体代价、旁人恐惧
