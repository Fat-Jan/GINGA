# 163. 微短剧特效与转场脚本

## 提示词内容

```json
{
  "task": "script_vfx_transition",
  "scene_a": "MC gets slapped (Modern day)",
  "scene_b": "MC wakes up as Empress (Ancient times)",
  "transition_effect": "Match Cut (Slap -> Thunderclap) / Zoom into eye",
  "sfx": "High-pitched ringing -> Silence -> Birds chirping",
  "visual_style": "Fast, jarring, high contrast"
}
```

## 使用场景
短剧/视频脚本。设计极具视觉冲击力的转场。

## 最佳实践要点
1.  **匹配剪辑**：利用动作或声音的相似性（Match Cut）连接两个时空，流畅且震撼。
2.  **快节奏**：短剧转场必须快准狠，不能拖泥带水。

## 示例输入
- 剧情：女主发现闺蜜背叛
- 转场：咖啡杯落地切到暴雨天台
- 特效风格：短剧高对比、字幕冲击、慢动作碎片
