# 134. 学院大考/测试题目设计

## 提示词内容

```json
{
  "task": "design_exam_questions",
  "subject": "Alchemy / Magic Theory / Military Strategy",
  "difficulty": "Hard (For geniuses)",
  "questions": [
    {
      "q": "How to stabilize a Fireball in a vacuum?",
      "a": "Inject mana into the core to create a self-sustaining oxidizer."
    },
    {
      "q": "Analyze the Battle of Red Cliff's weather impact.",
      "a": "The wind direction shift was the key variable."
    }
  ],
  "mc_answer": "Unconventional but correct (Shocking the professor)"
}
```

## 使用场景
学院流/学霸文。体现主角的智商或独特见解。

## 最佳实践要点
1.  **硬核知识**：题目看起来要有一定的专业逻辑（即使是架空体系）。
2.  **创新解法**：主角的答案不能是死记硬背，必须体现创新或对本质的理解。

## 示例输入
- 科目：魔法阵稳定学
- 难度：学院终试，面向天才班
- 主角解法：用反向符文抵消灵压震荡
