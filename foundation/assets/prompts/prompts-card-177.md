# 177. 虚拟现实 (VR) 登录/登出过程

## 提示词内容

```json
{
  "task": "describe_vr_dive",
  "process": [
    "Put on helmet -> Darkness",
    "Neural link connection (Prickling sensation)",
    "Loading screen (White void / Logo)",
    "Spawn in game world (Senses return one by one)"
  ],
  "contrast": "From cramped apartment to vast fantasy landscape"
}
```

## 使用场景
网游/赛博朋克。描写现实与虚拟的切换。

## 最佳实践要点
1.  **感官重置**：描写从现实肉体感觉消失，到虚拟感官建立的过程（如先听到声音，再看到光）。
2.  **反差**：现实的逼仄阴暗与虚拟世界的宏大绚丽形成鲜明对比。

## 示例输入
- 设备：神经接入头盔。
- 过程：黑屏、白色加载空间、触觉回归、登出时眩晕与记忆延迟。
