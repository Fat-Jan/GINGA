# 38. 金手指设计 (番茄定制版)

## 提示词内容

```json
{
  "task": "design_cheat_code",
  "type": "{{type}}",
  "constraints": [
    "成长性可视化",
    "副作用/限制条件 (增加剧情张力)",
    "与主角核心欲望绑定"
  ],
  "output_structure": {
    "name": "系统/能力名称",
    "trigger_mechanism": "触发机制",
    "core_function": "核心功能",
    "upgrade_path": "升级路线 (Lv1-Lv5)",
    "first_showcase": "首秀场景 (如何震惊路人)"
  }
}
```

## 使用场景
设定阶段。设计符合番茄风格（可视化、爽感强）的金手指。

## 最佳实践要点
1.  **可视化成长**：强调 `upgrade_path`，让读者清晰看到主角的变强路径。
2.  **首秀设计**：直接规划 `first_showcase`，确保金手指一出场就带来高潮。

## 示例输入
将 `{{type}}` 替换为“直播打赏变现系统”。
