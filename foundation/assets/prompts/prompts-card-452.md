# 452. 润色：招式技能描写 (Power Visualization)

## 提示词内容

```json
{
  "task": "describe_power_move",
  "skill_name": "Rasengan / Bankai / Domain Expansion",
  "element": "Wind / Fire / Space / Mental",

  "sensory": "Sound (Humming/Roaring), Temperature, Pressure",

  "style": "Anime spectacle / Grimdark realism"
}
```

## 使用场景
同人/战斗。将抽象的技能名称转化为极具画面感的文字描写。

## 最佳实践要点
1.  **光影与音效**：不仅仅写动作，要写技能释放时的光效和独特的声音（如千鸟的鸟鸣声）。
2.  **前摇描写**：描写积蓄能量的过程（如空气震动、地面碎裂），增加期待感。

## 示例输入
描写主角释放“螺旋手里剑”的过程。重点描写风属性查克拉的高频刺耳噪音，以及像手里剑一样旋转的白色光晕。
