# 371. 润色：冷笑话/吐槽生成器 (One-Liner Generator)

## 提示词内容

```json
{
  "task": "generate_witty_remarks",
  "context": "MC falls into a trap / MC sees a weird monster",
  "style": "Sarcastic / Deadpan / Fourth-wall breaking",
  "character_voice": "A cynical system / A bored protagonist",
  "count": 5,
  "example": "If stupidity was energy, you could power a Dyson sphere."
}
```

## 使用场景
写作/润色。增加对话的幽默感和记忆点。

## 最佳实践要点
1.  **出其不意**：笑点往往来自于逻辑的突然转折。
2.  **人设**：吐槽必须符合角色的人设（如高冷的人吐槽更致命）。

## 示例输入
主角看到反派变身失败，生成几句毒舌吐槽。
