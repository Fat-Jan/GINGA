# 405. 设定：血脉进阶与返祖 (Bloodline Evolution)

## 提示词内容

```json
{
  "task": "design_bloodline_evolution",
  "trigger": "Ancestor breaks through / Fuses with Divine Blood",
  "visual": "Giant phantom over clan / Blood Qi covers the sky",
  "resonance": "Descendants feel blood boiling / Bottlenecks shattering",
  "effect": "Mass level-ups / Awakening special physiques (e.g., Dragon Body)",
  "reaction": "Clan kneels in worship / Enemies tremble",
  "outcome": "Clan strength skyrockets -> Loyalty maxed out"
}
```

## 使用场景
家族文/多子多福。表现“一人得道，鸡犬升天”的家族凝聚力高潮。

## 最佳实践要点
1.  **宏大异象**：通过天地异象烘托主角的强大。
2.  **集体爽感**：重点描写家族成员成片突破的震撼场面，而非单打独斗。

## 示例输入
主角晋升“真神”，所有拥有他血脉的后代全部自动提升一个大境界，并觉醒“荒古圣体”。
