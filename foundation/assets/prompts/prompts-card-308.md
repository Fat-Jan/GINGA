# 308. 写作工具：大纲细化助手 (Outline Expander)

## 提示词内容

```json
{
  "task": "expand_outline_point",
  "brief": "MC enters the sect exam and wins first place.",
  "expansion": [
    "Scene 1: The mockery before the exam (Setup)",
    "Scene 2: The first test (Strength) - MC surprises everyone",
    "Scene 3: The second test (Willpower) - Rival cheats but fails",
    "Scene 4: The final duel - MC uses a basic skill to win",
    "Climax: Elders fighting over MC as a disciple"
  ],
  "pacing": "Fast-paced, high conflict"
}
```

## 使用场景
通用/大纲。将一句话梗概扩充为具体的章节细纲。

## 最佳实践要点
1.  **起承转合**：把一个事件拆分为铺垫、发展、高潮、结尾。
2.  **冲突前置**：在每个小环节都加入冲突。

## 示例输入
扩写情节：主角在拍卖会上拍下了一件没人要的破烂（神器）。
