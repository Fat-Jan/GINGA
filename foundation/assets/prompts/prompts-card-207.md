# 207. 深海/亚特兰蒂斯文明设定

## 提示词内容

```json
{
  "task": "design_underwater_civ",
  "name": "Neo-Atlantis / The Abyssal Kingdom",
  "location": "Mariana Trench",
  "adaptation": "Gills / Bioluminescence / Pressure resistance",
  "architecture": "Coral spires / Transparent aluminum domes",
  "technology": "Hydro-thermal power / Taming sea monsters",
  "culture": "Worship of the Leviathan / Silence is sacred"
}
```

## 使用场景
奇幻/科幻文。构建独特的水下世界观。

## 最佳实践要点
1.  **感官差异**：水下没有火，声音传播不同，光线随深度递减，需体现这些物理特性。
2.  **生物科技**：利用海洋生物（如发光水母照明、巨鲸运输）作为科技基础。

## 示例输入
```json
{
  "name": "蓝渊王国",
  "location": "海沟热泉带",
  "culture": "说话被视为亵渎，贵族用发光纹路交流",
  "technology": "热泉发电与巨鲸运输"
}
```
