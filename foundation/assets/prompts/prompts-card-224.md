# 224. 密室杀人：机械诡计设计

## 提示词内容

```json
{
  "task": "design_locked_room_trick",
  "room": "Sealed from inside (Tape/Wax)",
  "method": "Ice dagger / String trick / Hidden mechanism",
  "execution": [
    "Killer locks door from outside using a string",
    "String is pulled through the keyhole",
    "String is burned or retrieved"
  ],
  "flaw": "A small burn mark on the door / Ash residue"
}
```

## 使用场景
悬疑/推理文。设计经典的本格推理诡计。

## 最佳实践要点
1.  **操作性**：诡计必须在物理上可行（虽然可能很复杂）。
2.  **破绽**：必须留下一个微小的物理证据供主角发现。

## 示例输入
- 房间：门窗从内侧封死的书房。
- 诡计：冰制凶器融化，细线从钥匙孔回收，留下误导性脚印。
