# 419. 流派：领主种田流 (Kingdom Building)

## 提示词内容

```json
{
  "task": "write_development_plot",

  "reaction": "Natives shocked / Subjects loyal",

}
```

## 使用场景
奇幻/历史。从无到有建设领地，利用知识改变世界。

## 最佳实践要点
1.  **基建狂魔**：详细描写新技术的应用过程和效果（如水泥城墙的坚固）。
2.  **民心所向**：描写领民生活水平的提高和对领主的崇拜。

## 示例输入
主角在贫瘠的封地推广“轮作法”和“曲辕犁”，一年后粮食爆仓，并在兽潮来袭时用火枪队守住了防线。
