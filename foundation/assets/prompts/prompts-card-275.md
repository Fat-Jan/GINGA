# 275. 恐怖：镜子里的诡异细节

## 提示词内容

```json
{
  "task": "write_mirror_horror",
  "trigger": "Looking in the mirror late at night / Washing face",
  "anomaly": "Reflection moves with a delay / Reflection smiles when MC is not / Reflection is looking elsewhere",
  "atmosphere": "Cold draft / Flickering light / Silence",
  "escalation": "The reflection starts to step out of the mirror",
  "reaction": "Paralyzing fear / Denial / Smashing the mirror"
}
```

## 使用场景
灵异/惊悚/悬疑。经典的恐怖元素，百试不爽。

## 最佳实践要点
1.  **延迟感**：最恐怖的往往是那一秒的延迟。
2.  **熟悉后的陌生**：镜子里是自己，又不是自己。

## 示例输入
洗脸时，抬头发现镜子里的自己没有抬头。
