# 66. 文娱文抄公作品发布反应

## 提示词内容

```json
{
  "task": "write_copycat_reaction",
  "work_type": "Song / Novel / Movie",
  "title": "{{work_title}}",
  "audience_reaction": [
    "Netizens: Shocked, viral memes",
    "Critics: Analyzing deep meaning (that MC didn't intend)",
    "Rivals: Jealousy, disbelief"
  ],
  "platform_metrics": "Trending #1, Sales breaking records",
  "mc_reaction": "Feigning humility / Counting money"
}
```

## 使用场景
文娱/文抄公文。描写经典作品搬运后的轰动效果。

## 最佳实践要点
1.  **过度解读**：让土著评论家深度解读作品，制造迪化爽感。
2.  **数据可视化**：用榜单排名、销量数字直观展示成功。

## 示例输入
将 `{{work_title}}` 替换为“消愁”。
