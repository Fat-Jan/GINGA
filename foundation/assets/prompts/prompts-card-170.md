# 170. 赛博格 (Cyborg) 身体恐怖描写

## 提示词内容

```json
{
  "task": "body_horror_cybernetics",
  "procedure": "Replacing an arm / Installing a neural chip",
  "sensation": "Phantom pain, cold metal touching bone, drilling sound inside skull",
  "psychology": "Dysmorphia (Is this hand still mine?)",
  "visual": "Wires weaving through muscle, oil mixing with blood"
}
```

## 使用场景
赛博朋克/惊悚。描写机械改造的痛苦与异化。

## 最佳实践要点
1.  **排异反应**：描写肉体对机械的本能排斥（发炎、疼痛），增加真实感。
2.  **自我认知**：探讨“忒修斯之船”——换了多少零件后，我还是我吗？

```json

  "body_part": "机械脊柱",
  "implant_function": "提升反应速度",
  "horror_detail": "金属节段会在梦中自行校准，发出骨骼错位声"
}
```
