# 390. 润色：通感修辞训练 (Synesthesia)

## 提示词内容

```json
{
  "task": "practice_synesthesia",
  "subject": "Music / Color / Pain",
  "method": "Describe sound using color / Describe taste using touch",
  "examples": [
    "The trumpet blared a bright yellow note.",
    "Her voice felt like velvet.",
    "The pain tasted like rusty iron."
  ],
  "goal": "Create unique and memorable descriptions"
}
```

## 使用场景
润色/文笔。让描写更加高级和艺术化。

## 最佳实践要点
1.  **跨感官**：打破感官的界限，用一种感官去形容另一种感官。
2.  **陌生化**：让读者对熟悉的事物产生新的感觉。

## 示例输入
用通感手法描写“刺骨的寒冷”。
