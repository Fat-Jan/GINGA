# 457. 润色：传说物品描写 (Legendary Item Description)

## 提示词内容

```json
{
  "task": "describe_legendary_item",
  "item_name": "Excalibur / Hylian Shield / Devil Fruit",
  "appearance": "Material, Glow, Runes, Wear & Tear",
  "aura": "Holy / Cursed / Ancient",
  "history": "Brief lore / Previous owner",
  "effect": "Passive buff / Active ability",
  "acquisition": "The moment MC touches it"
}
```

## 使用场景
同人/玄幻。通过描写物品的质感和历史，提升获得的爽感。

## 最佳实践要点
1.  **历史厚重感**：提到该物品的前任主人或传说，增加其格调。
2.  **接触反馈**：描写主角触碰物品时的特殊感觉（如电流流过全身、听到剑鸣）。

## 示例输入
描写主角获得“流刃若火”（死神）的瞬间。重点描写刀身散发的古老热浪，以及仿佛能烧尽一切的压迫感。
