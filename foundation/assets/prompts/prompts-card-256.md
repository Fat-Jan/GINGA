# 256. 克苏鲁：不可名状的梦境

## 提示词内容

```json
{
  "task": "describe_lovecraftian_dream",
  "visuals": "Non-euclidean geometry / Cyclopean cities / Tentacles in the sky",
  "sensation": "Drowning in thick air / Hearing colors / Seeing sounds",
  "entity": "A sleeping god shifting",
  "aftermath": "Waking up with a nosebleed and a strange symbol carved on arm"
}
```

## 使用场景
克苏鲁/诡秘流。描写接触古神时的精神污染。

## 最佳实践要点
1.  **通感**：打破感官界限（听到颜色、看到声音），表现精神的错乱。
2.  **无法描述**：多用“难以形容”、“违背常理”等词汇，强调人类语言的匮乏。

## 示例输入
- 梦境：主角走进一座楼梯向下却通往星空的城市。
- 感官：空气像海水一样浓稠，钟声有颜色，影子比身体先跪下。
- 余波：醒来后掌心多出一枚会呼吸的螺旋印记。
