# 208. 天空城/浮空岛生态系统

## 提示词内容

```json
{
  "task": "design_sky_city",
  "mechanism": "Anti-gravity crystals / Giant balloons / Ancient magic",
  "water_source": "Cloud harvesting / Rain catchers",

  "hazard": "High winds / Falling off / Lightning storms",
  "social_structure": "Upper city (Nobles) vs. Lower city (Shadowed slums)"
}
```

## 使用场景
奇幻/蒸汽朋克。设计垂直分布的社会结构。

## 最佳实践要点
1.  **资源限制**：水和土在天空城是稀缺资源，应围绕此设计冲突。
2.  **坠落恐惧**：时刻提醒读者“掉下去就是死”，增加环境的危险感。

## 示例输入
- 城市：靠巨鲸骨架悬浮的天空城
- 生态重点：水源循环、空中农田、飞行兽迁徙
- 危机：浮力核心开始衰减
