# 270. 科幻：巨型结构 (BDO) 描写

## 提示词内容

```json
{
  "task": "write_megastructure_description",
  "object": "Dyson Sphere / Ringworld / Space Elevator / Planet-sized Ship",
  "scale": "Blocking the sun / Extending beyond the horizon",
  "feeling": "Insignificance / Awe / Fear / Vertigo",
  "detail": "Clouds forming inside the structure / Artificial weather / Infinite metal surface",
  "perspective": "View from a tiny shuttle approaching the object"

```

## 使用场景
科幻/星际/探索。BDO (Big Dumb Object) 带来的视觉震撼。

## 最佳实践要点
1.  **巨物恐惧/崇拜**：强调人类的渺小与造物的宏大。
2.  **感官描写**：不仅仅是视觉，还有重力的变化、压迫感。

## 示例输入
主角第一次驾驶飞船接近“戴森球”。
