# 292. 写作：伏笔埋设技巧 (Foreshadowing Tool)

## 提示词内容

```json
{
  "task": "generate_foreshadowing",
  "future_event": "The betrayal of the best friend / The hero's death / The true villain",
  "clue_type": "Dialogue slip / Object detail / Weather omen / Recurring dream",
  "subtlety": "High (Reader won't notice until re-read) / Medium (Suspicious)",
  "example": "He always rubbed his left ring finger (where the ring used to be)",
  "placement": "Chapter 3, casual conversation"
}
```

## 使用场景
通用/写作辅助。帮助作者设计精妙的伏笔。

## 最佳实践要点
1.  **看似无意**：伏笔要伪装成闲笔或环境描写。
2.  **呼应**：确保生成的伏笔在后文有明确的回收计划。

## 示例输入
想为“师父其实是幕后黑手”埋一个伏笔。
