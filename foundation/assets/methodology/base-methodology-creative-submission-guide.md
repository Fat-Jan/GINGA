# 网文平台投稿规范指南

该资产用于投稿前的最后一轮平台适配和风险自查。它不替代平台最新规则，应作为内部检查清单和差异化改写依据。

## 输出格式

建议输出 `platform`、`pass_risk`、`required_changes`、`redline_notes`、`submission_readiness` 五类字段。
