# 古代官职层级校验指南

本资产用于拦截古代题材中“名词像真的、权力链是错的”的硬伤。它服务设定和正文校验，不替代专业史料。

## 最小事实门

每次使用官职前，至少回答：时代口径是什么，角色的正式身份是什么，能管谁，不能管谁，当前行为是否需要授权。
