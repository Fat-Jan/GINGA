# 平台分析指南

平台分析用于把同一个创意改写成不同平台能理解的卖点。不要只问“能不能发”，要问“在哪个平台最容易被读者读懂并持续追读”。

## 分析字段

输出至少包含平台定位、读者入口、创新接受度、开篇策略、标签策略、审核风险和签约路径。
