# 题材跨平台适配矩阵

该资产用于立项时选择平台或判断同一题材是否需要改写。评级只代表适配入口，不能替代具体平台榜单和审核规则。

## 评级解释

星级越高，表示题材、读者期待、商业化方式和平台推荐逻辑越一致。低星并非绝对不能写，而是需要更强差异化或改包装。
