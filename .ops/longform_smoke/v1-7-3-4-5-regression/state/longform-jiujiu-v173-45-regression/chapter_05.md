# 第5章 · 蚀骨之尘

黑暗并非虚无，而是某种粘稠的、具有重量的实体。它像冷却后的沥青，层层叠叠地包裹着意识，将时间切割成破碎的片段。无明在窒息的边缘挣扎，肺部像是被灌入了碎玻璃，每一次呼吸都伴随着尖锐的撕裂感。他试图睁开眼，但眼皮沉重得如同铅块，视野中只有混沌的红黑交织，仿佛视网膜上烙印着尚未褪去的血腥残影。

记忆是一片荒芜的冻土。没有名字，没有过往，只有本能——一种刻入骨髓的杀戮本能，以及对“存在”本身的极度恐惧。这种恐惧比死亡更寒冷，因为它意味着他可能从未真正活过，或者早已死去，此刻不过是某种执念的回光返照。

指尖传来冰冷的触感。那是金属的寒意，粗糙而真实。无明的手指痉挛般地收紧，握住了一柄短刃。刀柄上的纹路硌进掌心，带来一丝微弱的痛觉，这痛觉成了他与现实世界唯一的锚点。他猛地吸气，胸腔剧烈起伏，终于强行撬开了沉重的眼睑。

映入眼帘的不是天空，而是一道横亘于天地之间的巨大裂痕。那便是“天堑”。它并非地理意义上的峡谷，而是空间结构的崩塌处，呈现出一种病态的灰紫色。裂隙深处翻滚着浑浊的气流，偶尔迸发出暗红色的电火花，像是在嘲笑这片大地的脆弱。无明发现自己正躺在天堑边缘的一块黑色岩石上，岩石表面布满了细密的裂纹，仿佛随时会崩解坠入下方的深渊。

周围死寂无声，连风都被冻结。空气中弥漫着一种难以言喻的味道，像是铁锈混合着腐烂的内脏，又夹杂着某种甜腻的香气，令人作呕。无明撑着身体坐起，动作僵硬而迟缓。他的肌肉酸痛欲裂，每一根神经都在尖叫，提醒着他身体的极限。但他顾不上这些，目光死死锁定在自己手中的短刃上。

这是一柄造型古朴的短刃，刃身呈暗哑的黑色，没有任何反光，仿佛能吞噬光线。刀脊上刻满了扭曲的符文，那些符文并非静止不动，而是在微微蠕动，如同拥有生命的寄生虫。当无明的视线聚焦在符文上时，脑海中突然闪过一阵剧烈的刺痛，一些模糊的画面碎片涌现：鲜血喷溅的声音、骨骼断裂的脆响、以及一双双充满绝望与贪婪的眼睛。

“这是……我的？”无明在心中默问，声音沙哑得如同砂纸摩擦。没有人回答，只有天堑深处传来的低沉轰鸣，像是巨兽沉睡时的鼾声。

他低下头，审视自己的身体。衣衫褴褛，沾满了干涸的血迹和泥土。皮肤苍白得近乎透明，隐约可见皮下青紫色的血管。然而，最令他感到不安的是胸口的位置。那里有一团诡异的阴影，正在缓慢地扩散。那不是淤青，也不是伤口，而是一种深入肌理的异物感。仿佛有什么东西在他的体内扎根，汲取着他的生命力。

无明抬起手，颤抖着解开衣襟。胸口的皮肤下，隐约可见无数细小的颗粒在游走。它们呈现出一种黯淡的灰色，像是尘埃，却又有着金属般的质感。每当这些颗粒移动，就会带来一阵钻心的剧痛，仿佛有无数根针在刺穿他的内脏。

“微粒……”这个词突兀地跳入脑海，带着一种本能的厌恶与渴望。在这个末日般的世界里，微粒是通货，是力量，也是诅咒。总量硬上限八亿四千万，这意味着每一个微粒的流动都伴随着血腥的争夺。而他体内这些游离的微粒，显然是被人强行植入的。

是谁？为什么？

无明握紧短刃，指节泛白。他必须弄清楚这一切。他站起身，双腿发软，险些跌倒。但他强迫自己站稳，目光扫视四周。天堑边缘是一片荒原，远处矗立着几座破败的建筑轮廓，像是巨兽的骸骨。空气中那种甜腻的香气愈发浓烈，源头似乎就在那片废墟之中。

他迈开脚步，每一步都踩在松软的灰烬上，发出轻微的沙沙声。影迹潜行的本能在他体内苏醒，尽管身体虚弱，但他的步伐依然轻盈而隐蔽，如同幽灵般穿梭在阴影之间。他不敢大意，在这片土地上，任何声响都可能引来掠食者。

随着距离废墟越来越近，那股甜腻的气味变得令人窒息。无明捂住口鼻，眉头紧锁。他看到前方的地面上散落着一些奇怪的物体，像是某种生物的甲壳，又像是破碎的机械零件。在这些杂物中间，躺着几具尸体。

尸体已经高度腐败，呈现出一种诡异的紫黑色。他们的皮肤溃烂，露出下面灰白色的骨骼。而在尸体的胸腔位置，都有一个空洞，仿佛心脏被某种力量生生挖去。无明蹲下身，仔细检查其中一具尸体。他发现尸体的手指紧紧攥着一块黑色的晶体，晶体表面闪烁着微弱的光芒。

当他触碰到那块晶体的瞬间，体内的灰色微粒突然躁动起来。一股热流从胸口涌向手臂，短刃上的符文也随之亮起，散发出幽蓝色的光芒。无明心中一惊，连忙缩回手。但那股热流并未消退，反而顺着经脉蔓延至全身，带来一种奇异的充盈感。

他意识到，这块晶体中蕴含着高纯度的微粒。而那些死者，正是因为无法承受微粒的冲击，或者被他人掠夺了微粒，才落得如此下场。在这个规则怪谈式的世界里，微粒不仅是财富，更是生存的基石。没有微粒，人就会变成行尸走肉；拥有过多微粒，则会引来杀身之祸。

无明深吸一口气，强压下心中的悸动。他不能在这里久留。那股甜腻的香气背后，隐藏着巨大的危险。他站起身，准备离开。然而，就在这时，一阵细微的脚步声从废墟深处传来。

脚步声很轻，很有节奏，不像是野兽，更像是人类。无明立刻进入警戒状态，身体紧绷，短刃横在胸前。他屏住呼吸，利用周围的阴影隐藏自己的身形。影迹潜行的能力让他几乎与环境融为一体，除非对方拥有极高的感知力，否则很难发现他的存在。

脚步声越来越近，最终停在了那几具尸体旁。一个身影从黑暗中走出。那是一个身材瘦削的男人，穿着一件破烂的风衣，脸上戴着一个狰狞的面具，面具上画着哭泣的笑脸。男人的手中拿着一把弯刀，刀刃上还滴着新鲜的血液。

男人蹲下身，开始搜刮尸体身上的物品。他的动作熟练而冷漠，仿佛在处理一堆垃圾。无明静静地观察着，心中盘算着对策。这个男人身上散发着一股强大的气息，显然是一名经验丰富的猎手。正面交锋，胜算极低。

就在男人即将拿起那块黑色晶体时，无明动了。他没有选择攻击，而是制造了一点动静——他用脚尖踢飞了一块小石子，石子滚落在远处的瓦砾堆中，发出清脆的响声。

男人猛地抬头，面具下的眼睛警惕地扫视四周。“谁？”他的声音沙哑而阴冷。

无明没有回答，而是继续隐藏在阴影中，等待着对方的下一步行动。他知道，在这种环境下，耐心往往比武力更重要。

男人站起身，弯刀横在身前，一步步向声音传来的方向走去。他的步伐谨慎，每一步都试探着地面的虚实。无明看着他的背影，眼中闪过一丝寒光。他在等待一个机会，一个能够一击必杀的机会。

然而，就在男人经过一块巨大的岩石时，异变突生。岩石表面的裂缝中突然伸出一只枯瘦的手，紧紧抓住了男人的脚踝。男人惊呼一声，身体失去平衡，重重地摔倒在地。紧接着，更多的枯手从岩石缝隙中伸出，将他牢牢抓住。

无明瞳孔微缩。这不是普通的陷阱，而是某种规则类的怪谈现象。在这片天堑边缘，存在着许多不可名状的禁忌。一旦触犯，便会遭到规则的惩罚。那个男人显然触犯了某种禁令，或许是踩踏了特定的区域，或许是发出了不该发出的声音。

男人拼命挣扎，弯刀挥舞，砍断了数只枯手。但那些枯手仿佛无穷无尽，源源不断地从岩石中涌出。很快，男人的身体就被完全覆盖，只剩下凄厉的惨叫声在空气中回荡。

无明没有上前救援，也没有趁机逃跑。他冷静地观察着这一幕，试图从中找出规律的线索。他发现，那些枯手只在男人倒下后才出现，而且它们的动作虽然迅猛，却缺乏灵活性，更像是某种程序化的反应。

片刻后，惨叫声戛然而止。岩石恢复了平静，那些枯手也缩回了缝隙中。地面上只剩下一滩血迹，以及那把掉落的弯刀。至于那个男人，仿佛从未存在过一般，消失得无影无踪。

无明缓缓走出阴影，来到岩石前。他小心翼翼地避开刚才男人倒下的区域，捡起那把弯刀。弯刀入手沉重，刃口锋利，是一件不错的武器。但他并没有将其据为己有，而是随手扔进了旁边的深渊。他不信任任何外来的物品，尤其是沾染了死亡气息的东西。

他转身看向那块黑色晶体。晶体依然静静地躺在那里，散发着诱人的光芒。无明犹豫了片刻，最终还是走了过去。他需要用微粒来强化自己的身体，否则根本无法在天堑边缘生存下去。

他伸出手，指尖轻轻触碰晶体。这一次，他没有退缩。体内的灰色微粒再次躁动，但与之前不同，这次它们表现出了一种强烈的渴望。短刃上的符文光芒大盛，仿佛在引导着什么。

无明咬紧牙关，忍受着体内翻江倒海般的痛苦。他感觉到一股庞大的能量涌入体内，冲刷着他的经脉。那些灰色的微粒开始融合、重组，逐渐转化为一种更加纯净的力量。他的视力变得更加清晰，听力也更加敏锐，甚至能听到远处风声的细微变化。

然而，代价也是巨大的。他的七窍开始流血，皮肤表面浮现出密密麻麻的血珠。这是一种透支生命力的表现。如果不能及时控制这股力量，他很可能会爆体而亡。

无明跪倒在地，大口喘息。他紧握短刃，将所有的意志力集中在一点，强行压制体内暴走的能量。汗水浸透了衣衫，他的身体剧烈颤抖，仿佛随时都会崩溃。

就在这时，短刃上的符文突然停止了蠕动，转而凝固成一种复杂的图案。图案的中心，是一个微小的漩涡，仿佛在吞噬着周围的一切。无明惊讶地发现，自己体内的微粒竟然开始向这个漩涡汇聚，并被其吸收、转化。

这是一种前所未有的体验。他感觉到自己的内宇宙正在发生某种变化，原本混乱的能量结构开始变得有序。虽然过程痛苦万分，但他能清晰地感受到力量的增长。

不知过了多久，体内的风暴终于平息。无明瘫软在地上，浑身无力。但他知道，自己活下来了。不仅活了下来，还获得了一丝掌控微粒的能力。

他抬起头，看向天堑深处的灰紫色雾气。那里隐藏着更多的秘密，更多的危险，以及更多的机遇。失忆的刺客无明，在这一刻，正式踏上了这条充满荆棘的道路。

他站起身，擦去嘴角的血迹，眼神变得冰冷而坚定。手中的短刃微微颤动，仿佛在回应着他的决心。前方是未知的黑暗，但他已不再迷茫。因为他知道，唯有不断掠夺、不断进化，才能在这残酷的世界中活下去。

风吹过，卷起地上的灰烬，形成一个个小小的漩涡。无明迈步向前，身影逐渐融入黑暗之中。在他身后，那块曾经放置黑色晶体的地方，留下了一道淡淡的痕迹，仿佛预示着某种新的规则正在诞生。
