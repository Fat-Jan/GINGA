# 第4章 · 脏腑里的窃贼

痛觉是第一个回归的感官，像是一根烧红的铁钎，从尾椎骨一路贯穿至天灵盖，将混沌的意识强行钉回这具残破的躯壳。

无明睁开眼，视野中是一片浑浊的灰白。没有天空，只有厚重得令人窒息的尘埃云团，如同腐烂的内脏壁膜，层层叠叠地包裹着这片被遗忘的废土。空气里弥漫着硫磺与陈旧血液混合的腥甜气味，每一次呼吸都像是在吞咽砂砾，肺叶发出拉风箱般的嘶鸣。他试图动弹手指，却发现肢体沉重如铅，仿佛每一块肌肉都被灌入了水银。

这里是“天堑”的边缘。

传说中，世界被四重不可逾越的天堑切割成孤岛，而他所处的位置，正是第一重天堑与第二重天堑之间的夹缝——“弃婴之地”。这里是被规则遗弃者的坟墓，也是微粒掠夺者们的狩猎场。

无明缓缓坐起，脊椎发出令人牙酸的咔吧声。他的记忆是一片空白，就像一张被烈火燎过的羊皮纸，只剩下焦黑的边缘和无法辨认的残痕。他不记得自己是谁，不记得为何会躺在这堆枯骨之中，更不记得手中紧握的那柄短刃从何而来。

那是一柄漆黑的短刃，刃身并非金属，而是一种类似黑曜石的物质，表面刻满了细密繁复的古老符文。那些符文在昏暗的光线下微微搏动，像是某种沉睡生物的心跳。当他的指尖触碰到刃柄时，一股冰冷的刺痛感顺着神经末梢蔓延开来，与此同时，脑海中闪过一个模糊的画面：无数微小的光点在黑暗中汇聚、吞噬、重组。

微粒。

这个词突兀地出现在他的意识深处，带着一种本能的贪婪与恐惧。在这个末日世界里，微粒不仅是能量，是货币，更是生命的本质。总量硬上限八亿四千万，每一粒都沾染着鲜血。

突然，一阵剧烈的痉挛袭击了他的腹部。

无明闷哼一声，双手死死按住小腹。那不是普通的疼痛，而是一种异物入侵的撕裂感。他能清晰地感觉到，有什么东西在他的肠胃里蠕动、啃噬。那种感觉既恶心又恐怖，仿佛有一只看不见的虫子正沿着他的食道逆流而上，试图钻进他的大脑。

他低下头，看向自己的腹部。衣衫褴褛之下，皮肤呈现出一种病态的青紫色，而在肚脐周围，隐约可见几条细小的黑色纹路，如同蛛网般向四周扩散。那些纹路在皮下跳动，每一次搏动都伴随着一阵钻心的剧痛。

“该死……”

无明咬着牙，冷汗瞬间浸透了后背。他意识到，自己不仅仅是一个失忆者，更是一个容器。有人——或者有什么东西——在他昏迷期间，将未知的微粒植入了他的体内。这些微粒并不安分，它们正在争夺控制权，试图将这具身体变成它们的巢穴。

他必须立刻处理掉这些寄生虫，否则不用等到敌人出现，他自己就会先崩溃成一滩肉泥。

无明深吸一口气，强迫自己冷静下来。尽管记忆缺失，但身体的本能还在。他的右手紧紧握住短刃，左手则探入怀中，摸索出一块粗糙的石片。这是他在醒来时身边唯一的工具，虽然简陋，但在绝境中足以成为救命稻草。

他将石片抵在腹部的黑色纹路上，用力按压。剧痛让他眼前发黑，但他没有丝毫退缩。随着压力的增加，那些黑色纹路开始剧烈颤抖，仿佛受到了惊吓的蛇群。紧接着，一股温热的液体从他的嘴角溢出，带着浓烈的铁锈味。

血。

但不是红色的血，而是夹杂着银色光泽的黑色粘液。

无明瞳孔骤缩。他猛地松开石片，举起短刃，毫不犹豫地刺向自己的腹部。

噗嗤。

刀刃没入血肉的声音沉闷而湿润。没有惨叫，只有喉咙里发出的低沉呜咽。无明的眼神冷冽如冰，仿佛正在切割的不是自己的身体，而是一块腐肉。他手腕翻转，刀尖在腹腔内搅动，精准地挑断了那几根黑色的脉络。

刹那间，一团黑色的雾气从伤口喷涌而出，在空中凝聚成无数细小的颗粒。那些颗粒闪烁着诡异的光芒，像是拥有生命一般，疯狂地撞击着周围的空气，发出细微的嗡嗡声。

这就是植入他体内的微粒。

它们试图逃逸，试图重新寻找宿主。但无明不会给它们这个机会。

他手中的短刃突然亮起，那些古老的符文仿佛被鲜血唤醒，散发出幽蓝色的光芒。一股强大的吸力从刃身爆发出来，如同黑洞般将那团黑色雾气强行拉扯过去。微粒们在空中挣扎、扭曲，最终一个个被吸入刃身的符文之中，消失不见。

随着最后一粒微粒被吞噬，无明感到体内的剧痛骤然减轻。取而代之的，是一种奇异的充盈感。那些被吸收的微粒并没有消散，而是通过短刃转化为一股暖流，涌入他的四肢百骸。他的视力变得更加清晰，听力也敏锐了许多，甚至能听到远处风吹过废墟的呼啸声。

然而，这种力量的代价是巨大的。

无明感到一阵强烈的眩晕，眼前的景象开始扭曲。他看到周围的空气中漂浮着无数细小的光点，那是游离的微粒，是这个世界最基础的构成单位。它们无处不在，却又难以捕捉。而他手中的短刃，似乎成为了连接他与这些微粒的桥梁。

“原来如此……”

他低声喃喃，声音沙哑得像是在砂纸上摩擦。他终于明白，这柄短刃不仅仅是一件武器，更是一个媒介，一个能够吞吐、转化微粒的规则载体。而那些古老的符文，则是控制这一过程的钥匙。

就在这时，远处传来了一阵沉重的脚步声。

咚、咚、咚。

每一步都像是踩在无明的心跳上，节奏缓慢而坚定。伴随着脚步声的，还有金属拖拽在地面上的刺耳声响。

无明迅速收敛气息，身体紧绷如弓弦。他环顾四周，发现这是一片巨大的废墟广场，周围散落着断裂的石柱和破碎的雕像。广场中央有一座高耸的祭坛，祭坛顶端悬挂着一具干枯的尸体，尸体的胸口插着一把长剑，剑柄上镶嵌着一颗黯淡的红宝石。

脚步声越来越近，一个高大的身影从迷雾中走出。

那是一个身穿重型铠甲的男人，脸上戴着一张狰狞的铁面具，只露出一双充满血丝的眼睛。他的手中拖着一把巨大的链锯剑，锯齿上还挂着未干的血肉。在他的身后，跟着两个身材瘦小的随从，他们手里拿着奇怪的仪器，正在扫描着地面的痕迹。

“找到了。”

铁面男停下脚步，目光锁定在无明身上。他的声音经过面具的过滤，变得沉闷而失真，“那个‘容器’还活着。”

无明没有说话，只是冷冷地盯着对方。他的右手紧紧握着短刃，左手悄悄摸向腰间，那里藏着一枚从刚才战斗中捡到的碎石。

“交出你体内的微粒，我可以让你死得痛快一点。”铁面男举起链锯剑，引擎启动，发出轰鸣声，“否则，我会把你切成碎片，一点一点地挖出来。”

无明嘴角勾起一抹嘲讽的弧度。他没有回答，而是身形一晃，消失在原地。

影迹潜行。

这是他在觉醒后获得的第一项能力，虽然等级不高，但在近距离突袭中足以致命。

铁面男显然没想到对方会有这样的速度，他愣了一下，随即怒吼一声，挥动链锯剑向四周横扫。火花四溅，碎石纷飞，但无明早已不在那里。

下一秒，无明出现在铁面男的侧后方，短刃如毒蛇出洞，直刺对方的咽喉。

铛！

一声清脆的金属撞击声响起。铁面男的脖子上竟然覆盖着一层透明的能量护盾，短刃刺在上面，只留下一道浅浅的白痕。

“有点意思。”铁面男冷笑一声，反手一拳轰向无明的胸口。

无明侧身闪避，拳风擦着他的脸颊掠过，带起一阵劲风。他顺势后退几步，拉开距离，心中暗自警惕。这个男人的实力远超他的预期，不仅装备精良，而且反应极快。更重要的是，他身上散发着一股浓郁的微粒波动，显然是一个高阶的掠夺者。

“别挣扎了。”铁面男一步步逼近，链锯剑再次启动，“在这座城里，没有人能逃过‘清道夫’的追捕。你的命运早已注定。”

清道夫？

无明脑海中闪过一丝疑惑。这个名字听起来熟悉，却又遥远。他隐约记得，在某些破碎的记忆片段中，曾出现过类似的称呼。那是专门负责清理违规者、回收微粒的特殊部队，隶属于某个庞大的组织。

难道自己曾经是他们的一员？还是说，自己是他们的猎物？

没有时间思考了。铁面男已经冲到了面前，链锯剑带着毁灭性的力量劈下。无明不得不全力应对，他挥舞短刃，格挡开攻击，同时寻找对方的破绽。

两人的身影在废墟中交错，刀光剑影间，火星四溅。无明的动作敏捷而狠辣，每一击都直指要害；而铁面男则依靠厚重的铠甲和强大的力量，稳扎稳打，步步为营。

随着时间的推移，无明逐渐感到体力不支。体内的微粒虽然提供了一定的能量补充，但毕竟数量有限，无法支撑长时间的高强度战斗。而铁面男却越战越勇，身上的能量护盾始终坚不可摧。

必须速战速决。

无明眼中闪过一丝决绝。他故意露出一个破绽，引诱铁面男进攻。果然，铁面男上当了，他大喝一声，链锯剑全力劈下。

就在刀刃即将触及无明肩膀的瞬间，无明身形一矮，从对方的腋下穿过，短刃顺势向上挑起，直刺铁面男腋下的铠甲缝隙。

这一次，没有护盾的保护。

噗嗤。

短刃深深没入血肉，鲜血喷涌而出。铁面男发出一声凄厉的惨叫，链锯剑脱手落地。他踉跄着后退几步，捂住伤口，难以置信地看着无明。

“你……怎么可能……”

无明没有给他喘息的机会。他拔出短刃，再次扑上前去，一刀割断了铁面男的喉咙。

鲜血染红了地面，铁面男的身体抽搐了几下，便不再动弹。那两个随从见状，吓得魂飞魄散，转身就想逃跑。

无明冷哼一声，身形如鬼魅般追上，两记手刀精准地击中他们的后颈。两人应声倒地，昏死过去。

战斗结束得很快，但无明并没有感到轻松。他看着地上的尸体，心中涌起一股莫名的空虚感。杀人对他来说，似乎并不陌生，甚至有一种诡异的熟练感。这种感觉让他感到恐惧，因为他不知道这种杀戮的本能究竟源自何处。

他走到铁面男的尸体旁，蹲下身，开始搜查战利品。除了那把链锯剑和一些零散的物资外，他在铁面男的口袋里找到了一块黑色的令牌。令牌上刻着一个复杂的图案，看起来像是一只张开翅膀的乌鸦。

“鸦巢……”

无明轻声念出令牌上的名字。这是一个陌生的词汇，但却让他感到一阵心悸。他隐约觉得，这个名字背后隐藏着巨大的秘密，或许与他失忆的真相有关。

他将令牌收好，站起身来，望向远方。迷雾依旧浓重，看不清前路。但他知道，自己不能再留在这里了。刚才的战斗动静很大，很快就会引来更多的敌人。

他必须离开，去寻找答案，去寻找那些被遗忘的记忆。

无明握紧短刃，感受着刃身传来的微弱震动。那些被吸收的微粒正在他的体内流转，带来一种奇异的力量感。他知道，这只是开始。在这个被规则支配的世界里，想要活下去，就必须变得更强，更冷酷，更无情。

他迈开脚步，向着迷雾深处走去。身后的废墟渐渐远去，前方的道路充满了未知与危险。但他没有回头，因为对于失忆者来说，过去已死，唯有未来可期。

而在他的体内，那些被吸收的微粒正在悄然发生变化。它们不再仅仅是能量，而是开始与他的血脉融合，形成一种新的平衡。这种平衡脆弱而危险，一旦失控，后果不堪设想。

无明并不知道这一点。他只是觉得，自己的身体似乎发生了一些微妙的变化。他的感官变得更加敏锐，思维也更加清晰。他甚至能感觉到，周围空气中的微粒流动轨迹，仿佛能看到一条条无形的丝线，连接着万物。

这是一种全新的视角，也是一种全新的诅咒。

他抬起头，看向头顶那片灰白的天空。在那里，隐约可见一道巨大的裂痕，横贯天际。那就是第一重天堑，隔绝了生与死的界限。

无明眯起眼睛，心中涌起一股莫名的冲动。他想知道，天堑的另一边是什么？是地狱，还是天堂？或者是另一个更加残酷的世界？

这个问题没有答案，至少现在没有。

他继续前行，身影逐渐融入迷雾之中，仿佛从未存在过一般。只有地面上残留的血迹，证明这里曾经发生过一场生死搏杀。

而在远处的阴影中，一双眼睛正静静地注视着这一切。那双眼睛冷漠而深邃，仿佛在审视一只蝼蚁的命运。

“有趣。”

一个低沉的声音在黑暗中响起，“竟然能吞噬‘蚀心微粒’而不疯魔。看来，这次的实验体，比预期的要有趣得多。”

声音落下，身影也随之消失，仿佛从未出现过。
