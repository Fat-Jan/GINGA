# 第8章 · 脏腑里的窃贼

痛觉不是从皮肤表面传来的，而是从骨髓深处炸开的。

无明睁开眼时，视野是一片浑浊的灰白。没有天空，没有地面，只有无数悬浮在空气中的尘埃，像死去的星屑一样缓慢旋转。他试图呼吸，肺部却像是被灌入了滚烫的铁水，每一次吸气都伴随着尖锐的撕裂感。他低下头，看见自己的胸膛微微起伏，但那节奏并不属于他自己——那是一种诡异的、同步的搏动，仿佛他的心脏旁边，还寄生着另一颗更小、更贪婪的心脏。

他躺在一片由破碎石板堆砌而成的废墟上。四周是扭曲的建筑残骸，那些曾经高耸入云的塔楼如今只剩下断裂的骨架，像巨兽死后暴露出的肋骨，刺向那片永远无法穿透的灰雾。这里是天堑的边缘，是被世界遗弃的排泄口。空气中弥漫着一股甜腻而腐朽的味道，那是高浓度微粒泄漏后与血肉混合发酵的气息。

无明挣扎着坐起，右手本能地摸向腰间。短刃还在。那柄刻有古老符文的金属利器冰冷刺骨，是他此刻唯一能确认的真实。他握住刀柄，指节因用力而发白，但随即，一阵剧烈的痉挛从手臂蔓延至全身。他低头看向自己的左手手背，那里的皮肤下正隆起一个个细小的鼓包，像是在皮下蠕动的虫卵。

“咳……”

一口黑血吐在地上。血液并未渗入石缝，而是迅速凝结成黑色的晶体，随后崩解为无数微小的光点，消散在空气中。无明瞳孔骤缩。那是微粒。他的血里含有微粒。而且，这些微粒正在主动逃离他的身体，或者说，正在被某种力量强行抽取。

记忆是一片空白，就像这漫天的灰雾。但他记得那种感觉——被植入的感觉。就在不久之前，有人将什么东西塞进了他的体内。不是药物，不是器械，而是一种活着的、具有意识的能量体。它在他的血管里游弋，啃食着他的生命力，将其转化为最纯粹的通货。

“该死的杂种。”无明低声咒骂，声音沙哑如磨砂纸摩擦。他强忍着剧痛，用短刃的尖端挑开左手手腕的皮肤。没有鲜血流出，只有一团暗红色的雾气涌出，雾气中隐约可见细密的金色丝线，它们纠缠在一起，构成了一张微缩的网。

这就是那个集团的手段。他们不杀人，他们养人。他们将这种名为“蚀心蛊”的微粒变种植入猎物的体内，让猎物成为移动的微粒矿藏。只要宿主还活着，微粒就会源源不断地产生，并通过特定的频率被远程收割。这是一种比死亡更残忍的剥削，它将人的存在本身货币化，将痛苦量化为数字。

无明冷笑一声，眼中的凶光逐渐凝聚。他不信任任何人，尤其是那些躲在幕后操纵规则的人。既然你们想把我变成提款机，那我就先把自己变成绞肉机。

他闭上眼，调动体内仅存的灵力。影迹潜行的本能在他脑海中苏醒，虽然他现在无法完全隐匿身形，但他可以感知周围能量的流动。他感觉到，那股来自体内的吸力并非无源之水，它指向西北方向，那里有一座半坍塌的神庙，神庙顶端闪烁着微弱却稳定的红光。

那是信号塔。也是收割者的巢穴。

无明站起身，双腿有些发软，但他强迫自己站稳。每一步落下，脚底传来的震动都让他感到恶心，仿佛大地也在排斥这个被污染的灵魂。他拖着沉重的步伐，向着红光走去。沿途，他看到了其他“矿工”。

那是一个蜷缩在墙角的女人，她的腹部高高隆起，但不是因为怀孕，而是因为体内积聚了过多的未排出微粒。她的皮肤呈现出半透明的青色，血管清晰可见，里面流淌的不是血，而是金色的液体。她双眼空洞地望着前方，嘴里喃喃自语：“还要更多……还要更多……孩子需要营养……”

在无明的世界观里，繁衍是神圣的契约，是延续血脉的唯一途径。但在天堑边缘，繁衍变成了诅咒。为了获得保护后代所需的微粒配额，父母们不得不接受这种植入，用自己的生命去换取孩子的生存权。这是一种扭曲的多子多福，是用父母的枯竭来浇灌下一代的繁荣。

女人似乎察觉到了无明的靠近，她猛地转过头，眼球已经完全变成了金色。“你……你也来了？”她的声音尖利而破碎，“把微粒给我……我的孩子饿了……”

无明停下脚步，冷冷地看着她。他没有怜悯，只有警惕。在这个地狱里，同情心是最廉价的毒药。他握紧短刃，符文在刀身上微微亮起，发出低沉的嗡鸣。

“滚开。”他说。

女人发出一声凄厉的尖叫，四肢着地，像一只蜘蛛般向他扑来。她的速度快得惊人，指甲锋利如刀，直取无明的咽喉。无明侧身闪避，动作流畅得如同本能。他在空中转身，短刃划出一道寒光，精准地切断了女人的左臂。

断臂落地，瞬间化作一团金色的烟雾消散。女人惨叫一声，却没有退缩，反而更加疯狂地扑上来。她的伤口处没有流血，只有更多的金色微粒喷涌而出，形成了一道道致命的射线。

无明眯起眼睛，他意识到这不是普通的攻击。这些微粒带有强烈的腐蚀性，一旦接触皮肤，就会立刻渗透进去，加剧体内的负担。他不能硬接。

他深吸一口气，体内的灵力疯狂运转，压制住那股想要吞噬他的吸力。他将所有的注意力集中在短刃上，感受着刀身传来的震动。符文似乎在回应他的情绪，变得越来越烫。

“破。”

无明低喝一声，身形骤然消失。影迹潜行发动，他在原地留下一个残影，真身已经出现在女人的身后。短刃无声无息地刺入女人的后颈，直抵脊椎。

女人的动作僵住了。她眼中的金光迅速黯淡，身体开始剧烈颤抖。无明没有拔出短刃，而是手腕一转，刀刃在体内搅动，彻底破坏了她的神经中枢。

女人瘫软在地，身体开始崩解。这一次，没有金色的烟雾，只有大量的黑色灰烬飘落。随着她的死亡，一股无形的波动扩散开来，无明感到体内的吸力突然减弱了一瞬。

他喘着粗气，靠在墙壁上。刚才那一击消耗了他大量的体力，体内的微粒残留变得更加活跃，它们在欢呼，在庆祝宿主的虚弱。无明咬紧牙关，从口袋里掏出一块粗糙的布条，紧紧缠住左手手腕，试图阻止微粒的外泄。

他知道，这只是开始。那个女人只是一个哨兵，真正的收割者还在神庙里等着他。如果他不能切断信号源，他最终也会变成那样一具空壳。

无明继续前行。越靠近神庙，空气中的压迫感就越强。周围的建筑变得更加扭曲，仿佛空间本身都在发生畸变。地面上出现了许多奇怪的符号，那是规则怪谈的禁令标识。

“禁止直视红光。”
“禁止在阴影中停留超过三秒。”
“禁止承认自己是人类。”

无明无视这些标语。在他看来，这些都是控制人心的枷锁。他小心翼翼地避开地上的阴影，每一步都计算得精确无比。他的心跳越来越快，不是因为恐惧，而是因为兴奋。一种嗜血的兴奋。

终于，他来到了神庙的入口。那是一座巨大的石门，门上雕刻着无数张痛苦的人脸，他们的嘴巴张开，仿佛在无声地呐喊。石门紧闭，但门缝中透出的红光却如同鲜血般浓郁。

无明站在门前，感受着体内那股吸力的源头。就在这里。他能听到里面传来的声音，那是无数细微的咀嚼声，像是亿万只蚂蚁在啃食骨头。

他举起短刃，用刀柄重重地敲击石门。

“咚。”

声音沉闷而遥远。

片刻后，石门缓缓打开。一股热浪扑面而来，夹杂着浓烈的血腥味和微粒的甜味。无明迈步走入黑暗，身影瞬间被红光吞没。

神庙内部是一个巨大的圆形大厅，中央矗立着一座由白骨堆砌而成的祭坛。祭坛上方，悬浮着一颗巨大的红色晶体，晶体内部流淌着金色的液体，正是那些被掠夺来的微粒。而在祭坛周围，跪坐着数十个身穿黑袍的人，他们低着头，双手合十，口中念念有词。

无明的出现并没有引起他们的注意，或者说，他们根本不在意一只闯入的蝼蚁。直到无明走到祭坛前，其中一个黑袍人才缓缓抬起头。

那是一张苍老而干枯的脸，眼窝深陷，里面没有眼球，只有两团跳动的火焰。

“你迟到了，容器。”老人的声音干涩而冰冷，像是两块石头在摩擦。

无明冷笑：“我不是容器，我是送葬者。”

老人嘴角勾起一抹嘲讽的弧度：“在这里，所有人都是容器。区别只在于，你是自愿的，还是被迫的。”

“我选择第三种。”无明握紧短刃，身上的杀气暴涨，“毁灭。”

话音未落，他已冲向祭坛。黑袍人们纷纷起身，手中浮现出各种诡异的法器。光芒交错，杀机四伏。

无明的身影在光影中穿梭，短刃挥舞间，带起一片片血雨。他不再保留，每一击都直奔要害。体内的微粒因为剧烈的运动而沸腾，疼痛几乎让他失去理智，但他享受着这种疼痛。这是他还活着的证明。

他冲上祭坛，一刀劈向那颗红色晶体。

“咔嚓。”

晶体表面出现了一道裂纹。金色的液体从中渗出，滴落在白骨上，发出滋滋的声响。

黑袍老人们发出了愤怒的咆哮，他们联手发动攻击，一道巨大的能量波向无明袭来。无明无处可躲，只能硬抗。他用短刃挡在身前，符文爆发出耀眼的光芒，勉强抵消了部分冲击。

巨大的冲击力将他震飞出去，重重地撞在墙壁上。无明吐出一口鲜血，感觉五脏六腑都移位了。但他没有倒下，他撑着短刃，摇摇晃晃地站起来。

他的眼中闪烁着疯狂的光芒。他看着那颗破裂的晶体，看着那些惊慌失措的黑袍人，心中涌起一股前所未有的快感。

他要毁了这里。毁了这一切。

他再次冲向祭坛，这一次，他不顾一切。短刃深深插入晶体之中，用力一搅。

“轰！”

爆炸发生了。红色的光芒照亮了整个大厅，金色的微粒如暴雨般倾泻而下。无明被气浪掀翻，但他死死抓住短刃，不肯松手。

在混乱中，他感觉到体内的吸力消失了。取而代之的，是一股暖流。那些逸散的微粒并没有伤害他，反而被他手中的短刃吸收，然后转化为一股纯净的能量，涌入他的体内。

他的伤口以肉眼可见的速度愈合，体力迅速恢复。更重要的是，他的记忆碎片开始松动。一些模糊的画面闪过脑海：一个穿着白衣的女人，一把染血的剑，还有一个承诺。

“活下去……为了……”

画面戛然而止。

无明跪在地上，大口喘息着。周围的黑袍人已经全部倒下，他们的身体化为灰烬，融入了这片废墟。神庙开始崩塌，石块不断掉落。

他站起身，捡起短刃。刀身上的符文变得更加明亮，仿佛在诉说着什么。他看了一眼自己的左手，那里的鼓包已经消失，皮肤恢复了正常。

他赢了。至少在这一刻，他赢了。

但他知道，这只是冰山一角。那个集团不会善罢甘休，天堑之外的世界依然充满危险。而他，这个失忆的刺客，才刚刚踏上这条不归路。

无明走出神庙，外面的灰雾依旧浓重。他抬头望向远方，那里隐约可见一座城市的轮廓。那是下一个目标，也是下一个战场。

他握紧短刃，身影逐渐融入阴影之中。
