# 第7章 · 骨血里的窃贼

痛觉并非如潮水般涌来，而是像无数根烧红的铁针，从骨髓深处向外穿刺。无明睁开眼时，视野是一片浑浊的灰白，仿佛眼球表面蒙了一层洗不净的尸油。他试图抬起手臂，却发现肢体沉重得如同灌满了铅汞，每一次肌肉的收缩都伴随着骨骼摩擦发出的令人牙酸的咯吱声。

这里不是天堑边缘那充满硫磺味的荒原，而是一处封闭、潮湿且散发着腐烂甜腥气的地下空间。空气中弥漫着一种粘稠的雾气，那不是水汽，而是高浓度的微粒尘埃。它们悬浮在半空，闪烁着微弱却致命的幽蓝光芒，像是无数只窥视的眼睛。无明本能地想要屏住呼吸，但肺部早已习惯了这种剧毒的空气，贪婪地将那些发光的尘埃吸入体内。随着每一次吸气，胸腔内便传来一阵剧烈的灼烧感，仿佛吞下了滚烫的铁砂。

他低下头，看向自己的双手。皮肤苍白如纸，皮下隐约可见青黑色的血管在搏动，但那搏动的节奏并不属于心脏，而是一种诡异的、多频段的震颤。在他的左手掌心，一道暗红色的裂痕正缓缓张开，没有血液流出，反而从中渗出丝丝缕缕的黑雾。黑雾在空中凝聚，并未消散，而是缠绕上他腰间那柄刻有古老符文的短刃。

短刃上的符文此刻正发出低沉的嗡鸣，那是金属与某种不可名状之物共振的声音。无明感到一阵强烈的恶心，脑海中闪过一些破碎的画面：冰冷的石室、戴着面具的祭司、以及一把插入自己胸口的匕首。记忆依旧是一片空白，但身体的反应比思维更快——他的右手猛地握紧短刃，指节因用力过度而泛白。

“醒了？”

一个沙哑的声音从阴影中传来。无明瞬间绷紧全身肌肉，影迹潜行的本能让他几乎要融入周围的黑暗，但体内的虚弱感强行将他拽回现实。他转动僵硬的脖颈，看向声音的来源。

那是一个蜷缩在角落里的男人，衣衫褴褛，身上布满了溃烂的伤口。男人的双眼已经失明，眼眶空洞洞地对着无明的方向，嘴角却挂着一丝诡异的笑容。“你闻起来……很新鲜。”男人舔了舔干裂的嘴唇，“你是‘容器’，还是‘祭品’？”

无明没有回答。他的目光死死盯着男人胸口起伏的节奏，那里有一团不规则的光点在跳动。那是微粒聚集的迹象。在这个被规则支配的世界里，微粒是通货，是力量，也是诅咒。总量硬上限八亿四千万，每一粒都被严格监控，每一粒的流动都伴随着血腥的交易或掠夺。而这个男人身上的微粒波动，虽然微弱，却异常纯净。

“我不记得我是谁。”无明开口，声音嘶哑得像是在砂纸上磨过，“但我记得怎么杀人。”

男人愣了一下，随即发出一阵刺耳的笑声：“杀人？在这里，杀人是最廉价的行为。真正昂贵的，是活着。尤其是像你这样，体内被植入了‘种子’的人。”

“种子？”无明皱眉，左手掌心的裂痕再次扩大，黑雾愈发浓重。

“他们叫你‘无明’，对吧？”男人向前爬行了几步，枯瘦的手指在地面上抓出深深的痕迹，“这是第七区的惯例。所有被选中植入未知微粒的实验体，都会失去名字，失去记忆，只剩下代号。你是这一批里唯一活下来的。其他的，要么疯了，要么变成了只会吞噬微粒的怪物。”

无明感到一阵寒意顺着脊椎爬升。他低头看向自己的腹部，那里的皮肤下似乎有什么东西在蠕动。那种感觉既熟悉又陌生，就像是有另一个生命体寄生在他的体内，正在汲取他的生命力，同时也赋予他某种超越常人的感知能力。他能听到墙壁后老鼠的心跳，能闻到远处下水道里腐败食物的气味，甚至能感觉到空气中微粒流动的轨迹。

“他们在找你。”男人突然压低声音，脸上的笑容消失殆尽，“掠夺集团的人。他们不会允许一个拥有‘活性种子’的实验体逃脱。你的存在本身，就是对现有微粒分配规则的挑衅。”

话音未落，头顶上方的石板突然传来一声闷响。灰尘簌簌落下，紧接着，三道黑影从天花板的破洞中跃下，稳稳地落在地面上。他们身穿黑色的紧身作战服，脸上戴着没有任何表情的白色面具，手中握着带有倒刺的金属长鞭。

“目标确认。编号7-09，状态：觉醒初期。”其中一人冷冷地说道，声音经过电子处理，显得毫无感情，“执行回收程序。若反抗，就地格杀，提取核心微粒。”

无明没有任何犹豫。在对方话音落下的瞬间，他的身体已经动了。影迹潜行发动，他的身影在昏暗的光线中变得模糊不清，仿佛化作了一缕烟雾。然而，体内的虚弱感让他的动作出现了一丝迟滞。就在这一刹那，一条金属长鞭如毒蛇般袭来，狠狠抽在他的左肩上。

剧痛袭来，皮开肉绽。鲜血飞溅而出，但在接触到空气的瞬间，那些血液竟然迅速凝固，变成了黑色的晶体。无明闷哼一声，借势翻滚，躲开了紧随其后的第二击。他手中的短刃挥出，划出一道寒光，切断了第三人的长鞭。

短刃上的符文在这一刻爆发出耀眼的光芒。无明感到一股热流从刀柄传入掌心，顺着手臂蔓延至全身。那股热流与他体内蠕动的“种子”产生了共鸣。原本混乱的微粒流动瞬间变得有序，空气中的蓝色尘埃仿佛受到了召唤，疯狂地向他汇聚而来。

“他在吸收微粒！”一名掠夺者惊呼，“阻止他！不能让他完成初次融合！”

三人同时扑了上来。无明咬紧牙关，强忍着肩部的剧痛，将短刃横在胸前。他不再逃避，而是迎头撞向最近的敌人。短刃刺入对方的胸膛，并没有遇到太多的阻力，仿佛切入了一块腐朽的木头。掠夺者的身体剧烈颤抖，眼中的光芒迅速黯淡，而他体内的微粒则通过短刃，源源不断地涌入无明的体内。

这是一种难以言喻的快感，也是一种极致的痛苦。微粒进入身体的瞬间，无明感到自己的每一个细胞都在尖叫，它们在重组，在进化，也在崩溃。他的视野开始扭曲，周围的世界变成了一片由线条和光点构成的抽象画卷。他看到了掠夺者体内微粒的流动路径，看到了他们弱点所在，甚至看到了他们灵魂深处对死亡的恐惧。

另外两名掠夺者见状，立刻改变了战术。他们不再近身攻击，而是从腰间取出两个黑色的球体，扔向无明。球体在空中爆炸，释放出大量的黑色烟雾。这些烟雾具有极强的腐蚀性，接触到的地面瞬间冒出白烟。

无明屏住呼吸，向后跃去。但他的速度终究慢了一步，左腿被烟雾擦过，裤管瞬间化为灰烬，皮肤上也出现了大片的灼伤。他单膝跪地，大口喘息着。体内的“种子”似乎因为吸收了足够的微粒而变得活跃起来，它在无明的丹田处旋转，形成一个微小的漩涡。这个漩涡不仅吞噬着外界的微粒，也在吞噬无明自身的生命力。

“咳……”无明吐出一口黑血，眼神却变得更加冰冷。他意识到，如果不尽快结束战斗，他会被这股力量反噬而死。

他站起身，拖着受伤的腿，一步步走向剩下的两名掠夺者。他的步伐缓慢而坚定，每一步都在地面上留下一个黑色的脚印。掠夺者们惊恐地发现，无明身上的伤口正在以肉眼可见的速度愈合，而那些黑色的晶体则重新融入了他的皮肤，使他的肌肤呈现出一种金属般的光泽。

“怪物……”一名掠夺者颤抖着后退，“他是怪物！”

“闭嘴！”另一名掠夺者怒吼道，举起手中的武器准备射击。

但无明已经来到了他的面前。短刃划过一道优美的弧线，精准地切断了对方的喉咙。鲜血喷涌而出，却被无明掌心的黑雾尽数吸收。最后一人见势不妙，转身欲逃。无明冷笑一声，身形一闪，出现在他的背后。短刃从后心刺入，从前胸透出。

战斗结束了。

无明站在原地，感受着体内澎湃的力量。三具尸体躺在地上，逐渐变得干枯，仿佛所有的精华都被抽空。他低头看向自己的手掌，那道裂痕已经完全闭合，取而代之的是一个淡淡的黑色印记，形状如同一只闭合的眼睛。

“这就是……微粒的力量吗？”无明喃喃自语。

角落里的那个盲眼男人发出了沉重的叹息：“你做得很好，但也更糟了。你开启了‘内宇宙’的第一阶。从今往后，你将不再是人类，而是微粒的奴隶，或者是它的主人。这取决于你能否控制住心中的凶性。”

无明转过身，看向男人：“你知道怎么控制它？”

男人摇了摇头：“没有人知道。每个觉醒者走的路都不一样。有些人成为了暴君，统治一方；有些人成为了疯子，自我毁灭。而你……”男人顿了顿，“你的短刃上有古老的符文，那是前文明留下的遗产。或许，它能指引你找到答案。”

无明握紧短刃，符文的光芒渐渐熄灭。他感到一阵深深的疲惫，但精神却前所未有的清醒。他走到男人面前，蹲下身：“告诉我，第七区的规则是什么？”

男人苦笑了一声：“规则？在这里，唯一的规则就是弱肉强食。但有一条禁令你必须记住：不要在满月之夜离开庇护所。否则，你会听到‘天堑’的呼唤，那是比死亡更可怕的命运。”

“天堑……”无明重复着这个词，脑海中闪过一些模糊的画面：巨大的裂缝、无尽的深渊、以及深渊底部那双注视着他的眼睛。

他站起身，环顾四周。这个地下空间虽然隐蔽，但并不安全。掠夺集团的援军随时可能赶到。他必须离开这里，去寻找更多的线索，去弄清楚自己究竟是谁，以及那个所谓的“种子”到底是什么。

他走到出口处，推开沉重的石门。外面的世界是一片废墟，断壁残垣间生长着扭曲的植物，天空中悬挂着一轮血红色的月亮。微风拂过，带来远处城市的喧嚣声和隐隐约约的哭嚎声。

无明深吸一口气，迈步走入夜色之中。他的身影很快消失在黑暗中，只留下一串黑色的脚印，延伸向未知的远方。

在他身后，盲眼男人静静地坐在角落里，听着脚步声远去，嘴角勾起一抹意味深长的笑容。他从怀中掏出一块破碎的镜片，镜片中映出的不是他自己的脸，而是无明离去的背影。而在无明的背后，隐约可见一个巨大的、由微粒构成的虚影，正缓缓张开翅膀，笼罩了整个天空。
