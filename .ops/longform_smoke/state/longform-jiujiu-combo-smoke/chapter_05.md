# 第5章 · 骨缝里的尘埃

痛觉并非如潮水般涌来，而是像生锈的铁钉，一颗一颗被强行楔入颅骨的缝隙。

无明睁开眼时，视野是一片浑浊的灰白。没有天空，没有大地，只有悬浮在空气中的、细密如雾的灰色颗粒。它们静止不动，却又仿佛在某种不可见的律动中微微震颤，像是无数双窥视的眼睛。他试图呼吸，肺部却传来撕裂般的灼烧感，每一次吸气都像是在吞咽碎玻璃。喉咙里堵着一团粘稠的腥甜，那是血，也是某种更古老、更冰冷的东西。

他躺在一片由黑曜石碎片铺就的荒原上。四周寂静得令人发狂，连风声都被这厚重的死寂吞噬。他的身体僵硬如尸，四肢百骸充斥着一种陌生的沉重感，仿佛每一块肌肉、每一根骨骼都被灌入了铅汞。他试图抬起右手，指尖触碰到的不是泥土，而是一柄冰冷刺骨的金属。

那是一柄短刃。

刃身狭长，呈现出一种近乎黑色的暗哑光泽，上面刻满了扭曲繁复的符文。那些符文并不发光，却在无明的注视下，隐隐流动着暗红色的微光，如同皮下潜伏的血管。当他的手指触碰到刀柄的那一刻，一股寒意顺着掌心直窜脊椎，紧接着，脑海中炸开了一瞬尖锐的耳鸣。

*“……饥饿。”*

一个声音在他脑海深处响起，非男非女，非老非少，带着一种原始的、贪婪的渴望。

无明猛地缩回手，心脏剧烈跳动，几乎要撞破胸腔。他警惕地环顾四周，除了漫无边际的灰雾和脚下的黑曜石，再无他物。他是谁？这里是何处？为何手中握着凶器？这些问题像是一团团乱麻，缠绕在他的意识深处，每当他试图理清，就会引发一阵剧烈的头痛。

他低下头，看向自己的胸口。衣衫褴褛，布料早已腐烂成絮状，裸露出的皮肤苍白如纸，布满了青紫色的淤痕和细小的伤口。而在左胸靠近心脏的位置，有一处明显的凸起，皮肤呈现出诡异的半透明状，隐约可见内部有什么东西在缓缓蠕动。

那不是伤口，也不是肿瘤。

无明颤抖着伸出左手，指尖轻轻按压在那处凸起上。触感坚硬，冰冷，且带有一种令人作呕的滑腻感。随着他的按压，那东西似乎受到了刺激，猛地收缩了一下，随即释放出一股细微的热流，瞬间扩散至全身。

剧痛再次袭来，但这次伴随着一种奇异的快感。他感觉到体内的血液开始加速流动，原本枯竭的力量正在一点点回归。与此同时，那些悬浮在空气中的灰色微粒，竟然开始向他汇聚。它们像是有生命一般，穿过他的毛孔，渗入他的皮肤，最终汇入左胸的那处凸起之中。

*“进食……”*

脑海中的声音再次响起，比之前更加清晰，也更加急切。

无明咬紧牙关，强忍着身体的不适，挣扎着坐起身来。每做一个动作，关节都会发出咔咔的声响，仿佛生锈的机械在强行运转。他握紧手中的短刃，刀刃上的符文光芒大盛，暗红色的流光在刃身上游走，仿佛在欢呼，在庆祝主人的苏醒。

他站起身，双腿有些踉跄，但很快便稳住了身形。多年的本能告诉他，在这种环境下，软弱意味着死亡。他必须动起来，必须找到答案，或者至少，找到活下去的方法。

前方的灰雾中，隐约可见一座巨大的轮廓。那是一座城，或者说，曾经是一座城。如今，它只剩下一具庞大的骨架，高耸的塔楼断裂倾斜，城墙崩塌大半，露出内部错综复杂的结构。整座城池笼罩在一层淡淡的血色光晕中，显得既神圣又邪恶。

无明眯起眼睛，瞳孔微微收缩。在他的视野中，那座废墟之城并非静止不变。他能看到空气中流动的微粒轨迹，它们从城市的各个角落升起，汇聚向中心的一座高塔。那里，似乎有什么东西在吸引并吞噬着这些微粒。

微粒。

这个词突然跳入他的脑海，带来一阵短暂的清明。他记得，这是一种通货，一种能量，也是一种诅咒。在这个世界，微粒是唯一的硬通货，总量恒定，不可再生。每个人都在争夺微粒，为了生存，为了权力，为了繁衍。

而他，似乎刚刚完成了一次微小的掠夺。

无明低头看向自己的左手，掌心中残留着几粒灰色的尘埃。它们正在迅速消散，融入他的皮肤。他能感觉到，自己的体力恢复了一丝，虽然微不足道，但这确凿无疑地证明了他的猜测：他的身体，或者说是他体内的那个异物，能够吸收微粒。

这是一种禁忌的能力。

在规则怪谈支配的世界里，任何违背既定秩序的行为都会招致惩罚。私自掠夺微粒，尤其是以这种直接吞噬的方式，无异于自寻死路。但他别无选择。体内的异物正在不断索取，如果不满足它的饥饿，他可能会先一步崩溃。

他深吸一口气，压下内心的不安，迈步向那座废墟之城走去。脚下的黑曜石碎片发出清脆的碎裂声，在这死寂的世界中显得格外刺耳。每一步落下，他都感到体内的异物在轻微颤动，仿佛在催促他加快脚步。

随着距离的拉近，空气中的微粒浓度越来越高。无明能清晰地看到，那些灰色的颗粒在空中交织成网，形成了一道道肉眼可见的屏障。这是天堑的边缘，是生与死的分界线。跨过这道线，便是真正的地狱。

但他没有退缩。相反，他的眼中闪过一丝凶狠的光芒。既然已经身处地狱，那便无需再畏惧魔鬼。

当他踏入废墟之城的范围时，周围的温度骤然下降。寒风呼啸而过，卷起地上的尘土，打在脸上生疼。街道两旁，残破的建筑如同巨兽的骸骨，张开着黑洞洞的大口，等待着猎物的进入。墙壁上，依稀可见一些斑驳的血迹和抓痕，诉说着曾经发生过的惨烈厮杀。

无明紧握短刃，目光扫视着四周。他的感官变得异常敏锐，能够听到远处传来的细微声响——那是脚步声，不止一人。

他立刻停下脚步，身体紧绷，进入了“影迹潜行”的状态。这是一种刻在骨子里的本能，不需要思考，身体自动做出了反应。他的气息收敛到了极致，身影仿佛融入了周围的阴影之中，变得模糊不清。

片刻后，三个身影从街角的阴影中走了出来。他们身穿破烂的皮甲，手持粗糙的铁棍和砍刀，脸上戴着狰狞的面具，看不清面容。他们的动作迟缓而僵硬，眼神空洞，仿佛失去了灵魂的行尸走肉。

但在无明的眼中，他们身上散发着浓郁的微粒波动。那是长期接触高浓度微粒环境所留下的痕迹，也是他们力量的来源。

其中一人似乎察觉到了什么，停下脚步，歪着头四处张望。他的面具下传出低沉的嘶吼声，像是在警告同伴。

无明屏住呼吸，心跳缓慢而有力。他知道，一旦暴露，必将面临围攻。这三个人的实力虽然不强，但在这种环境下，任何意外都可能导致致命的后果。他必须速战速决。

他观察着三人的站位，寻找着最佳的切入点。左侧那人最为松懈，右侧那人警惕性最高，中间那人则处于两者之间。这是一个典型的三角阵型，但也存在着致命的弱点。

就在左侧那人转身的一瞬间，无明动了。

他没有发出任何声音，如同一道黑色的闪电，瞬间掠过了数米的距离。短刃划破空气，带起一道凄厉的风声，精准地刺入了左侧那人的咽喉。

鲜血喷涌而出，溅在无明的脸上，温热而粘稠。那人甚至来不及发出一声惨叫，身体便软软地倒了下去。

另外两人反应过来，怒吼着挥动手中的武器扑了上来。铁棍带着风声砸向无明的头顶，砍刀则横扫他的腰部。

无明侧身避开铁棍，同时手腕翻转，短刃顺势划过持刀者的手腕。刀刃切入血肉的声音沉闷而悦耳，那只握着砍刀的手齐腕而断，掉落在地上，手指还在微微抽搐。

持棍者见状，眼中闪过一丝惊恐，但随即又被疯狂所取代。他高举铁棍，再次砸下。这一次，无明没有躲避，而是迎了上去。他用左臂格挡铁棍，巨大的冲击力让他手臂一阵麻木，但他借着这股力量，身体旋转，短刃如毒蛇般探出，刺入了对方的腹部。

三人倒地，鲜血染红了地面。

无明喘着粗气，拔出短刃。刀刃上的符文光芒黯淡了一些，但很快又重新亮起。他看着地上的尸体，心中没有丝毫波澜，只有一种冰冷的计算。

他蹲下身，伸手按在其中一人的胸口。果然，那里的皮肤下也有类似的凸起，只是规模较小，颜色也更为暗淡。他用力一捏，那凸起破裂开来，流出了一滩黑色的液体，以及几颗晶莹剔透的微粒。

这些微粒散发着柔和的光芒，与空气中的灰色微粒截然不同。它们是纯净的，是高浓度的精华。

无明毫不犹豫地抓起这些微粒，塞入口中。

一股暖流瞬间涌遍全身，驱散了之前的寒冷和疲惫。体内的异物发出了满足的叹息声，那种饥饿感暂时得到了缓解。

他站起身，擦去脸上的血迹，目光投向城市中心的那座高塔。刚才的战斗消耗了他不少体力，但也让他确认了一件事：这座城里，还有更多的猎物，更多的微粒。

而他，才刚刚开始狩猎。

风更大了，卷起地上的血腥味，弥漫在整个废墟之中。无明握紧短刃，迈过尸体，向着黑暗深处走去。他的背影孤独而决绝，仿佛一把即将出鞘的利刃，注定要切开这绝望的天幕。
