# 第2章 · 血肉契约与锈蚀之息

教堂沉重的橡木门在身后合拢，发出沉闷的撞击声，将最后一点烛火的余温彻底隔绝。无明没有回头，他的脊背紧绷如拉满的弓弦，每一步踏在碎石铺就的小径上，都像是在试探深渊的边缘。空气变得粘稠，带着一种陈腐的铁锈味和某种难以名状的甜腥，那是血液干涸后混合了霉菌的气息。四周的雾气不再是单纯的白色，而是呈现出一种病态的灰黄，仿佛无数细小的尘埃在光线中扭曲、蠕动。他握紧手中的短刃，指节因用力而泛白，刀刃上的古老符文在昏暗的光线下隐约闪烁，像是沉睡野兽睁开的一只眼。这种光芒并不温暖，反而透着一股刺骨的寒意，顺着掌心蔓延至手臂，甚至深入骨髓。无明感到一阵轻微的眩晕，脑海中那些破碎的记忆碎片再次翻涌，却依旧无法拼凑出完整的画面。他只记得自己是一个刺客，一个被遗忘的影子，而现在，这片土地正试图吞噬他。脚下的路面开始变得松软，像是踩在腐烂的肉块上，每一步都会陷下去几分，发出令人牙酸的挤压声。远处的天空被一道巨大的裂痕撕裂，那便是传说中的“天堑”，它横亘在天际，如同神明挥刀留下的伤疤，从中渗出黑色的雾气，遮蔽了星辰。在这片被诅咒的土地上，生命不再是恩赐，而是一种需要不断偿还债务的诅咒。

前方的迷雾逐渐散去，露出一座废弃的石桥，桥身布满了青苔和暗红色的污渍，仿佛曾经流淌过大量的鲜血。桥头立着一块石碑，上面刻着扭曲的文字，字迹已经被风雨侵蚀得模糊不清，但依稀能辨认出几个关键词：“禁止独行”、“血脉相连”、“繁衍即罪”。无明停下脚步，目光扫过这些文字，心中升起一股莫名的烦躁。这些规则看似荒诞，却在这末日般的世界中有着绝对的权威。违反规则的人，往往会在瞬间化为灰烬，或者变成某种不可名状的怪物。他深吸一口气，强行压下内心的不安，继续向前走去。就在他的脚刚刚踏上石桥的瞬间，周围的空气骤然凝固，原本流动的雾气停止了飘动，仿佛时间在这一刻静止。紧接着，一阵细微的沙沙声从桥下的阴影中传来，像是无数昆虫在啃食骨头。无明猛地转身，短刃横在胸前，眼神冷冽如冰。只见三个身影从迷雾中缓缓走出，他们身穿破烂的黑袍，脸上戴着惨白的面具，手中握着生锈的铁钩和骨刀。他们的动作僵硬而诡异，关节处发出咔咔的声响，仿佛是由死人的肢体拼接而成。为首的一人抬起头，面具后的双眼闪烁着红光，声音沙哑而刺耳：“外来者，你身上有‘纯净’的味道。”

无明没有回答，他的身体已经进入了战斗状态，肌肉紧绷，呼吸放缓，整个人如同一张拉满的弓，随时准备射出致命的一箭。他知道，这些人并非普通的强盗，而是“微粒掠夺集团”的外围成员，专门猎杀落单的旅人，抽取他们体内的微粒以维持自身的存在。在这个资源匮乏的世界里，微粒是唯一的通货，也是生命的源泉。总量硬上限八亿四千万的限制，使得每一粒微粒都显得弥足珍贵，也引发了无尽的杀戮与争夺。无明感到体内的微粒残留开始躁动，那种被植入的未知力量在他的血管中奔流，带来一阵灼热的痛感。他咬紧牙关，强忍着不适，目光死死锁定着对面的三人。为首的掠夺者发出一声怪笑，挥舞着手中的铁钩冲了上来，另外两人则从两侧包抄，形成夹击之势。无明身形一闪，施展“影迹潜行”，整个人如同融入黑暗一般，消失在原地。下一秒，他出现在左侧敌人的身后，短刃划过一道寒光，精准地切断了对方喉咙处的动脉。鲜血喷涌而出，溅在无明的脸上，温热而粘稠。他没有丝毫停顿，顺势一脚踢在另一名敌人的膝盖上，伴随着骨骼碎裂的声音，那人惨叫倒地。然而，为首的掠夺者并未退缩，反而变得更加疯狂，他身上的黑袍鼓胀起来，仿佛有什么东西在里面蠕动。

无明警惕地后退半步，短刃上的符文突然亮起，散发出幽蓝色的光芒。他感到一股强大的吸力从刀刃上传来，仿佛在渴望吞噬更多的生命能量。与此同时，他体内的未知微粒也开始剧烈波动，与短刃产生了一种奇异的共鸣。这种共鸣让他感到既恐惧又兴奋，仿佛某种被封印的力量正在苏醒。掠夺者发出一声怒吼，身上的黑袍炸裂开来，露出下面布满黑色纹路的皮肤。那些纹路如同活物一般，在他身上游走、扩张，最终汇聚成一张狰狞的面孔。无明瞳孔微缩，他意识到，这不仅仅是简单的肉体变异，而是某种规则层面的扭曲。在这个被规则支配的世界里，违背禁忌的人往往会付出惨痛的代价，但也可能获得超越常人的力量。他不再犹豫，身形再次消失，这一次，他直接出现在了掠夺者的面前，短刃直刺其心脏。然而，就在刀刃即将触及皮肤的瞬间，掠夺者胸口的黑色纹路突然张开，形成了一张巨大的嘴，一口咬住了短刃。无明感到一股巨大的力量从刀柄传来，几乎要将他的手臂扯断。他闷哼一声，双脚蹬地，强行稳住身形，同时调动体内的微粒，注入短刃之中。

短刃上的符文光芒大盛，幽蓝色的火焰顺着刀刃蔓延，瞬间点燃了掠夺者身上的黑色纹路。掠夺者发出凄厉的惨叫，身体开始迅速枯萎，仿佛被抽干了所有的生命力。那些黑色的纹路在高温下扭曲、断裂，最终化为灰烬。无明趁机抽出短刃，顺势一挥，斩下了掠夺者的头颅。头颅滚落在地，面具碎裂，露出一张干枯如树皮的脸庞，眼中充满了绝望与不甘。另外两名倒地的敌人见状，吓得魂飞魄散，挣扎着想要逃跑。无明冷冷地看着他们，没有丝毫怜悯。在这个弱肉强食的世界里，仁慈只会带来死亡。他快步上前，两刀结果了他们的性命。随着三具尸体的倒下，周围的空间微微震动，一些微小的光点从尸体中飘出，缓缓飞向无明。这些光点便是微粒，虽然数量极少，但对于现在的他来说，却是至关重要的补给。他伸出手，任由那些光点融入体内，感到一股暖流在四肢百骸中流转，之前的疲惫和伤痛得到了些许缓解。然而，这种缓解只是暂时的，他清楚地知道，刚才的战斗已经暴露了他的位置，更多的掠夺者很快就会赶来。

无明迅速检查了一下自己的身体，确认没有受到严重的伤害后，他开始搜刮尸体上的物品。除了几枚劣质的微粒结晶外，他还找到了一本破旧的笔记。笔记的纸张已经发黄变脆，上面的字迹潦草而混乱，记录着一些关于“内宇宙”和“血脉契约”的片段信息。无明快速浏览着笔记，眉头越皱越紧。根据笔记记载，每个人的体内都拥有一个微型的“内宇宙”，分为四个阶段：萌芽、生长、成熟、衰败。只有通过不断的繁衍和血脉延续，才能激活内宇宙的潜能，从而获得更强大的力量。然而，这种繁衍并非自然的过程，而是需要通过特定的仪式和契约来完成。违反契约的人，将会受到规则的惩罚，轻则失去记忆，重则沦为行尸走肉。无明看着手中的笔记，心中涌起一股寒意。他想起自己在教堂醒来时的迷茫，以及脑海中那些破碎的记忆碎片，难道这一切都与这个所谓的“血脉契约”有关？他是否也曾参与过这样的仪式？又或者，他是一个失败的实验品？这些问题像毒蛇一样缠绕在他的心头，让他感到窒息。

他将笔记收入怀中，站起身来，望向远方的天堑。那道巨大的裂痕依然悬挂在天际，黑色的雾气从中源源不断地涌出，笼罩着整个世界。无明知道，自己必须尽快离开这里，找到一个安全的地方，理清思绪，并寻找恢复记忆的方法。他收起短刃，转身走向石桥的另一端。然而，就在他即将离开石桥的时候，脚下突然传来一阵剧烈的震动。石桥开始崩塌，石块纷纷落入下方的深渊，发出轰隆隆的巨响。无明心中一惊，连忙加速奔跑，试图在石桥完全坍塌之前到达对岸。然而，就在他距离对岸还有几步之遥时，一只巨大的黑色触手从深渊中伸出，狠狠地拍向他的后背。无明感到一股巨力袭来，整个人被抛向空中，重重地摔在对岸的地面上。他吐出一口鲜血，感到五脏六腑都在移位。那只黑色触手并没有追击，而是缓缓缩回深渊，仿佛只是在警告他不要靠近。无明挣扎着爬起来，擦去嘴角的血迹，眼神变得更加冰冷。他意识到，这个世界比他想象的还要危险，不仅要面对人类的掠夺，还要应对来自天堑本身的威胁。

他环顾四周，发现这里是一片荒芜的废墟，残垣断壁间长满了黑色的荆棘，空气中弥漫着浓烈的硫磺味。远处隐约可见一座城市的轮廓，城墙高耸入云，上面闪烁着诡异的红光。那便是下一座城，也是他接下来的目的地。无明深吸一口气，调整好自己的状态，迈步向城市走去。他知道，前方等待着他的，将是更加残酷的规则和更加激烈的争斗。但他别无选择，只有不断前行，才能在这末日的世界中活下去。每走一步，他都能感觉到体内的微粒在缓慢增长，那是刚才战斗的战利品，也是他生存的资本。短刃静静地挂在他的腰间，符文的光芒已经黯淡下去，仿佛重新陷入了沉睡。但无明知道，它随时都可能再次苏醒，成为他最锋利的武器，也可能成为他最大的诅咒。他摸了摸怀中的笔记，心中暗暗发誓，一定要解开这个世界的真相，找回属于自己的记忆。无论付出多大的代价，他都不会放弃。因为他是刺客，是影子，是注定要在黑暗中行走的人。

夜幕降临，天空中的黑色雾气变得更加浓郁，几乎遮蔽了所有的光线。无明在一处断墙下找到了一个相对隐蔽的角落，坐下来休息。他拿出那本笔记，借着微弱的光线仔细阅读。笔记的后半部分记录了一些关于“多子多福”的禁忌知识，提到了一种古老的仪式，可以通过牺牲后代的生命力来强化自身的内宇宙。这种仪式被视为极端的邪恶，但在某些绝望的时刻，却成为了许多人唯一的选择。无明看着这些文字，心中涌起一股恶心感。他无法想象，究竟是什么样的人，才会做出如此残忍的事情。然而，他也明白，在这个资源极度匮乏的世界里，道德和伦理早已崩塌，生存才是唯一的法则。他合上笔记，闭上眼睛，试图让自己平静下来。然而，脑海中的记忆碎片再次浮现，这一次，他看到了一张女人的脸，那张脸苍白而美丽，眼中充满了悲伤和爱意。女人张开嘴，似乎在说着什么，但声音却被风声掩盖。无明猛地睁开眼睛，大口喘着粗气，额头上渗出了细密的汗珠。那个女人是谁？她和自己有什么关系？为什么她的出现会让自己的心如此疼痛？

他站起身来，走到断墙边，望着远方那座闪烁着红光的城市。城市的轮廓在夜色中显得格外狰狞，仿佛一头潜伏在黑暗中的巨兽，等待着猎物的自投罗网。无明握紧了拳头，指甲深深嵌入掌心，疼痛让他保持清醒。他知道，自己不能再逃避了，必须直面这一切。无论是记忆的缺失，还是身体的异变，亦或是这个世界的残酷规则，他都要一一破解。他拔出短刃，轻轻抚摸着上面的符文，感受着那股熟悉的寒意。短刃似乎回应了他的心意，微微颤动了一下，散发出一丝微弱的光芒。无明嘴角勾起一抹冷笑，眼中闪过一丝狠厉。既然这个世界想要玩弄他，那他就要反过来玩弄这个世界。他要利用手中的短刃，利用体内的微粒，利用一切可以利用的资源，在这末日的天堑边缘，杀出一条血路。他要让那些隐藏在黑暗中的敌人知道，刺客无明，绝非任人宰割的羔羊。夜风呼啸，卷起地上的尘土，发出呜呜的声响，仿佛在为即将到来的风暴奏响序曲。无明整理好装备，深吸一口气，迈开步伐，向着那座充满未知的城市走去。他的身影在夜色中逐渐拉长，最终消失在黑暗的尽头，只留下地面上那一串深深的脚印，证明着他曾经来过。
