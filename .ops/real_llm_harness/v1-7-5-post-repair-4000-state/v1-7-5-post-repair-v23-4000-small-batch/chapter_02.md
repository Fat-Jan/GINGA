# 第2章 · 锈蚀的脐带与内宇宙的胎动

痛觉并未因意识的回归而消退，反而像是一根烧红的铁钎，顺着脊椎的缝隙狠狠楔入，将无明从混沌的深渊中强行拽回现实。他并没有立刻起身，而是保持着蜷缩的姿态，手指深深扣进黑曜石碎片的缝隙里。指尖传来的触感冰冷、粗糙，带着一种令人作呕的滑腻感——那是某种生物体液干涸后留下的结晶，混合着尘埃，构成了这片荒原的地表。

灰白的雾气依旧悬浮在半空，静止得如同凝固的尸块。无明缓缓抬起头，脖颈处的关节发出细微的咔哒声，像是生锈的齿轮在强行咬合。他的视线穿过稀薄的空气，落在自己紧握的右手上。那柄短刃依然在手，刀身黯淡无光，唯有那些古老的符文在昏暗的光线下隐约流淌着暗红色的微光，仿佛血管中搏动的血液。

但他感觉到的不是武器的重量，而是来自体内的异样。

那种感觉并非疼痛，而是一种充盈。就像是有无数细小的沙粒被强行灌入了他的骨髓，沿着神经末梢蔓延至每一寸肌肉纤维。它们在他体内游走，碰撞，发出只有他能听见的、细密如蚕食桑叶般的沙沙声。这不是外来的入侵者，更像是某种早已潜伏在他血肉深处的种子，在这一刻破土而出。

无明深吸一口气，肺部扩张时带来一阵剧烈的撕裂感。空气中的尘埃颗粒随着呼吸涌入鼻腔，刺激着脆弱的黏膜。他试图调动记忆，去追溯这种异样的来源，但脑海中只有一片空白，如同被大火焚烧过的废墟，连灰烬都未曾留下。他是谁？为何在此？这柄短刃又从何而来？这些问题像是一群饥饿的乌鸦，在他的颅骨内盘旋尖叫，却得不到任何回应。

他松开左手，掌心向上摊开。掌纹间嵌满了黑色的污垢，但在污垢之下，皮肤呈现出一种不自然的苍白，甚至能清晰地看到皮下青紫色的血管网络。而在那些血管的交汇处，似乎有什么东西在微微隆起，像是皮肤下埋藏着一颗颗微小的珍珠。

无明眯起眼睛，集中精神感知那股游走在体内的“沙粒”。随着注意力的聚焦，那些沙沙声逐渐变得清晰，进而转化为一种奇异的律动。它们不再杂乱无章，而是遵循着某种既定的轨迹，在他的体内构建出一个微型的循环系统。这个系统独立于他的心脏和肺叶之外，自成一体，贪婪地汲取着他身体原本的能量，却又反过来释放出一种冰冷的、充满压迫感的力量。

这就是“微粒”吗？

这个词突兀地跳入他的脑海，伴随着一阵尖锐的刺痛。无明猛地捂住额头，指节因用力而发白。他不知道这个词意味着什么，但它带来的恐惧却是真实的。这是一种对未知的本能排斥，也是对自身存在被篡改的愤怒。

他挣扎着站起身，双腿因为长时间的僵硬而有些颤抖。黑曜石碎片在他的脚下发出清脆的碎裂声，在这死寂的世界里显得格外刺耳。无明环顾四周，视野所及之处，除了无尽的灰白雾气和黑色的岩石，再无其他。这里没有风，没有声音，没有生命的气息。整个世界仿佛已经死去，只剩下他这一个还在呼吸的活物。

然而，就在这极致的寂静中，无明的耳朵捕捉到了一丝极其细微的动静。

那是脚步声。

沉重，缓慢，拖沓。每一步落下，都会在地面上激起一圈肉眼可见的尘埃涟漪。声音来自雾气的深处，距离他大约百米左右。无明的瞳孔瞬间收缩，身体本能地紧绷起来。手中的短刃微微调整角度，刀尖指向声音传来的方向。尽管失去了记忆，但他的肌肉记忆依然保留着战斗的本能。影迹潜行的技巧在他的脑海中自动浮现，虽然无法完全施展，但足以让他的气息变得更加隐蔽，身形更加低伏。

脚步声越来越近，伴随着一种沉重的喘息声。那喘息声浑浊而粗重，像是破旧的风箱在拉扯。无明屏住呼吸，将自己融入周围阴影的轮廓中。他的心跳加速，但节奏依然平稳，这是多年杀戮生涯留下的烙印。

终于，一个身影穿透了灰白的雾气，出现在他的视野中。

那是一个男人，或者说，曾经是一个男人。他的身体极度扭曲，四肢以怪异的角度弯曲着，皮肤呈现出一种病态的灰褐色，上面布满了溃烂的疮口和增生的肉瘤。他的双眼浑浊不堪，眼白部分已经被黑色的血丝覆盖，瞳孔扩散到了极致，几乎占据了整个眼球。最引人注目的是他的腹部，那里高高隆起，像是怀胎十月的孕妇，但透过半透明的皮肤，可以清晰地看到里面蠕动着的东西——那不是胎儿，而是一团团纠缠在一起的、散发着微弱荧光的黑色物质。

无明认出了那种荧光。和他体内游走的“沙粒”有着相同的质感，只是更加狂暴，更加混乱。

那个扭曲的男人停了下来，头颅机械地转动着，似乎在寻找着什么。他的鼻子抽动着，嗅探着空气中的味道。当他的目光扫过无明所在的位置时，停顿了一瞬。虽然没有直接对视，但无明感觉到了一股冰冷的视线锁定在自己身上。

“饿……”

男人发出一声嘶哑的低吼，声音像是砂纸摩擦过粗糙的石面。他迈开步子，朝着无明的方向走来。每走一步，他腹部的隆起就剧烈地颤动一下，里面的黑色物质仿佛在欢呼，在渴望。

无明没有后退。他知道，在这片死寂的荒原上，逃避没有任何意义。唯一的出路，就是杀出一条血路。他握紧短刃，刀刃上的古老符文似乎感应到了主人的杀意，光芒骤然亮起，暗红色的流光在刀身上蜿蜒流动，如同苏醒的毒蛇。

就在男人踏入攻击范围的瞬间，无明动了。

他没有选择正面硬撼，而是身形一闪，如同一道鬼魅般绕到了男人的侧后方。影迹潜行的效果虽然大打折扣，但在如此近距离的情况下，依然足以迷惑对方的感官。短刃划出一道凄厉的弧线，精准地切向男人颈部的动脉。

然而，预想中的鲜血喷溅并没有发生。

刀刃切入皮肤的瞬间，发出了一声沉闷的钝响，像是砍在了坚韧的老牛皮上。男人的皮肤下并没有流出鲜红的血液，而是涌出了一股粘稠的黑色液体。那股液体接触到空气后，迅速凝固成坚硬的晶体，将刀刃牢牢卡住。

无明心中一惊，立刻松手后退。但已经晚了。

男人猛地转过身，那双浑浊的眼睛死死盯着无明，嘴角咧开一个夸张的笑容，露出满口参差不齐的黑牙。他腹部的隆起突然破裂，一团黑色的物质从中喷射而出，直扑无明的面门。

无明侧身闪避，黑色物质擦着他的脸颊飞过，击中了他身后的黑曜石地面。瞬间，那块岩石被腐蚀出一个深坑，冒出阵阵刺鼻的黑烟。

“更多……需要更多……”男人喃喃自语，声音中充满了疯狂的渴望。他再次扑了上来，这次的速度比之前快了一倍，四肢并用，像一只巨大的蜘蛛。

无明捡起地上的短刃，刀刃上还残留着黑色的晶体。他感到体内的“沙粒”开始躁动起来，仿佛受到了某种召唤，疯狂地向手臂汇聚。一股前所未有的力量涌入他的右手，短刃上的符文光芒大盛，暗红色的流光几乎要溢出刀身。

这一次，他没有犹豫。

当男人扑到面前时，无明迎头而上，短刃直刺男人的胸口。这一次，刀刃毫无阻碍地穿透了那层坚韧的皮肤，深深没入其中。男人发出一声凄厉的惨叫，身体剧烈地抽搐着。无明手腕一抖，刀刃在男人体内搅动，将那团纠缠的黑色物质彻底绞碎。

黑色的液体从伤口中喷涌而出，溅了无明一身。但他顾不上擦拭，因为他感觉到，随着那团黑色物质的破碎，一股精纯的能量顺着刀刃流入他的体内。这股能量与他体内的“沙粒”完美融合，使得那种充盈感变得更加强烈，同时也带来了一种难以言喻的快感。

男人倒在地上，身体迅速干瘪下去，最终化作一堆灰烬，随风消散。只留下一枚指甲盖大小的黑色晶体，静静地躺在地上，散发着微弱的光芒。

无明弯腰捡起那枚晶体。触手冰凉，表面光滑如镜。当他握住晶体的瞬间，脑海中突然闪过一幅画面：一座高耸入云的黑色巨塔，塔顶悬挂着一轮血红色的月亮，无数人影在塔下跪拜，口中念念有词。画面转瞬即逝，取而代之的是一行冰冷的文字，直接烙印在他的意识深处：

**【微粒回收完成。余额：1。】**

无明愣住了。微粒？余额？这些词汇对他来说陌生而又熟悉，仿佛是他生命中不可或缺的一部分，却又遥远得如同前世梦境。他低头看着手中的晶体，又看了看自己布满黑色污渍的双手，心中涌起一股莫名的寒意。

他意识到，自己不仅仅是一个失忆的刺客，更是一个被卷入这场诡异游戏的棋子。而那些所谓的“微粒”，不仅仅是通货，更是维持这个世界运转的根本，也是导致这一切扭曲和疯狂的根源。

远处，更多的脚步声响起。此起彼伏，密密麻麻，如同潮水般涌来。无明抬起头，望向雾气深处。在那里，无数扭曲的身影正向他逼近，他们的眼中闪烁着同样的贪婪和疯狂。

无明握紧短刃，嘴角勾起一抹冷酷的笑意。既然无法逃避，那就战吧。在这片被规则支配的末日荒原上，唯有强者才能生存，唯有掠夺才能延续。

他体内的“沙粒”开始沸腾，内宇宙的雏形在他意识深处悄然展开。那是一个黑暗、深邃的空间，等待着更多的微粒去填充，去构建。而无明知道，这条路注定充满血腥与背叛，但他别无选择。

因为在这个世界里，活着本身，就是一种罪孽。
