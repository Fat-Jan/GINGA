# 第一章 · 锈蚀的脐带

痛觉不是从皮肤表面传来的，而是像某种粘稠的黑色沥青，直接从骨髓深处涌出，沿着脊椎攀爬，最终在颅骨内炸开。无明睁开眼时，并没有看到天空，只有一片被灰白雾气切割得支离破碎的穹顶。那雾气并非水汽，而是无数细小的、死去的尘埃颗粒悬浮在半空，它们静止不动，仿佛时间在这里凝固成了实体。

他躺在一片由黑曜石碎片铺成的荒原上，身下是冰冷刺骨的岩石，每一块石头都像是某种巨大生物脱落的鳞片，边缘锋利如刀。空气稀薄得令人窒息，每一次呼吸都像是在吞咽砂纸，喉咙里泛起铁锈般的腥甜。无明试图坐起，但身体沉重得像是一具灌满铅水的皮囊。他的记忆是一片空白，就像是被大火烧尽后的废墟，连灰烬都被风吹散，只剩下一种本能的、深入灵魂的警觉——那是猎手在濒死前对危险的最后直觉。

他低下头，看向自己的双手。那是一双苍白、修长却布满细小伤痕的手，指关节粗大，掌心有着长期握持利器留下的老茧。在他的右手掌心中，紧紧攥着一把短刃。短刃的材质非金非玉，呈现出一种暗哑的黑灰色，刃口处流淌着若有若无的微光，仿佛内部封印着一条即将干涸的河流。刀柄上刻满了繁复而古老的符文，那些符文扭曲盘绕，如同某种不知名生物的血管脉络，在无明的指尖微微搏动，传递出一股微弱却清晰的温热感。

无明并不认识这些符文，但当他的目光触及它们时，脑海中闪过几个破碎的画面：鲜血喷溅在雪地上，黑色的旗帜在风中撕裂，以及一声凄厉至极的婴儿啼哭。这些画面转瞬即逝，留下的只有剧烈的头痛和一种难以言喻的悲伤。他不知道这把刀从何而来，也不知道自己是谁，但他知道，这把刀是他此刻唯一的依靠，也是他与这个陌生世界之间唯一的联系。

他挣扎着站起身，膝盖发出轻微的咔哒声。四周寂静得可怕，没有风声，没有虫鸣，甚至连自己的心跳声都显得过于响亮。在这片死寂中，无明感到一种莫名的违和感。他的体内似乎有什么东西在蠕动，那不是肠胃的痉挛，而是一种更深层、更隐秘的异物感。它位于丹田下方，靠近耻骨的位置，像是一颗冰冷的种子，正在缓慢地生根发芽。

无明皱起眉头，伸手按住腹部。指尖触碰到皮肤的那一刻，他感觉到皮下有一个微小的硬块，随着他的呼吸节奏轻微起伏。他用力按压，硬块并未消失，反而传来一阵尖锐的刺痛，紧接着，一股奇异的暖流从硬块处扩散开来，迅速蔓延至四肢百骸。这股暖流带着一种诡异的熟悉感，仿佛是他身体的一部分，却又完全不属于他。

就在这时，远处的地平线上出现了一道裂痕。那道裂痕横贯整个视野，将灰白的天空与漆黑的大地强行分割开来。裂痕之中，翻滚着浓稠如墨的黑雾，黑雾中隐约可见巨大的阴影在游动，像是深海中的巨兽，又像是某种不可名状的古老存在。无明本能地后退半步，手中的短刃微微抬起，刃尖指向那道裂痕。他知道，那里就是“天堑”，是这个世界的边界，也是死亡的代名词。

在天堑的边缘，矗立着一座孤城。那座城池由黑色的巨石堆砌而成，城墙高耸入云，表面布满了岁月侵蚀的痕迹和干涸的血迹。城门紧闭，门上悬挂着一块巨大的铁牌，上面用猩红的颜料书写着几行大字。距离太远，无明看不清上面的内容，但他能感受到从那座城中散发出的压抑气息，那是一种混合了恐惧、欲望和绝望的味道，浓烈得几乎让人作呕。

无明决定向那座城走去。他没有其他选择，体内的异物感越来越强烈，那种冰冷的种子似乎在渴望某种养分，而他的身体也在逐渐虚弱。他需要答案，需要食物，需要弄清楚自己究竟变成了什么。每一步踏在黑曜石地面上，都会发出清脆的回响，在这死寂的世界里显得格外刺耳。无明尽量压低身形，利用周围散落的石块作为掩护，保持着最高的警惕。他的感官在这一刻变得异常敏锐，他能听到远处风掠过岩石缝隙的声音，能闻到空气中弥漫的腐朽气息，甚至能感知到脚下大地深处传来的微弱震动。

随着距离的拉近，那座孤城的轮廓逐渐清晰。城墙上的血迹并非随意涂抹，而是形成了一种复杂的图案，像是某种祭祀仪式留下的痕迹。城门前的广场上，散落着许多白骨，有些已经风化，有些还残留着新鲜的肉屑。无明停下脚步，仔细观察着周围的景象。他发现，那些白骨大多集中在城门附近，而且姿态各异，有的呈跪拜状，有的呈挣扎状，还有的则像是被某种力量瞬间撕碎。

突然，一阵低沉的轰鸣声从城门方向传来。厚重的石门缓缓打开，露出一条幽深的通道。通道内昏暗不明，只能看到两侧墙壁上镶嵌着的发光晶体，散发出惨白的光芒。无明握紧短刃，深吸一口气，迈步走入通道。通道内的空气比外面更加寒冷，带着一股浓郁的霉味和血腥味。墙壁上的晶体光芒闪烁不定，仿佛在警告闯入者不要前行。

走了约莫百米，无明来到了一处宽敞的大厅。大厅中央摆放着一座巨大的石台，石台上躺着一具尸体。那是一具年轻女性的尸体，面容姣好，但双眼圆睁，充满了惊恐。她的腹部被剖开，内脏暴露在外，而在她的子宫位置，竟然生长着一簇黑色的晶体，那些晶体晶莹剔透，散发着迷人的光泽，与周围腐烂的血肉形成了鲜明的对比。

无明走近石台，目光落在那簇黑色晶体上。就在这一瞬间，他腹部的异物感突然加剧，那颗冰冷的种子剧烈跳动起来，仿佛受到了某种召唤。短刃上的符文也开始发光，光芒越来越盛，直至将整个大厅照亮。无明感到一阵眩晕，脑海中再次闪过那些破碎的画面，但这次更加清晰：一个身穿黑袍的男人站在祭坛前，手中捧着一颗黑色的晶体，口中念念有词；一群人在台下欢呼，他们的脸上洋溢着狂喜和贪婪；最后，画面定格在一个婴儿的脸上，那个婴儿的眼睛是纯黑色的，没有任何眼白。

“这是……微粒？”无明喃喃自语，虽然他不记得这个词的含义，但当他说出口时，心中涌起一股莫名的确信。他伸出手，想要触碰那簇黑色晶体，但在指尖即将接触到的瞬间，短刃突然发出一声嗡鸣，一股强大的吸力从刃口爆发出来。那簇黑色晶体瞬间崩解，化作无数细小的黑色粉末，被吸入短刃之中。

与此同时，无明感到一股庞大的能量涌入体内，那股能量狂暴而混乱，冲击着他的经脉和脏腑。他痛苦地跪倒在地，全身颤抖，汗水浸透了衣衫。腹部的异物感消失了，取而代之的是一种充盈的力量感，但这种力量感伴随着强烈的副作用——他的意识开始模糊，耳边响起无数嘈杂的低语声，那些声音来自四面八方，有的在哭泣，有的在咒骂，有的在祈求。

“不……不能这样……”无明咬紧牙关，试图抵抗这股力量的侵蚀。他想起短刃上的符文，那些符文此刻正散发着柔和的光芒，仿佛在安抚着他躁动的灵魂。他集中精神，引导着体内的能量流向短刃，试图将其重新封印。这是一个艰难的过程，每一秒都像是一个世纪那么漫长。终于，当最后一丝能量被吸入短刃后，低语声消失了，大厅恢复了平静。

无明瘫软在地，大口喘着粗气。他看着手中的短刃，发现刃身上的符文变得更加明亮，而那些原本暗淡的部分也重新焕发了生机。他意识到，这把短刃不仅仅是一件武器，它是一个容器，一个能够吞噬和储存“微粒”的容器。而刚才那具女尸体内的黑色晶体，正是所谓的“微粒”。

他站起身，感觉身体轻盈了许多，之前的虚弱感一扫而空。但他知道，这只是暂时的。体内的能量虽然稳定下来，但那种饥饿感依然存在，甚至比之前更加强烈。他需要更多的微粒，否则这种平衡很快就会被打破。

无明走出大厅，继续向城内深处探索。街道上空无一人，两旁的房屋门窗紧闭，窗帘拉得严严实实。偶尔能看到一些黑影在窗户后面晃动，但没有人敢出来。这座城像是一座坟墓，埋葬着活人的希望和梦想。

在一处街角，无明发现了一块石碑。石碑上刻满了密密麻麻的文字，字迹潦草而急促，像是有人在极度恐慌中写下的。无明凑近观看，辨认出其中的几行字：

“禁止直视天堑。”
“禁止在午夜后出门。”
“禁止拒绝繁衍契约。”
“微粒即生命，掠夺即罪恶。”
“血脉延续，方得永生。”

这些规则简单而残酷，每一条都透着深深的绝望。无明注意到，“繁衍契约”这四个字被反复强调，字体也比其他部分更大更深。他回想起刚才看到的画面，那个黑袍男人和欢呼的人群，以及那个纯黑眼睛的婴儿。难道这座城的生存法则，就是通过某种方式获取微粒，并通过繁衍来延续这种能力？

无明感到一阵恶心。他不想成为这种规则的一部分，但他别无选择。在这个末日般的世界里，要么遵守规则活下去，要么违背规则死去。而他，作为一个失忆的刺客，既没有过去，也没有未来，只有当下的生存本能。

他继续前行，来到了一座宏伟的建筑前。建筑的大门敞开，里面传出阵阵欢声笑语和酒杯碰撞的声音。无明犹豫了一下，最终还是走了进去。大厅内灯火通明，装饰奢华，与外面的破败景象形成了鲜明的对比。数十名衣着华丽的人围坐在长桌旁，正在享用美食。他们的脸上洋溢着满足和快乐，仿佛忘记了外面的恐怖。

无明的出现引起了众人的注意。笑声戛然而止，所有人都转过头，用警惕和好奇的目光打量着他。一个身穿红色长袍的中年男子站起身，微笑着走向无明。

“欢迎，陌生的客人。”男子的声音温和而富有磁性，“我是这座城的执政官，你可以叫我红袍。看你一身风尘仆仆，想必是刚从外面回来吧？”

无明没有回答，只是冷冷地看着他。他的手始终放在短刃上，随时准备应对突发状况。

红袍似乎并不在意无明的沉默，他挥了挥手，示意侍者端上一杯酒。“喝一杯吧，这里的美酒可以缓解疲劳。另外，我想提醒你，今晚是‘新月之夜’，按照规矩，所有外来者都需要签订一份‘临时庇护契约’。当然，代价是你身上所有的微粒。”

无明心中一凛。他刚刚获得的微粒，难道就要这样交出去？他看了一眼手中的短刃，符文微微闪烁，仿佛在回应他的疑虑。

“如果我不签呢？”无明冷冷地问道。

红袍的笑容僵了一下，随即变得更加深邃。“如果不签，你就无法在这座城里过夜。而外面的夜晚……可是很危险的。尤其是对于你这样，身上带着‘未净化’微粒的人来说。”

无明眯起眼睛，他听出了红袍话语中的威胁。但他并没有退缩，反而向前迈了一步，逼近红袍。“我想知道，什么是‘未净化’微粒？还有，所谓的‘繁衍契约’，到底是什么？”

红袍的眼神变得锐利起来，他上下打量着无明，仿佛在评估他的价值。“看来，你是个有趣的人。好吧，我可以告诉你一些真相。但在此之前，你需要证明你的诚意。”

他从怀中掏出一枚黑色的硬币，扔给无明。“拿着它，去城西的‘育婴堂’走一趟。那里有一个孩子，需要你帮忙‘处理’一下。如果你能做到，我就告诉你一切。如果你做不到……”红袍顿了顿，嘴角勾起一抹残忍的笑意，“那你就会成为下一个‘原料’。”

无明接住硬币，感受着上面冰冷的触感。他知道，这是一个陷阱，也是一个机会。他必须做出选择，是接受这个充满血腥味的交易，还是转身离开，面对未知的危险。

他看了一眼手中的短刃，符文的光芒愈发耀眼。他深吸一口气，将硬币收入囊中。

“带路。”无明说道。

红袍满意地点点头，转身向大厅深处走去。无明紧随其后，心中的警惕并未放松半分。他知道，这场游戏才刚刚开始，而他手中的短刃，将是决定胜负的关键。
