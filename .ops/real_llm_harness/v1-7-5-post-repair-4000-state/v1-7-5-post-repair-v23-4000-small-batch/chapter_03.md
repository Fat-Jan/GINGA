# 第3章 · 锈蚀的脐带与内宇宙的胎动

痛觉并未因意识的回归而消退，反而像是一根烧红的铁钎，顺着脊椎的缝隙狠狠楔入，将无明从混沌的深渊中强行拽回现实。他并没有立刻起身，而是保持着蜷缩的姿态，手指深深扣进黑曜石碎片的缝隙里。指尖传来的触感冰冷、粗糙，带着一种令人作呕的滑腻感——那是某种生物体液干涸后留下的结晶，混合着尘埃，构成了这片荒原的地表。

灰白的雾气依旧悬浮在半空，静止得如同凝固的尸块。无明缓缓抬起头，脖颈处的关节发出细微的咔哒声，仿佛生锈的齿轮在强行咬合。视野中是一片死寂的灰白，没有天空，没有大地，只有无尽的、压抑的雾霭，以及脚下这片由破碎骨骼和黑色岩石拼凑而成的废墟。

记忆是一片空白。不是那种温和的遗忘，而是被暴力剜去后的空洞，边缘还残留着血肉模糊的创口。他不知道自己是谁，不知道为何在此，甚至不知道这具躯壳是否还属于自己。唯一清晰的，是手中紧握的那柄短刃。

刀身漆黑，非金非玉，表面刻满了细密如蚁穴般的古老符文。那些符文并非静止，而是在微微蠕动，像是无数只微小的眼睛，正贪婪地窥视着周围的一切。当无明的目光落在符文上时，脑海中突然闪过一丝尖锐的刺痛，紧接着，一个冰冷的声音在他意识深处响起，不带任何情感，却充满了不容置疑的威严：

*“吞食。”*

无明猛地一颤，手中的短刃险些脱手。他低下头，看向自己的胸口。那里有一道暗红色的伤口，正在渗出黑色的血液。但更让他感到恐惧的，是伤口内部传来的异样搏动。那不是心脏的跳动，而是一种更为原始、更为野蛮的节奏，仿佛有什么东西在他的胸腔内孕育、生长，试图破体而出。

他颤抖着伸出左手，按在伤口上。指尖触碰到了一团温热的、粘稠的物质。那团物质在他的掌心下缓缓蠕动，散发出一种令人眩晕的微光。那是微粒。纯粹的、未被污染的微粒。

在这个末日般的世界里，微粒是唯一的通货，是生命的源泉，也是死亡的诅咒。总量硬上限八亿四千万，每一粒都沾染着鲜血与罪恶。而无明体内的这一团，显然不属于他自己。它是被植入的，是被强行塞进这具躯壳里的异物。

*“吞食。”*

那个声音再次响起，比之前更加急促，更加饥饿。

无明咬紧牙关，强忍着胸腔内的剧痛，将短刃缓缓举起。刀刃上的符文开始疯狂闪烁，原本漆黑的刀身逐渐泛起一层暗红色的光芒。他感觉到一股强大的吸力从刀尖传来，直指胸口的伤口。

这是一种本能。一种深植于骨髓深处的杀戮本能。他不需要思考，不需要犹豫，只需要顺从这股冲动。

他将短刃刺向自己的胸口。

刀刃切入皮肉的瞬间，没有鲜血喷涌，只有一阵刺耳的嘶鸣声。那团黑色的微粒仿佛受到了惊吓，剧烈地收缩起来，试图逃离短刃的吸引。但短刃上的符文如同活物一般，迅速蔓延开来，化作无数条细小的黑色触手，死死缠绕住那团微粒。

无明感到一阵剧烈的眩晕，仿佛整个灵魂都被抽离了身体。他看到自己的意识沉入了一片黑暗的海洋，而在海洋的中心，一座巨大的、由无数微粒构成的岛屿正在缓缓升起。那是他的内宇宙。

在内宇宙的深处，他看到了那团被捕获的微粒。它不再是一团混乱的能量，而是呈现出一种诡异的形态——一颗微型的、跳动的胚胎。胚胎的表面覆盖着密密麻麻的血管，每一根血管中都流淌着暗红色的光芒。

*“血脉……”*

无明的脑海中浮现出这个词。这不是普通的微粒，这是带有血脉印记的微粒。它是某个强大存在的后裔，是被剥夺了身份、被流放到这片荒原的弃子。

短刃上的符文开始疯狂吞噬这颗胚胎。每一次吞噬，无明都能感觉到一股暖流涌入四肢百骸，修复着他受损的身体。但同时，他也感受到了一股强烈的排斥反应。那颗胚胎在反抗，它在尖叫，它在诅咒。

*“不……不要……”*

一个稚嫩的声音在无明的意识中回荡。那是胚胎的意识，微弱而绝望。

无明的心头一震。他停下了动作。短刃悬停在半空，符文的光芒忽明忽暗。他看着那颗挣扎的胚胎，心中涌起一股莫名的情绪。那不是怜悯，也不是同情，而是一种更深层次的共鸣。

他也是被遗弃的。他也是没有过去的。他和这颗胚胎一样，都是这个残酷世界的牺牲品。

*“吞食。”*

短刃上的符文再次催促，声音中带着一丝不耐烦。

无明深吸一口气，眼中的迷茫逐渐消散，取而代之的是一种冷酷的决绝。他知道，如果不吞食这颗胚胎，他很快就会因为失血过多而死。或者，更糟糕的是，这颗胚胎会在他的体内孵化，将他变成一具行尸走肉。

他没有选择。

他重新握紧短刃，用力刺下。

这一次，没有任何阻碍。短刃轻易地穿透了胚胎的外壳，将其彻底粉碎。无数细小的微粒碎片迸溅开来，被短刃贪婪地吸收。无明感到一股前所未有的力量涌入体内，他的肌肉紧绷，骨骼发出噼啪的声响，仿佛正在经历一次重生。

当最后一丝微粒被吸收殆尽时，短刃上的符文终于停止了闪烁，恢复了原本的漆黑。无明瘫软在地，大口喘着粗气。他的胸口已经愈合，只留下一道淡淡的疤痕。

他抬起头，看向四周。灰白的雾气似乎变得更加浓重了，空气中弥漫着一股血腥味。远处，传来了沉重的脚步声。

有人来了。

无明迅速站起身，将短刃收回鞘中。他的身体虽然虚弱，但感官却变得异常敏锐。他能听到风中传来的细微声响，能闻到空气中隐藏的杀意。

脚步声越来越近，伴随着金属摩擦地面的声音。三个身影从雾气中走出。他们身穿破烂的黑色长袍，脸上戴着狰狞的面具，手中握着锈迹斑斑的铁链。

“找到了。”其中一个身影说道，声音沙哑而冷漠，“目标确认。体内有高阶微粒残留。”

“清理掉。”另一个身影回应道，“上面要活的，但如果反抗，就剁碎了喂狗。”

无明眯起眼睛，身体微微下沉，进入了战斗姿态。他不知道这些人是谁，也不知道他们口中的“上面”指的是什么。但他知道，这些人想要他的命，或者说，想要他体内的微粒。

第一个身影挥动铁链，向无明扑来。铁链在空中划出一道弧线，带着呼啸的风声。无明侧身躲过，同时拔出短刃，一刀斩向对方的手腕。

刀刃划过空气，发出一声轻响。对方的手腕应声而断，鲜血喷涌而出。那人发出一声惨叫，踉跄着后退。

另外两个身影见状，立刻围了上来。他们的动作协调而默契，显然经过严格的训练。铁链交织成一张网，将无明困在中间。

无明冷静地观察着他们的动作。他的脑海中闪过无数种应对方案，每一种都精准而致命。这是他的本能，是他作为刺客的天赋。

他等待着一个机会。

当左边的身影挥动铁链攻击他的头部时，无明猛地蹲下，避开了攻击。同时，他右手一挥，短刃刺入了对方的腹部。那人瞪大了眼睛，难以置信地看着无明，随后缓缓倒下。

右边的身影趁机发动攻击，铁链缠住了无明的脚踝，将他拉倒在地。无明顺势翻滚，摆脱了束缚，然后一跃而起，一脚踢在对方的胸口。那人倒飞出去，重重地撞在一块黑曜石上，昏死过去。

剩下的最后一个身影惊恐地看着无明，手中的铁链微微颤抖。他没想到，这个看似虚弱的年轻人竟然如此可怕。

“你……你是谁？”他颤抖着问道。

无明没有回答。他只是冷冷地看着对方，手中的短刃滴着鲜血。

“告诉我，你们是谁派来的。”无明的声音低沉而冰冷，如同来自地狱的寒风。

那人咽了一口唾沫，艰难地说道：“是……是‘收割者’公会。我们只是底层执行者……求求你，放过我……”

“收割者公会……”无明重复着这个名字，脑海中没有任何印象。但这并不重要。重要的是，这些人想要他的微粒，想要他的命。

他一步步走向那人，短刃上的符文再次开始闪烁。

“不……不要……”那人惊恐地后退，直到背部抵住了一块岩石。

无明停下脚步，看着对方恐惧的眼神。他突然意识到，自己并不想杀人。至少，现在不想。他需要信息，需要知道这个世界究竟发生了什么，需要知道自己的过去。

“滚。”无明冷冷地说道。

那人愣了一下，随即连滚带爬地逃进了雾气中。

无明站在原地，看着那人消失的方向，心中涌起一股复杂的情绪。他刚刚完成了一次杀戮，却没有感到丝毫的快感，只有深深的疲惫和空虚。

他低下头，看向手中的短刃。刀刃上的符文依旧在微微蠕动，仿佛在嘲笑他的软弱。

*“还不够。”*

那个声音再次响起，带着一丝讥讽。

无明握紧短刃，指节发白。他知道，这条路才刚刚开始。在这个充满规则怪谈和血脉契约的世界里，他必须变得更强，才能生存下去。

他转过身，看向远方。灰白的雾气中，隐约可见一座城市的轮廓。那座城市高耸入云，塔尖闪烁着诡异的光芒。

那是他的目的地。也是他的战场。

无明迈开步伐，向着那座城走去。每一步都沉重而坚定，仿佛在踏碎过去的阴影，走向未知的未来。

在他的身后，黑曜石碎片上留下了两行深深的脚印，很快就被灰白的雾气掩盖，仿佛从未存在过。
