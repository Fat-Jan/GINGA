# 第2章 · 血契与灰墙

肺叶像被粗糙的铁砂纸反复打磨，每一次呼吸都伴随着灼烧般的刺痛。无明没有停下脚步，他的双腿机械地交替摆动，肌肉纤维在极限拉伸中发出细微的崩裂声。身后的嗡嗡声虽然减弱，但那股令人作呕的腥甜味依然黏附在他的后颈皮肤上，如同无数只看不见的触手在试探他的体温。他冲进那片建筑群的阴影时，并没有感到丝毫的安全，反而是一种更深层的窒息感扑面而来。这里的空气凝固得像陈年的油脂，光线被高耸入云的黑色石墙切割得支离破碎，地面上铺满了不知名生物的骨粉，踩上去发出令人牙酸的咯吱声。他背靠着一面布满苔藓的石壁滑坐在地，手指紧紧扣住短刃的柄部，指节因过度用力而泛白。那把刻有古老符文的短刃此刻正微微发烫，刃口处吞吐着几缕肉眼难辨的黑色微粒，这些微粒像是拥有生命一般，顺着他的指尖渗入皮肤，带来一阵冰冷的麻痹感。这不是普通的武器，它更像是一个饥饿的器官，正在贪婪地汲取周围环境中残留的能量。无明低头看向自己的手掌，掌心的纹路里嵌入了细小的黑点，那是刚才逃亡时被昆虫群叮咬留下的痕迹，但现在，那些伤口正在以肉眼可见的速度愈合，取而代之的是一种诡异的紫色脉络，沿着手臂向上蔓延。

这种愈合并非恩赐，而是一种掠夺。无明清晰地感觉到，体内的某种平衡被打破了。那些黑色的微粒不仅仅是能量，它们携带着某种古老的、暴虐的信息素，强行改写着他的生理结构。记忆是一片空白，但身体却记得如何杀戮，如何生存。他试图调动内宇宙的力量去压制那股异变，却发现自己的精神海如同干涸的河床，只剩下零星的水洼。在这个末日般的世界里，微粒是唯一的通货，也是唯一的权力象征。总量硬上限八亿四千万的微粒，决定了谁能站在天堑之上，谁只能在天堑之下腐烂。而他，一个连名字都记不起来的刺客，体内却凭空多出了一股不属于这个世界的微粒流。这股力量既让他恐惧，又让他兴奋。恐惧源于未知，兴奋则源于对力量的本能渴望。他抬起头，目光穿过昏暗的巷道，望向远处那座巍峨的黑塔。塔身表面流动着暗红色的光芒，那是“繁衍契约”的标志。在这座城里，生存的权利不是靠武力争取的，而是靠血脉的延续来换取的。每一个新生儿都是一笔巨大的微粒投资，每一个死亡都是一次残酷的资源回收。

巷道的尽头传来沉重的脚步声，伴随着金属拖曳地面的刺耳声响。无明瞬间收敛了所有的气息，身体紧贴着石壁，仿佛与阴影融为一体。影迹潜行的能力在这一刻发挥到了极致，他的心跳频率降低到每分钟三十次，血液流动的声音几乎消失。三个身穿灰色长袍的身影出现在视野中，他们的脸上戴着没有任何五官的面具，手中提着闪烁着寒光的钩锁。这是“清道夫”，专门负责清理那些违反规则或失去价值的个体。他们停在了无明藏身的石壁前，其中一个清道夫抬起手，面具上的孔洞中射出一道红光，扫描着周围的区域。红光扫过无明的身体，却没有引发任何警报。无明屏住呼吸，他能感觉到那道红光中蕴含的探测微粒，正试图穿透他的皮肤，探查他体内的异常。短刃上的符文突然剧烈闪烁了一下，一股无形的波动扩散开来，将那些探测微粒全部吞噬。清道夫似乎察觉到了什么，转过头，面具朝向无明的方向，但他看到的只是一片虚无的黑暗。无明知道，自己不能一直躲在这里。清道夫的搜索范围会不断扩大，一旦被发现，他将面临的是整个城邦规则的碾压。他必须主动出击，或者找到离开这里的方法。

就在清道夫准备继续前进时，无明动了。他没有选择逃跑，而是像一只猎豹般从阴影中扑出，短刃在空中划出一道凄厉的弧线。第一个清道夫甚至来不及反应，喉咙就被割开，鲜血喷涌而出，溅在无明的脸上。温热粘稠的液体带着铁锈味，刺激着他的神经。第二个清道夫迅速转身，手中的钩锁挥舞过来，试图缠住无明的四肢。无明侧身避开，钩锁擦着他的脸颊飞过，带起一串火花。他顺势欺身而上，短刃刺入对方的腋下，那里是防护最薄弱的地方。第三个清道夫见状，立刻后退，同时从腰间掏出一个黑色的球体，用力捏碎。球体炸裂开来，释放出大量的绿色烟雾，这是一种致幻剂，能让人产生强烈的幻觉和恐惧。无明感到一阵眩晕，眼前的景象开始扭曲，清道夫的身影变成了无数个张牙舞爪的怪物。但他没有退缩，反而闭上了眼睛，依靠听觉和嗅觉来定位敌人。短刃上的符文再次亮起，这次的光芒更加耀眼，直接将周围的绿色烟雾驱散。无明猛地睁开眼，瞳孔中闪过一丝金色的光芒，那是内宇宙觉醒的征兆。他冲向最后一个清道夫，短刃精准地刺入对方的心脏。

战斗结束得很快，但代价巨大。无明跪倒在地，大口喘着粗气，体内的微粒流变得更加狂暴，仿佛在庆祝这场胜利。他看着地上的三具尸体，心中没有丝毫怜悯，只有冷漠的计算。他伸出手，按在第一个清道夫的胸口，短刃上的符文开始旋转，形成一个微小的漩涡。尸体中的微粒被强行抽取出来，汇入无明的体内。这是一种禁忌的行为，被称为“尸噬”，通常只有高阶的掠夺者才会使用。随着微粒的涌入，无明感到一股暖流遍布全身，之前的疲惫和伤痛迅速消退。但他也感觉到，自己的意识深处多了一些陌生的碎片。那是清道夫生前的记忆片段：冰冷的指令、无尽的巡逻、以及对“主母”的绝对忠诚。这些记忆碎片像病毒一样侵蚀着他的思维，试图将他同化。无明咬紧牙关，用意志力将这些碎片压制下去。他知道，如果放任不管，自己最终会变成另一个没有思想的傀儡。他站起身，捡起清道夫身上的身份牌，上面刻着一个复杂的符号，那是进入内城区的通行证。

天色逐渐暗淡，天空呈现出一种病态的紫红色。远处的黑塔发出了低沉的轰鸣声，那是宵禁开始的信号。无明知道，自己必须在宵禁之前找到一个安全的落脚点。他沿着巷道向深处走去，周围的建筑越来越密集，墙壁上贴满了各种告示和禁令。其中一张告示引起了他的注意，上面画着一个孕妇的剪影，旁边写着一行血红色的字：“血脉即财富，繁衍即救赎。”在这座城市里，女性的地位极高，因为她们是微粒的生产者。每一个成功诞生的孩子，都能为家族带来巨额的微粒奖励。而那些无法生育或拒绝履行繁衍契约的人，则会被视为废物，被剥夺生存的权利。无明看着这张告示，心中涌起一股莫名的厌恶。这种将生命量化为资源的制度，比任何怪兽都要可怕。他撕下告示，揉成一团，扔进路边的排水沟。排水沟里流淌着黑色的污水，散发着恶臭。他跨过污水，继续向前走去。前方出现了一座破旧的庙宇，庙门半掩，里面透出微弱的烛光。无明犹豫了一下，还是走了进去。庙宇内部空旷而阴冷，中央供奉着一尊没有面孔的神像。神像的手中捧着一个婴儿的雕像，婴儿的眼睛是用两颗红色的宝石镶嵌而成的，显得格外诡异。

无明走到神像前，仔细观察着那颗红宝石。他发现，宝石内部竟然有微粒在流动，形成了一种奇特的图案。这图案与他短刃上的符文有着惊人的相似之处。他伸出手指，轻轻触碰那颗宝石。刹那间，一股强大的吸力从宝石中传来，试图将他的灵魂吸入其中。无明大惊失色，连忙收回手，但已经晚了。他的意识被强行拉入了一个虚幻的空间。在这个空间里，他看到了无数的画面：战火纷飞的战场、堆积如山的尸体、哭泣的母亲、以及一个个在黑暗中诞生的婴儿。这些画面快速闪回，最终定格在一个女人的脸上。那个女人有着和他一样的金色瞳孔，她的眼神中充满了悲伤和绝望。她张开嘴，似乎在说着什么，但无明听不到声音。接着，画面破碎，他回到了现实世界。无明大口喘着气，冷汗浸透了衣衫。他不知道刚才看到的是什么，但他感觉到，自己与这个世界之间的联系变得更加紧密了。那颗红宝石黯淡了下去，变成了一块普通的石头。无明将它取下，放入怀中。他隐约觉得，这块石头可能隐藏着关于他身世的重要线索。

庙宇外传来了嘈杂的人声，似乎是有人在寻找失踪的清道夫。无明知道，这里不再安全。他迅速检查了一下自己的装备，确认短刃和身份牌都在身上。然后，他吹灭了蜡烛，消失在庙宇的后门。后门通向一条狭窄的小巷，小巷的尽头是一扇生锈的铁门。无明推开铁门，发现外面是一个废弃的广场。广场上堆满了垃圾和废弃物，空气中弥漫着腐烂的味道。在广场的中央，有一个巨大的坑洞，坑洞里散发着幽幽的蓝光。无明走近坑洞，发现里面竟然是一个地下入口。入口处有一块石碑，上面刻着几个大字：“内宇宙·第四阶”。无明心中一震，内宇宙是传说中强者才能进入的地方，那里蕴含着无尽的奥秘和力量。但他现在的实力，根本不足以应对里面的危险。然而，身后的追兵越来越近，他已经没有别的选择。他深吸一口气，跳进了坑洞。下落的过程漫长而黑暗，耳边只有风声呼啸。当他终于落地时，发现自己身处一个巨大的洞穴中。洞穴的墙壁上镶嵌着发光的晶体，照亮了整个空间。在洞穴的中央，有一座祭坛，祭坛上摆放着一本古老的书籍。

无明走向祭坛，拿起那本书。书的封面是用某种生物的皮革制成的，摸起来柔软而冰冷。他翻开书页，发现里面记载的并不是文字，而是一幅幅动态的画面。这些画面展示了一个个强大的战士如何利用微粒强化自己的身体，如何突破天堑的限制，如何掌控内宇宙的力量。无明看得如痴如醉，他感觉到，自己体内的微粒流开始按照书中的指引运转起来。这是一种全新的修炼方法，不同于他之前所知的任何一种。随着修炼的进行，他的身体发生了翻天覆地的变化。肌肉变得更加结实，感官变得更加敏锐，甚至连视力都得到了极大的提升。他能够清晰地看到空气中流动的微粒轨迹，能够听到远处敌人的呼吸声。更重要的是，他感觉到自己的意识海开始扩张，原本干涸的精神海重新充满了活力。他知道，自己已经迈出了觉醒的第一步。但这仅仅是开始，前方的路依然充满荆棘。他合上书，将其收入怀中。就在这时，洞穴的入口处传来了一阵脚步声。无明警觉地转过身，握紧了短刃。黑暗中，一个身影缓缓走出，那是一个穿着白色长袍的老人，他的脸上布满了皱纹，眼神中透着智慧的光芒。老人看着无明，微微一笑，说道：“你终于来了，无明。”
