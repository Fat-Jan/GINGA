# 第4章 · 血月下的胎动

那枚徽章在掌心残留着冰冷的触感，像是一块刚从尸堆里挖出的冻骨。无明没有立刻将其收起，而是拇指用力碾过徽章表面粗糙的纹路，直到指腹传来细微的刺痛，确认这并非幻觉或某种致幻剂引发的虚影。血月悬于天堑之上，暗红色的光晕如同凝固的血痂，层层叠叠地覆盖在这片被遗忘的荒原边缘。他站起身，膝盖关节发出轻微的爆响，那是长时间保持蹲伏姿态后的生理抗议，也是这具身体尚且存活的证明。风从天堑深处吹来，带着硫磺与腐肉混合的腥气，刮擦着他破损的衣角。他没有回头去看身后那片刚刚结束杀戮的空地，那里躺着三具尸体，他们的血液正顺着地面的裂缝渗入地下，成为滋养这片贫瘠土地的肥料。对于无明而言，死亡不是终结，而是资源流转的开始。他握紧手中的短刃，刀柄上那些古老而扭曲的符文在月光下隐隐泛起幽蓝的光泽，仿佛在呼吸，又像是在嘲笑他的无知。这种光泽让他感到不安，却又莫名熟悉，就像是在梦境深处反复出现的某个场景，醒来后却只剩下一片空白的恐惧。他必须移动，停留意味着暴露，而暴露在这座被规则支配的城市边缘，等同于自杀。

脚步踩在碎石铺就的小径上，发出沙沙的声响，每一步都经过精确的计算，确保不会触发周围可能存在的警戒陷阱。无明的感官被放大到了极致，听觉捕捉着风中每一丝异常的波动，视觉扫视着阴影中每一个可疑的轮廓。他的心跳平稳而缓慢，这是多年训练形成的本能，即便大脑一片空白，肌肉记忆依然忠实地执行着生存的最高指令。然而，就在跨过一道断裂的石梁时，一阵剧烈的绞痛从腹部深处爆发，如同有一把烧红的铁钳在搅动他的内脏。无明闷哼一声，单膝跪地，左手死死按住腹部，右手紧握短刃，警惕地环顾四周。这不是外伤，也不是中毒，而是一种来自内部的、陌生的侵蚀感。他低下头，透过破损的衣衫，看到自己的皮肤下似乎有什么东西在蠕动，像是无数细小的虫子在皮下穿梭，勾勒出诡异的图案。那种感觉既恶心又令人战栗，仿佛他的身体不再完全属于自己，而是成为了某种未知存在的容器。汗水瞬间浸透了后背，冷风一吹，带来刺骨的寒意，但他不敢松懈，因为在这种状态下，任何一点疏忽都可能导致致命的后果。

疼痛并未持续太久，而是逐渐转化为一种温热的流动感，沿着血管蔓延至四肢百骸。无明深吸一口气，强迫自己冷静下来，调动内息去探查那股热流的源头。在内宇宙的感知层面，他“看”到了一团混沌的黑色雾气，正盘踞在他的丹田位置，不断吞噬着周围的生命能量。这团雾气并非静止不动，而是在有节奏地收缩与扩张，如同一个正在孕育中的胚胎。这个比喻让无明感到一阵莫名的厌恶，但他无法否认这种感觉的真实性。他想起了之前在那具尸体身上发现的线索——微粒掠夺集团不仅仅是在掠夺资源，他们似乎在通过某种方式，将掠夺来的微粒植入他人的体内，进行某种实验或改造。难道自己就是那个实验品？这个念头如同一根冰锥，刺穿了他原本就脆弱的心理防线。失忆带来的存在性恐惧再次涌上心头，如果连自己的身体都不再纯粹，那么“我”究竟是谁？这个问题没有答案，只有腹部那团黑色雾气发出的微弱脉动，像是在回应他的质疑，又像是在嘲弄他的无力。

为了压制住体内的异动，无明不得不运转起残存的灵力，试图将那团黑色雾气封锁在丹田之内。这是一个危险的过程，稍有不慎就会导致灵力逆流，经脉尽断。他咬紧牙关，额头青筋暴起，汗水顺着脸颊滑落，滴在干燥的地面上，瞬间蒸发成白色的蒸汽。短刃上的符文似乎感应到了主人的痛苦，光芒变得更加明亮，一股清凉的气息从刀柄传入掌心，顺着手臂流向全身，暂时缓解了那股灼热感。无明抓住这个机会，迅速调整呼吸节奏，引导灵力形成一道屏障，将那团黑色雾气包裹其中。虽然无法彻底清除它，但至少可以限制它的活动范围，防止其进一步扩散。这个过程耗费了他大量的精力，当一切平息下来时，他感到一阵强烈的虚弱感袭来，双腿几乎无法支撑身体的重量。他靠在旁边的岩石上，大口喘息着，眼神中充满了疲惫与警惕。他知道，这只是暂时的平衡，那团雾气随时可能突破封印，而他必须在它失控之前，找到解决的办法。

休息了片刻后，无明重新站起身，继续向城市的方向前进。远处的城墙高耸入云，黑色的砖石在血月的照耀下显得格外阴森，城墙上悬挂着巨大的锁链，随风摆动，发出沉闷的撞击声，仿佛是这座城市的脉搏。城门紧闭，门口站着两名身穿黑甲的守卫，他们的脸上戴着狰狞的面具，手中握着长矛，目光冷漠地扫视着过往的行人。无明压低帽檐，混入人群中，尽量让自己看起来像一个普通的流浪者。人群拥挤而嘈杂，空气中弥漫着汗臭、廉价香料和绝望的味道。每个人都在小心翼翼地前行，避免与他人发生肢体接触，更避免眼神交流。在这里，信任是一种奢侈品，而怀疑则是生存的必需品。无明感受着周围压抑的氛围，心中那股不安愈发强烈。他注意到，许多人的脖子上都戴着一个金属项圈，项圈上闪烁着微弱的红光，那是身份识别器，也是控制装置。一旦违反规则，项圈就会释放电流，甚至直接引爆。这种赤裸裸的控制手段，让这座城市变成了一座巨大的监狱，而居民们则是其中的囚徒。

穿过城门时，无明感到那道扫描光束在自己身上停留了片刻，心脏不由自主地收紧。幸运的是，项圈并没有报警，守卫也没有阻拦，只是用冷漠的眼神瞥了他一眼，便放行通过。进入城内，景象更加触目惊心。街道狭窄而曲折，两旁是破败的房屋，窗户大多被封死，只留下几条缝隙透出一丝昏暗的光线。墙上贴满了各种告示，上面写着密密麻麻的规则条文，字迹潦草而疯狂，像是出自精神失常者之手。“禁止在午夜后外出”、“禁止直视血月超过三秒”、“禁止与非血缘者交换体液”……每一条规则背后，都隐藏着无数的血腥故事。无明快速浏览着这些告示，试图从中找出有用的信息，但大多数内容都毫无逻辑可言，更像是某种宗教狂热的产物。然而，当他看到一条关于“繁衍契约”的告示时，脚步不由得停顿了一下。告示上画着一个复杂的符号，与他短刃上的符文有着惊人的相似之处。符号下方写着一行小字：“血脉即权力，子嗣即财富。签订契约者，可得永生之钥。”这句话如同一道闪电，击中了他的脑海，一些模糊的记忆碎片开始浮现。

记忆的画面破碎而混乱：一个阴暗的房间，一张巨大的圆桌，围坐着几个身影模糊的人。他们在争论着什么，声音低沉而急促。桌上放着一份羊皮卷，上面画着同样的符号。有人伸出手，按在羊皮卷上，鲜血渗出，染红了符号。接着是一阵剧烈的疼痛，以及一个女人的尖叫声。画面戛然而止，无明猛地摇了摇头，试图甩掉这些令人不适的影像。他不知道这些记忆是否真实，也不知道它们与自己现在的处境有何关联。但他隐约感觉到，那份“繁衍契约”或许就是解开这一切谜题的关键。在这个末日般的世界里，人口锐减，生育率极低，而“多子多福”不再是一句吉祥话，而是一种残酷的资源分配机制。拥有更多子嗣的人，能够获得更多的微粒配额，更高的社会地位，甚至更长的寿命。而无明体内的黑色雾气，或许正是这种契约的副产品，或者是某种失败的实验结果。这个推测让他感到一阵寒意，如果他真的是某个实验的失败品，那么等待他的命运，恐怕比死亡更加可怕。

为了验证自己的猜想，无明决定前往城市的中心区域，那里是微粒交易的核心地带，也是情报最密集的地方。他沿着主干道前行，避开那些巡逻的黑甲卫队，利用“影迹潜行”的能力，在阴影中穿梭。这项能力是他目前唯一可以依赖的技能，虽然等级不高，但在复杂的环境中足以让他如鱼得水。街道两旁的店铺大多关门歇业，只有少数几家还在营业，橱窗里展示着各种奇异的物品：封装好的微粒晶体、变异生物的器官、甚至是婴儿的襁褓。无明的目光在一个襁褓上停留了片刻，那里面空空如也，却标价高昂。这是一种讽刺，也是一种警示。在这个世界里，生命本身已经成为了一种商品，可以被买卖、被掠夺、被销毁。他感到一阵恶心，加快了脚步，想要尽快离开这个充满罪恶气息的地方。然而，就在他转过一个街角时，一股熟悉的腥甜味扑面而来，让他不得不停下脚步。

前方是一个小型的广场，广场上聚集了一群人，他们围成一个圈，中间似乎在进行着什么仪式。无明隐藏在阴影中，眯起眼睛观察。只见圆圈中央，一个男人正跪在地上，双手高举，口中念念有词。他的面前摆放着一个祭坛，祭坛上放着一颗散发着微弱光芒的微粒晶体。男人的周围，站着几个身穿红袍的人，他们的脸上涂着油彩，表情狂热而扭曲。随着男人的吟唱，那颗微粒晶体开始震动，光芒越来越亮，最终化作一道光束，射入男人的眉心。男人发出一声惨叫，身体剧烈抽搐，随后瘫软在地，生死不知。红袍人走上前，检查了一下男人的状况，然后满意地点了点头，挥手示意其他人将男人抬走。整个过程无声而诡异，围观的人群没有任何反应，仿佛这一切都是司空见惯的日常。无明感到一阵毛骨悚然，他意识到，这不仅仅是一场简单的仪式，而是一种献祭，一种用生命换取力量的交易。而这种交易，很可能与所谓的“繁衍契约”有着密切的联系。

就在这时，无明体内的黑色雾气突然剧烈波动起来，一股强烈的渴望从他心底升起，指向那颗刚刚被吸收的微粒晶体残留的能量场。这种渴望如此强烈，以至于他几乎无法控制自己的身体，想要冲出去抢夺那份力量。他死死咬住嘴唇，直到尝到血腥味，才勉强压制住这股冲动。他知道，如果自己现在冲出去，不仅得不到任何东西，反而会暴露自己的行踪，成为下一个祭品。他必须忍耐，必须等待更好的时机。他深吸一口气，转身离开广场，朝着相反的方向走去。每一步都走得异常艰难，仿佛在与自己的本能进行搏斗。体内的黑色雾气仍在躁动，像是在抗议他的克制，又像是在催促他行动。无明知道，这场斗争才刚刚开始，而他必须在这场斗争中存活下来，才能揭开真相，找回失去的记忆，以及……那个可能存在的、与他血脉相连的秘密。

夜色渐深，血月的光芒变得更加浓郁，将整个城市笼罩在一片血红之中。无明找到一处废弃的建筑，躲了进去。这里曾经是一座教堂，如今只剩下断壁残垣，十字架倒在地上，沾满了灰尘和血迹。他靠在墙角，闭上眼睛，开始冥想。在内宇宙的深处，他再次看到了那团黑色雾气。这一次，他不再试图压制它，而是尝试着去沟通，去理解它的本质。他将意识沉入其中，感受到了一股古老而苍凉的气息，仿佛穿越了漫长的时光，回到了世界的起源。在那里，他看到了一片荒芜的大地，天空中悬挂着四重天堑，大地之上，无数微小的粒子在飞舞、碰撞、融合。这些粒子，就是微粒，是构成这个世界的基本单位，也是所有生命的源泉。而在粒子的海洋中，有一个巨大的漩涡，正在吞噬着周围的一切。那个漩涡的中心，隐约可见一个身影，手持短刃，刀刃上的符文闪烁着耀眼的光芒。那个身影，似乎就是他自己。
