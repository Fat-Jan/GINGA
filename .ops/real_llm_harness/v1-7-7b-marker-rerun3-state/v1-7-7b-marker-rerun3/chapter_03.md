# 第3章 · 血契与微粒的初诞

无明的指节因过度用力而泛出青白，债刃上的古老符文在昏暗中并未发光，却像是一张饥饿的嘴，贪婪地吮吸着周围稀薄的空气。他没有回答老人的问候，因为“无明”这个名字对他而言，不过是一串毫无意义的音节，如同这具躯壳里残留的陌生记忆碎片一样空洞。老人的笑容并未因他的沉默而收敛，反而更加深邃，那皱纹堆叠的眼角微微下垂，像是在审视一件即将完工的艺术品。老人抬起枯瘦的手，指尖在空中轻轻一点，周围的黑暗仿佛有了实质，化作无数细小的黑色丝线，向无明缠绕而来。这些丝线并非实体，而是某种高浓度的微粒凝聚体，它们带着刺骨的寒意，试图钻进无明的毛孔，渗透进他的血液。无明本能地后退半步，脚跟踩碎了地面上早已风化的骨渣，发出清脆的碎裂声。他手中的债刃横在胸前，刃口对准了那些逼近的黑丝。他能感觉到，体内的某种东西正在苏醒，那不是记忆，而是一种更原始、更暴戾的本能——对微粒的渴望，以及对威胁的本能排斥。黑丝触碰到债刃的瞬间，发出了滋滋的声响，仿佛冷水滴入滚油，债刃表面的符文开始剧烈震颤，一股灼热的气流顺着刀柄涌入无明的掌心。这股热量并不温暖，反而带着一种腐蚀性的痛楚，像是无数根烧红的针在扎刺他的神经。无明咬紧牙关，没有松手，反而握得更紧，他将所有的注意力都集中在刀刃上，感受着那股力量在体内奔涌。他知道，一旦松手，那些黑丝就会彻底吞噬他，将他变成这末日天堑边缘的一具行尸走肉。老人的眼神中闪过一丝赞赏，但更多的是冷漠，他缓缓开口，声音沙哑而低沉：“你比预想中要顽强，无明。但这还不够，你需要明白，在这个世界，生存不是权利，而是交易。”随着话音落下，更多的黑丝从黑暗中涌出，这一次，它们不再试探，而是如暴雨般倾泻而下，将无明完全笼罩。无明感到呼吸困难，肺部像是被灌入了铅水，沉重而冰冷。他试图调动体内的能量去抵抗，却发现自己的经脉干涸如沙漠，没有任何可供调用的力量。绝望感如潮水般袭来，但他眼中的凶性却愈发炽烈。他不想死，至少现在不想。这种强烈的求生欲激发了潜藏在身体深处的某种机制，债刃上的符文倏地爆发出耀眼的红光，光芒瞬间照亮了整个空间。红光所过之处，黑丝纷纷溃散，化作点点星光消散在空气中。无明大口喘着粗气，汗水浸透了衣衫，他看着手中仍在颤抖的债刃，心中涌起一股莫名的熟悉感。这把刀，似乎认识他，或者说，认识他灵魂深处的某个部分。

老人并没有因为攻击被化解而感到惊讶，他只是静静地站在那里，仿佛在等待着一个必然的结果。当红光散去，无明发现周围的景象发生了变化。原本漆黑的空间变得清晰起来，四周矗立着巨大的石柱，石柱上刻满了繁复的图案，那些图案扭曲而诡异，像是某种生物的血管网络，又像是星辰运行的轨迹。地面不再是碎石和骨渣，而是一片光滑如镜的黑色石面，倒映着上方虚无的天空。老人向前迈了一步，脚下的石面泛起涟漪，仿佛水面一般。“欢迎来到‘内宇宙’的边缘，”老人说道，“这里是规则的起点，也是终点。你之所以能醒来，是因为你的血脉中流淌着古老的契约。这份契约，是你存在的唯一理由，也是你痛苦的根源。”无明警惕地盯着老人，手中的短刃依然紧握，但他能感觉到，刚才那一击已经耗尽了他大半的体力。他的双腿微微发抖，视线也开始模糊。他必须尽快恢复状态，否则接下来的任何变故都可能致命。他深吸一口气，强迫自己冷静下来，大脑飞速运转，分析着眼前的局势。老人显然拥有远超常人的力量，而且对这里的环境了如指掌。硬拼绝非明智之举，他需要找到对方的弱点，或者利用环境中的某些因素来扭转局面。就在这时，他注意到石柱上的图案似乎在流动，那些线条像是在呼吸，每一次起伏都伴随着微弱的微粒波动。无明心中一动，他尝试着将意识集中在一根石柱上，想象着自己能够感知到其中的微粒流动。起初，什么也没有发生，但随着他专注力的提升，他逐渐感受到了一股细微的牵引力。那股力量来自石柱内部，像是某种召唤，引诱着他靠近。他犹豫了片刻，最终还是迈出了脚步，向着最近的一根石柱走去。老人没有阻止他，只是嘴角勾起一抹意味深长的笑容，仿佛在欣赏一只落入陷阱的猎物。

当无明的手指触碰到石柱的那一刻，一股庞大的信息流瞬间冲入他的脑海。那不是语言，也不是图像，而是一种纯粹的感受——痛苦、挣扎、繁衍、死亡。他看到了无数个画面：男人在战场上厮杀，女人在产床上呻吟，孩子在饥荒中哭泣，老人在病榻上离世。这些画面交织在一起，形成了一幅宏大而悲惨的画卷。在这幅画卷的中心，是一个巨大的漩涡，漩涡中充满了黑色的微粒，它们相互碰撞、融合、分裂，最终形成了新的生命。无明感到一阵剧烈的头痛，像脑袋要被撑裂开来。他跪倒在地，双手抱头，发出痛苦的嘶吼。然而，在这极致的痛苦中，他却清晰地听到了一个声音，那个声音来自他的胸骨后方，冷漠而坚定：“接受它，这是你的血契。”这个声音让他感到恐惧，但也让他感到一种奇异的安宁。他停止了挣扎，任由那股信息流冲刷着自己的意识。渐渐地，疼痛减轻了，取而代之的是一种充盈感。他感觉到自己的体内多了一些东西，那是微粒，是构成这个世界的基本单位。它们在他的血液中流淌，在他的骨骼中沉积，在他的肌肉中跳跃。他睁开眼，发现眼前的世界变得更加清晰，他甚至能看到空气中漂浮的微小尘埃，以及它们之间复杂的相互作用。他站起身，感觉身体轻盈了许多，之前的疲惫一扫而空。他看向老人，眼中多了一丝自信，也多了一丝疑惑。“这是什么？”他问道，声音有些沙哑。老人微微一笑，说道：“这是‘血脉’的力量，也是‘繁衍契约’的馈赠。你每杀死一个敌人，每孕育一个新的生命，都会获得微粒。微粒越多，你的力量就越强，你的寿命就越长。但同时，你也越容易迷失自我，成为规则的奴隶。”无明皱了皱眉，他对这种说法感到厌恶。他不相信有什么所谓的馈赠，这一切听起来更像是一场精心设计的骗局。但他不得不承认，刚才的经历确实让他获得了实实在在的力量。他低头看了看自己的手掌，发现皮肤下隐隐有红光流转，那是微粒在活跃的迹象。他握紧拳头，感受着那股力量的涌动，心中涌起一股难以抑制的冲动。他想试试这股力量究竟有多强大，也想看看，这个所谓的“内宇宙”，到底隐藏着多少秘密。
